<p><b>Walter G. Bright</b> is an American <a href="/wiki/Computer_programmer" class="mw-redirect" title="Computer programmer">computer programmer</a> who created the <a href="/wiki/D_(programming_language)" title="D (programming language)">D programming language</a>,<sup id="cite_ref-wired_1-0" class="reference"><a href="#cite_note-wired-1">&#91;1&#93;</a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> the <a href="/w/index.php?title=Zortech_C%2B%2B&amp;action=edit&amp;redlink=1" class="new" title="Zortech C++ (page does not exist)">Zortech C++</a> compiler, and the <i><a href="/wiki/Empire_(1977_video_game)" title="Empire (1977 video game)">Empire</a></i> computer game.
</p><p>Bright was the main developer of the first <a href="/wiki/C%2B%2B" title="C++">C++</a> compiler to translate source code directly to object code without using <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a> as an intermediate,<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (September 2020)">citation needed</span></a></i>&#93;</sup> and Zortech C++ (later becoming <a href="/wiki/NortonLifeLock" title="NortonLifeLock">Symantec</a> C++, now <a href="/wiki/Digital_Mars" title="Digital Mars">Digital Mars</a> C++). Before designing the C++ <a href="/wiki/Compiler" title="Compiler">compiler</a>, he developed the <a href="/wiki/Datalight_C" class="mw-redirect" title="Datalight C">Datalight C</a> compiler, also sold as Zorland C and later Zortech C.<sup id="cite_ref-Zortech_C++_1991_3-0" class="reference"><a href="#cite_note-Zortech_C++_1991-3">&#91;3&#93;</a></sup>
</p>

<h2><span class="mw-headline" id="Biography">Biography</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Walter_Bright&amp;action=edit&amp;section=1" title="Edit section: Biography">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Bright graduated from <a href="/wiki/California_Institute_of_Technology" title="California Institute of Technology">Caltech</a> in 1979 with a <a href="/wiki/Bachelor_of_Science" title="Bachelor of Science">Bachelor of Science</a> in aerospace engineering.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> While at university he wrote the <i><a href="/wiki/Empire_(1977_video_game)" title="Empire (1977 video game)">Empire</a></i> wargame for the <a href="/wiki/PDP-10" title="PDP-10">PDP-10</a> mainframe; it was completed in 1977.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p><p>Walter Bright is the creator of the D programming language.  He has implemented compilers for several other languages, and is considered an expert in many areas related to compiler technology.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup> Walter regularly writes scientific and magazine articles about compilers and programming and was a blogger for <i><a href="/wiki/Dr._Dobb%27s_Journal" title="Dr. Dobb&#39;s Journal">Dr. Dobb's Journal</a></i>.
</p><p>He was hired by <a href="/wiki/Facebook" title="Facebook">Facebook</a><sup class="noprint Inline-Template" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Manual_of_Style/Dates_and_numbers#Chronological_items" title="Wikipedia:Manual of Style/Dates and numbers"><span title="The time period mentioned near this tag is ambiguous. (April 2019)">when?</span></a></i>&#93;</sup> to write a fast C/C++ preprocessor in D.<sup id="cite_ref-wired_1-1" class="reference"><a href="#cite_note-wired-1">&#91;1&#93;</a></sup>
</p>
