<p><b>Арктика</b>&#160;— северная полярная часть <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F" title="Земля">земного шара</a>. Включает в себя: вода&#160;— <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%9B%D0%B5%D0%B4%D0%BE%D0%B2%D0%B8%D1%82%D1%8B%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Северный Ледовитый океан">Северный Ледовитый океан</a>, небольшая часть <a href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Атлантический океан">Атлантического</a> и <a href="/wiki/%D0%A2%D0%B8%D1%85%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Тихий океан">Тихого океана</a>; суша&#160;— часть <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a>, <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%B0%D0%B4%D0%B0" title="Канада">Канады</a>, северная часть <a href="/wiki/%D0%A1%D0%BA%D0%B0%D0%BD%D0%B4%D0%B8%D0%BD%D0%B0%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Скандинавский полуостров">Скандинавии</a>, часть американского штата <a href="/wiki/%D0%90%D0%BB%D1%8F%D1%81%D0%BA%D0%B0" title="Аляска">Аляска</a>, датской <a href="/wiki/%D0%93%D1%80%D0%B5%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Гренландия">Гренландии</a>, норвежский архипелаг <a href="/wiki/%D0%A8%D0%BF%D0%B8%D1%86%D0%B1%D0%B5%D1%80%D0%B3%D0%B5%D0%BD" title="Шпицберген">Шпицберген</a>.
</p>

<p><b><a href="/wiki/%D0%90%D1%80%D0%BA%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%8D%D0%BA%D1%81%D0%BF%D0%B5%D0%B4%D0%B8%D1%86%D0%B8%D1%8F_%D0%9F%D0%B8%D1%80%D0%B8_(1908%E2%80%941909)" title="Арктическая экспедиция Пири (1908—1909)">Экспедиция Пири 1908—1909 гг.</a></b> — шестая <a href="/wiki/%D0%90%D1%80%D0%BA%D1%82%D0%B8%D0%BA%D0%B0" title="Арктика">арктическая</a> экспедиция <a href="/wiki/%D0%9F%D0%B8%D1%80%D0%B8,_%D0%A0%D0%BE%D0%B1%D0%B5%D1%80%D1%82" title="Пири, Роберт">Роберта Пири</a> и его третья попытка достигнуть <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%BF%D0%BE%D0%BB%D1%8E%D1%81" title="Северный полюс">Северного полюса</a>, успешно завершившаяся <a href="/wiki/6_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="6 апреля">6 апреля</a> <a href="/wiki/1909_%D0%B3%D0%BE%D0%B4" title="1909 год">1909 года</a>. По словам руководителя, завершала его 23-летнюю борьбу за покорение Северного полюса. Экспедиция ставила перед собой сугубо спортивные задачи, поэтому её вклад в развитие науки невелик. Сразу после возвращения достижение Пири было поставлено под сомнение <a href="/wiki/%D0%9A%D1%83%D0%BA,_%D0%A4%D1%80%D0%B5%D0%B4%D0%B5%D1%80%D0%B8%D0%BA" title="Кук, Фредерик">Ф. Куком</a>, заявившим, что достиг Северного полюса ранее — <a href="/wiki/21_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="21 апреля">21 апреля</a> <a href="/wiki/1908_%D0%B3%D0%BE%D0%B4" title="1908 год">1908 года</a>. Точка в этом споре не поставлена по сей день, а во второй половине ХХ в. распространились версии, что Пири вообще не достиг Северного полюса.

<li>В <a href="/wiki/XXI_%D0%B2%D0%B5%D0%BA" title="XXI век">XXI веке</a> в <a href="/wiki/%D0%A7%D1%83%D0%BA%D0%BE%D1%82%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Чукотское море">Чукотском море</a> <b><a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%BF%D0%BE%D0%BB%D1%8E%D1%81-38" title="Северный полюс-38">борьба за право на природные ресурсы</a></b> ведётся с помощью <a href="/wiki/%D0%9D%D0%B0%D1%83%D0%BA%D0%B0" title="Наука">науки</a>, а не <a href="/wiki/%D0%9E%D1%80%D1%83%D0%B6%D0%B8%D0%B5" title="Оружие">оружия</a>.</li>
<li><b><a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D0%BE%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D1%8D%D0%BA%D1%81%D0%BF%D0%B5%D0%B4%D0%B8%D1%86%D0%B8%D1%8F" title="Русская полярная экспедиция">Русская полярная экспедиция</a></b> упустила <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%97%D0%B5%D0%BC%D0%BB%D1%8F" title="Северная Земля">мировое географическое открытие</a> из-за погони за <a href="/wiki/%D0%97%D0%B5%D0%BC%D0%BB%D1%8F_%D0%A1%D0%B0%D0%BD%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2%D0%B0" title="Земля Санникова">призраком</a>.</li></ul>

<div style="display:block; border:1px solid #699; vertical-align:top; background:white; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<div class="floatright"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Fridtjof_Nansen.jpg" class="image" title="Фритьоф Нансен около 1900 года"><img alt="Фритьоф Нансен около 1900 года" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/12/Fridtjof_Nansen.jpg/120px-Fridtjof_Nansen.jpg" decoding="async" width="120" height="139" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/12/Fridtjof_Nansen.jpg/180px-Fridtjof_Nansen.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/12/Fridtjof_Nansen.jpg/240px-Fridtjof_Nansen.jpg 2x" data-file-width="2339" data-file-height="2718" /></a></div>
<p><b><a href="/wiki/%D0%9D%D0%B0%D0%BD%D1%81%D0%B5%D0%BD,_%D0%A4%D1%80%D0%B8%D1%82%D1%8C%D0%BE%D1%84" title="Нансен, Фритьоф">Фритьоф Нансен</a></b> (полное имя — <b>Фри́тьоф Ве́дель-Я́рлсберг На́нсен</b>, <a href="/wiki/%D0%9D%D0%BE%D1%80%D0%B2%D0%B5%D0%B6%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Норвежский язык">норв.</a> <span lang="no" style="font-style:italic;">Fridtjof Wedel-Jarlsberg Nansen</span>; <a href="/wiki/10_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="10 октября">10 октября</a> <a href="/wiki/1861" class="mw-redirect" title="1861">1861</a> — <a href="/wiki/13_%D0%BC%D0%B0%D1%8F" title="13 мая">13 мая</a> <a href="/wiki/1930" class="mw-redirect" title="1930">1930</a>) — <a href="/wiki/%D0%9D%D0%BE%D1%80%D0%B2%D0%B5%D0%B3%D0%B8%D1%8F" title="Норвегия">норвежский</a> полярный исследователь, учёный — доктор зоологии, основатель новой науки — <a href="/wiki/%D0%9E%D0%BA%D0%B5%D0%B0%D0%BD%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Океанология">физической океанографии</a>, политический и общественный деятель, <a href="/wiki/%D0%93%D1%83%D0%BC%D0%B0%D0%BD%D0%B8%D1%81%D1%82" class="mw-redirect" title="Гуманист">гуманист</a>, <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B0%D0%BD%D1%82%D1%80%D0%BE%D0%BF" title="Филантроп">филантроп</a>, лауреат <a href="/wiki/%D0%9D%D0%BE%D0%B1%D0%B5%D0%BB%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%B5%D0%BC%D0%B8%D1%8F_%D0%BC%D0%B8%D1%80%D0%B0" title="Нобелевская премия мира">Нобелевской премии мира</a> за <a href="/wiki/1922_%D0%B3%D0%BE%D0%B4" title="1922 год">1922 год</a>, удостоен наград многих стран, в том числе России. Именем Нансена названы географические и астрономические объекты, в том числе кратер на Северном полюсе Луны.
</p><p>В молодости стал известен как <a href="/wiki/%D0%A1%D0%BF%D0%BE%D1%80%D1%82%D1%81%D0%BC%D0%B5%D0%BD" class="mw-redirect" title="Спортсмен">спортсмен</a>-<a href="/wiki/%D0%9B%D1%8B%D0%B6%D0%BD%D0%B8%D0%BA" class="mw-redirect" title="Лыжник">лыжник</a> и <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%8C%D0%BA%D0%BE%D0%B1%D0%B5%D0%B6%D0%B5%D1%86" class="mw-redirect" title="Конькобежец">конькобежец</a>. В 27-летнем возрасте впервые в истории пересёк на лыжах <a href="/wiki/%D0%93%D1%80%D0%B5%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D0%BB%D0%B5%D0%B4%D1%8F%D0%BD%D0%BE%D0%B9_%D1%89%D0%B8%D1%82" title="Гренландский ледяной щит">ледниковый покров</a> острова <a href="/wiki/%D0%93%D1%80%D0%B5%D0%BD%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Гренландия">Гренландия</a>, что было воспринято широкой публикой как грандиозное спортивное достижение. В ходе попытки достижения Северного полюса — <a href="/wiki/%D0%9D%D0%BE%D1%80%D0%B2%D0%B5%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D0%BE%D0%BB%D1%8F%D1%80%D0%BD%D0%B0%D1%8F_%D1%8D%D0%BA%D1%81%D0%BF%D0%B5%D0%B4%D0%B8%D1%86%D0%B8%D1%8F_(1893%E2%80%941896)" title="Норвежская полярная экспедиция (1893—1896)">экспедиции на корабле</a> «<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BC" title="Фрам">Фрам</a>» — достиг <a href="/wiki/8_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="8 апреля">8 апреля</a> <a href="/wiki/1895_%D0%B3%D0%BE%D0%B4" title="1895 год">1895 года</a> 86° 13′ 36" с. ш. Хотя после этого Нансен не участвовал в первопроходческих предприятиях, методы передвижения и выживания во льдах и используемое им оборудование стали примером для подражания для многих полярников мирового класса; Нансен регулярно консультировал полярных исследователей из разных стран.
