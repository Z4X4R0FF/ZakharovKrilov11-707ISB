Омывается <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B8%D0%B1%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Карибское море">Карибским морем</a> и <a href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Атлантический океан">Атлантическим океаном</a> на севере, граничит с <a href="/wiki/%D0%93%D0%B0%D0%B9%D0%B0%D0%BD%D0%B0" title="Гайана">Гайаной</a> на востоке, <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D1%8F" title="Бразилия">Бразилией</a>&#160;— на юге и <a href="/wiki/%D0%9A%D0%BE%D0%BB%D1%83%D0%BC%D0%B1%D0%B8%D1%8F" title="Колумбия">Колумбией</a>&#160;— на западе.

<b>Протесты в Венесуэле в 2014 году</b> (<a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Испанский язык">исп.</a>&#160;<span lang="es"><i>Protestas en Venezuela de 2014</i></span> или <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Испанский язык">исп.</a>&#160;<span lang="es"><i>Primavera Venezolana</i></span>) — массовые акции и демонстрации, начавшиеся <a href="/wiki/4_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="4 февраля">4 февраля</a> <a href="/wiki/2014_%D0%B3%D0%BE%D0%B4" title="2014 год">2014 года</a> со студенческих протестов в <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%9A%D1%80%D0%B8%D1%81%D1%82%D0%BE%D0%B1%D0%B0%D0%BB%D1%8C_(%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B0)" class="mw-redirect" title="Сан-Кристобаль (Венесуэла)">Сан-Кристобале</a>, столице штата <a href="/wiki/%D0%A2%D0%B0%D1%87%D0%B8%D1%80%D0%B0" title="Тачира">Тачира</a>, а затем перекинувшиеся на другие города, в частности на <a href="/wiki/%D0%9C%D0%B5%D1%80%D0%B8%D0%B4%D0%B0_(%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B0)" title="Мерида (Венесуэла)">Мериду</a>. Их участники жаловались на недостаточные меры безопасности в университетских кампусах, а также протестовали против экономического кризиса, вызванного, как они считают, политикой правительства. Позже протесты перекинулись и на столицу страны, <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0%D0%BA%D0%B0%D1%81" title="Каракас">Каракас</a>. Некоторые участники этих акций были задержаны силами безопасности, что дало толчок новым протестам, вылившимся к столкновениям с полицией и человеческим жертвам. <i>(<a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B5%D1%81%D1%82%D1%8B_%D0%B2_%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B5_%D0%B2_2014_%D0%B3%D0%BE%D0%B4%D1%83" class="mw-redirect" title="Протесты в Венесуэле в 2014 году">далее…</a>)</i><div style="font-size:85%; text-align:right; clear:both"><a href="/wiki/%D0%A1%D0%BB%D1%83%D0%B6%D0%B5%D0%B1%D0%BD%D0%B0%D1%8F:EditPage/%D0%9F%D0%BE%D1%80%D1%82%D0%B0%D0%BB:%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B0/%D0%98%D0%BD%D1%82%D0%B5%D1%80%D0%B5%D1%81%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D0%B0%D1%82%D1%8C%D1%8F" title="Служебная:EditPage/Портал:Венесуэла/Интересная статья">Править</a> |&#160;<a href="/w/index.php?title=%D0%9F%D0%BE%D1%80%D1%82%D0%B0%D0%BB:%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B0/%D0%98%D0%BD%D1%82%D0%B5%D1%80%D0%B5%D1%81%D0%BD%D0%B0%D1%8F_%D1%81%D1%82%D0%B0%D1%82%D1%8C%D1%8F/%D0%90%D1%80%D1%85%D0%B8%D0%B2&amp;action=edit&amp;redlink=1" class="new" title="Портал:Венесуэла/Интересная статья/Архив (страница отсутствует)">Архив</a></div></div>

<p><b>Себастья́н Франси́ско де Мира́нда-и-Родри́гес</b> (<a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Испанский язык">исп.</a>&#160;<span lang="es"><i>Sebastián Francisco de Miranda y Rodríguez</i></span>; <span class="nowrap"><span data-wikidata-property-id="P569" class="no-wikidata"><a href="/wiki/28_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="28 марта">28&#160;марта</a> <a href="/wiki/1750_%D0%B3%D0%BE%D0%B4" title="1750 год">1750</a></span><span class="noprint" style="display:none"> (<span class="bday">1750-03-28</span>)</span></span>, <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0%D0%BA%D0%B0%D1%81" title="Каракас">Каракас</a>&#160;— <span data-wikidata-property-id="P570" class="no-wikidata"><a href="/wiki/14_%D0%B8%D1%8E%D0%BB%D1%8F" title="14 июля">14 июля</a> <a href="/wiki/1816_%D0%B3%D0%BE%D0%B4" title="1816 год">1816</a></span>, <a href="/wiki/%D0%A1%D0%B0%D0%BD-%D0%A4%D0%B5%D1%80%D0%BD%D0%B0%D0%BD%D0%B4%D0%BE_(%D0%9A%D0%B0%D0%B4%D0%B8%D1%81)" title="Сан-Фернандо (Кадис)">Сан-Фернандо</a>, <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F" title="Испания">Испания</a>)&#160;— руководитель борьбы за независимость испанских колоний в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Америка">Южной Америке</a>. Революционер, национальный герой <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B0" title="Венесуэла">Республики Венесуэла</a>. Автор <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3_%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D1%8B" title="Флаг Венесуэлы">флага Венесуэлы</a>. Участник <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D1%84%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B0%D1%8F_%D1%80%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Великая французская революция">Великой французской революции</a>.
</p><p>Родился <a href="/wiki/28_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="28 марта">28 марта</a> <a href="/wiki/1750_%D0%B3%D0%BE%D0%B4" title="1750 год">1750 года</a> в <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0%D0%BA%D0%B0%D1%81" title="Каракас">Каракасе</a>, столице испанской колонии Венесуэлы. Его отец Себастья́н де Мира́нда Равело был уроженцем Оротавы на острове <a href="/wiki/%D0%A2%D0%B5%D0%BD%D0%B5%D1%80%D0%B8%D1%84%D0%B5" title="Тенерифе">Тенерифе</a> и потомком <a href="/wiki/%D0%93%D1%83%D0%B0%D0%BD%D1%87%D0%B8" title="Гуанчи">гуанчей</a>. Переселившись в <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0%D0%BA%D0%B0%D1%81" title="Каракас">Каракас</a>, Себастья́н в
</p>
1749&#160;г. женился на местной уроженке Франси́ске-Антонии Родригес де Эспиноса, также имевшей канарских предков. Быть может, именно в родовых корнях следует искать истоки той анти-колониальной и анти-испанской идеологии, которую ещё в юные годы разработал и «пропустил через себя» Себастья́н Франси́ско де Мира́нда-и-Родри́гес. <i>El Guancho Miranda</i>&#160;— как величают его уругвайские историки. <ul><li>Венесуэльский оппозиционер выиграл <b><a href="/wiki/%D0%A0%D0%B5%D0%B3%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5_%D0%B2%D1%8B%D0%B1%D0%BE%D1%80%D1%8B_%D0%B2_%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B5_(2017)" title="Региональные выборы в Венесуэле (2017)">региональные выборы</a></b>, но отказался присягать своим противникам в <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%BE%D0%B5_%D1%81%D0%BE%D0%B1%D1%80%D0%B0%D0%BD%D0%B8%D0%B5_%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D1%8B_(2017)" class="mw-redirect" title="Конституционное собрание Венесуэлы (2017)">Конституционном собрании</a>.</li>
<li>Название „<a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D0%B0" title="Венесуэла">Венесуэла</a>“ придумал спутник <a href="/wiki/%D0%9A%D0%BE%D0%BB%D1%83%D0%BC%D0%B1" class="mw-redirect" title="Колумб">Колумба</a> по имени <b><a href="/wiki/%D0%90%D0%BB%D0%BE%D0%BD%D1%81%D0%BE_%D0%B4%D0%B5_%D0%9E%D1%85%D0%B5%D0%B4%D0%B0" class="mw-redirect" title="Алонсо де Охеда">Алонсо де Охеда</a></b>.</li>
<li>По предложению <a href="/wiki/%D0%A3%D0%B3%D0%BE_%D0%A7%D0%B0%D0%B2%D0%B5%D1%81" class="mw-redirect" title="Уго Чавес">Уго Чавеса</a> конь на <b><a href="/wiki/%D0%93%D0%B5%D1%80%D0%B1_%D0%92%D0%B5%D0%BD%D0%B5%D1%81%D1%83%D1%8D%D0%BB%D1%8B" title="Герб Венесуэлы">гербе Венесуэлы</a></b> стал бежать не вправо, а влево.</li>
<li><a href="/wiki/%D0%A2%D1%80%D1%83%D1%89%D0%BE%D0%B1%D1%8B" title="Трущобы">Трущобы</a> бывают не только горизонтальными, но и <b><a href="/wiki/%D0%A4%D0%B8%D0%BD%D0%B0%D0%BD%D1%81%D0%BE%D0%B2%D1%8B%D0%B9_%D1%86%D0%B5%D0%BD%D1%82%D1%80_%D0%9A%D0%BE%D0%BD%D1%84%D0%B8%D0%BD%D0%B0%D0%BD%D1%81%D0%B0%D1%81" title="Финансовый центр Конфинансас">вертикальными</a></b>.</li>
<li><b><a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B4%D0%BE-%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D1%82%D1%82%D0%B0,_%D0%90%D0%BD%D1%82%D0%BE%D0%BD%D0%B8%D0%BE_%D0%A5%D0%BE%D1%81%D0%B5" title="Пардо-Андретта, Антонио Хосе">Президент лыжной федерации Венесуэлы</a></b> занялся <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BD%D0%BE%D0%BB%D1%8B%D0%B6%D0%BD%D1%8B%D0%B9_%D1%81%D0%BF%D0%BE%D1%80%D1%82" title="Горнолыжный спорт">спортом</a> профессионально только после потери работы брокера.</li>
<li>Водный образ жизни <b><a href="/wiki/%D0%92%D0%B0%D1%80%D0%B0%D0%BE_(%D0%BD%D0%B0%D1%80%D0%BE%D0%B4)" class="mw-redirect" title="Варао (народ)">племени варао</a></b> дал название Венесуэле (что значит <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Испанский язык">по-испански</a> „маленькая <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%86%D0%B8%D1%8F" title="Венеция">Венеция</a>“).</li>
<li>Многие путешественники, приезжающие в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Америка">Южную Америку</a> из <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">Европы</a>, посещают город <b><a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0%D0%BA%D0%B0%D1%81" title="Каракас">Каракас</a></b> или, по крайней мере, аэропорт венесуэльской столицы. Объясняется это тем, что межатлантические воздушные лайнеры обычно совершают в столице Венесуэлы промежуточную посадку.</li>
<li><b><a href="/wiki/%D0%9C%D0%B5%D1%80%D0%BA%D0%BE%D1%81%D1%83%D1%80" class="mw-redirect" title="Меркосур">Меркосур</a></b> — крупнейшая международная организация в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Америка">Южной Америке</a>, в неё входят <a href="/wiki/%D0%90%D1%80%D0%B3%D0%B5%D0%BD%D1%82%D0%B8%D0%BD%D0%B0" title="Аргентина">Аргентина</a>, <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D1%8F" title="Бразилия">Бразилия</a>, Венесуэла, <a href="/wiki/%D0%9F%D0%B0%D1%80%D0%B0%D0%B3%D0%B2%D0%B0%D0%B9" title="Парагвай">Парагвай</a> и <a href="/wiki/%D0%A3%D1%80%D1%83%D0%B3%D0%B2%D0%B0%D0%B9" title="Уругвай">Уругвай</a>, а в качестве ассоциированных членов — <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B8%D0%B2%D0%B8%D1%8F" title="Боливия">Боливия</a>, <a href="/wiki/%D0%93%D0%B0%D0%B9%D0%B0%D0%BD%D0%B0" title="Гайана">Гайана</a>, <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%83" title="Перу">Перу</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BB%D1%83%D0%BC%D0%B1%D0%B8%D1%8F" title="Колумбия">Колумбия</a>, <a href="/wiki/%D0%A1%D1%83%D1%80%D0%B8%D0%BD%D0%B0%D0%BC" title="Суринам">Суринам</a>, <a href="/wiki/%D0%A7%D0%B8%D0%BB%D0%B8" title="Чили">Чили</a> и <a href="/wiki/%D0%AD%D0%BA%D0%B2%D0%B0%D0%B4%D0%BE%D1%80" title="Эквадор">Эквадор</a>.
