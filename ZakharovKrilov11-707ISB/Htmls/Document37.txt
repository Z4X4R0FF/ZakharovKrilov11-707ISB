<p><b>Геноци́д армя́н</b> (<a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Армянский язык">арм.</a>&#160;<span lang="hy">Հայոց Ցեղասպանություն</span>, <a href="/wiki/%D0%A2%D1%83%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Турецкий язык">тур.</a> <span lang="tr" style="font-style:italic;">Ermeni soykırımı</span> — геноцид армян, <a href="/wiki/%D0%A2%D1%83%D1%80%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Турецкий язык">тур.</a> <span lang="tr" style="font-style:italic;">Ermeni Kırımı</span> — армянская резня; по-армянски чаще используется выражение <span lang="hy">Մեծ Եղեռն</span> [<small>МФА:</small> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[mɛts jɛˈʁɛrn]</a></span>] — <i>Великое Злодеяние</i>) — <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D1%86%D0%B8%D0%B4" title="Геноцид">геноцид</a>, организованный и осуществлённый в <a href="/wiki/1915_%D0%B3%D0%BE%D0%B4" title="1915 год">1915 году</a> (по мнению некоторых источников, продлившийся до <a href="/wiki/1923_%D0%B3%D0%BE%D0%B4" title="1923 год">1923 года</a>) на территориях, контролируемых властями <a href="/wiki/%D0%9E%D1%81%D0%BC%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Османская империя">Османской империи</a>. Геноцид осуществлялся путём физического уничтожения и депортации, включая перемещение <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B6%D0%B4%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" class="mw-redirect" title="Гражданское население">гражданского населения</a> в условиях, приводящих к неминуемой смерти.
</p><p>Геноцид армян проводился в несколько стадий: разоружение армянских солдат, избирательная депортация армян из пограничных районов, принятие закона о высылке, массовая депортация и убийства армян. Некоторые историки включают в него <a href="/wiki/%D0%9C%D0%B0%D1%81%D1%81%D0%BE%D0%B2%D1%8B%D0%B5_%D1%83%D0%B1%D0%B8%D0%B9%D1%81%D1%82%D0%B2%D0%B0_%D0%B0%D1%80%D0%BC%D1%8F%D0%BD_%D0%B2_1894%E2%80%941896_%D0%B3%D0%BE%D0%B4%D0%B0%D1%85" title="Массовые убийства армян в 1894—1896 годах">убийства 1890-х годов</a>, <a href="/wiki/%D0%A0%D0%B5%D0%B7%D0%BD%D1%8F_%D0%B2_%D0%A1%D0%BC%D0%B8%D1%80%D0%BD%D0%B5" title="Резня в Смирне">резню в Смирне</a> и действия турецких войск в Закавказье в 1918 году (см. раздел <a href="#Османская_империя_и_Республика_Армения_в_1918—1923_годах">Османская империя и Республика Армения в 1918—1923 годах</a>).
</p><p>Основными организаторами геноцида считаются лидеры <a href="/wiki/%D0%9C%D0%BB%D0%B0%D0%B4%D0%BE%D1%82%D1%83%D1%80%D0%BA%D0%B8" title="Младотурки">младотурок</a> <a href="/wiki/%D0%9C%D0%B5%D1%85%D0%BC%D0%B5%D0%B4_%D0%A2%D0%B0%D0%BB%D0%B0%D0%B0%D1%82-%D0%BF%D0%B0%D1%88%D0%B0" title="Мехмед Талаат-паша">Талаат</a>, <a href="/wiki/%D0%90%D1%85%D0%BC%D0%B5%D0%B4_%D0%94%D0%B6%D0%B5%D0%BC%D0%B0%D0%BB%D1%8C-%D0%BF%D0%B0%D1%88%D0%B0" title="Ахмед Джемаль-паша">Джемаль</a> и <a href="/wiki/%D0%AD%D0%BD%D0%B2%D0%B5%D1%80-%D0%BF%D0%B0%D1%88%D0%B0" title="Энвер-паша">Энвер</a>, а также руководитель «<a href="/wiki/%D0%A2%D0%B5%D1%88%D0%BA%D0%B8%D0%BB%D1%8F%D1%82-%D0%B8_%D0%9C%D0%B0%D1%85%D1%81%D1%83%D1%81%D0%B0" title="Тешкилят-и Махсуса">Особой организации</a>» <a href="/wiki/%D0%A8%D0%B0%D0%BA%D0%B8%D1%80,_%D0%91%D0%B5%D1%85%D0%B0%D1%8D%D0%B4%D0%B4%D0%B8%D0%BD" title="Шакир, Бехаэддин">Бехаэддин Шакир</a>. Одновременно с геноцидом армян в Османской империи имели место <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D1%86%D0%B8%D0%B4_%D0%B0%D1%81%D1%81%D0%B8%D1%80%D0%B8%D0%B9%D1%86%D0%B5%D0%B2" title="Геноцид ассирийцев">геноцид ассирийцев</a> и <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D1%86%D0%B8%D0%B4_%D0%BF%D0%BE%D0%BD%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B3%D1%80%D0%B5%D0%BA%D0%BE%D0%B2" title="Геноцид понтийских греков">геноцид понтийских греков</a>. Бо́льшая часть <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B4%D0%B8%D0%B0%D1%81%D0%BF%D0%BE%D1%80%D0%B0" title="Армянская диаспора">армянской диаспоры</a> возникла от бежавших из Османской империи армян.

<ul><li>В <a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Армения">Армении</a>, на территории <a href="/wiki/%D0%90%D1%80%D0%B0%D1%80%D0%B0%D1%82_(%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" class="mw-redirect" title="Арарат (область)">Араратской области</a>, близ городов <a href="/wiki/%D0%92%D0%B5%D0%B4%D0%B8_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Веди (город)">Веди</a> и <a href="/w/index.php?title=%D0%93%D0%BE%D1%80%D0%B0%D0%B2%D0%B0%D0%BD_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)&amp;action=edit&amp;redlink=1" class="new" title="Гораван (город) (страница отсутствует)">Гораван</a> расположена небольшая песчаная <a href="/wiki/%D0%9F%D1%83%D1%81%D1%82%D1%8B%D0%BD%D1%8F" title="Пустыня">пустыня</a>&#160;— <a href="/wiki/%D0%93%D0%BE%D1%80%D0%B0%D0%B2%D0%B0%D0%BD_(%D0%BF%D1%83%D1%81%D1%82%D1%8B%D0%BD%D1%8F)" class="mw-redirect" title="Гораван (пустыня)">Гораванская</a>, которая сосредотачивает характерную для пустыни <a href="/wiki/%D0%AD%D0%BA%D0%BE%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0" title="Экосистема">экосистему</a>.</li></ul>
<ul><li><a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Армения">Армения</a> на своей территории сосредатачивает сразу 6 <a href="/wiki/%D0%9A%D0%BB%D0%B8%D0%BC%D0%B0%D1%82_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8" title="Климат Армении">климатических зон</a>.</li></ul>
<ul><li><a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Армения">Армения</a> занимает одно из первых мест в мире, предположительно первое, по количеству видов растений относительно территории.</li></ul>
<ul><li><a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Армения">Армения</a> является мировым передовиком по запасам <a href="/wiki/%D0%92%D1%83%D0%BB%D0%BA%D0%B0%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%82%D1%83%D1%84" title="Вулканический туф">вулканического туфа</a> и <a href="/wiki/%D0%A4%D0%B5%D0%BB%D1%8C%D0%B7%D0%B8%D1%82" title="Фельзит">фельзита</a>, также рекордсменом на территории стран <a href="/wiki/%D0%A1%D0%9D%D0%93" class="mw-redirect" title="СНГ">СНГ</a> по запасам <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B8%D0%B1%D0%B4%D0%B5%D0%BD" title="Молибден">молибдена</a>, <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%BB%D0%B8%D1%82_(%D0%BC%D0%B5%D1%82%D0%B0%D0%BB%D0%BB%D0%BE%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5)" title="Перлит (металловедение)">перлита</a>.</li></ul>
<ul><li>В <a href="/wiki/%D0%90%D1%80%D1%82%D0%B8%D0%BA" title="Артик">Артикском</a> районе под несколько сантиметровым слоем земли расположены залежи <a href="/wiki/%D0%A2%D1%83%D1%84" title="Туф">туфа</a>.</li></ul>
<ul><li>Подвид <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D0%BE%D1%80%D0%B5%D0%BB%D1%8C" class="mw-redirect" title="Севанская форель">Севанской форели</a>&#160;— Гегаркуни <span lang="la" style="font-style:italic;">(Salmo ischchan gegarkuni)</span> в 1930-х годах был искусственно переселён в пресноводное озеро <a href="/wiki/%D0%98%D1%81%D1%81%D1%8B%D0%BA-%D0%9A%D1%83%D0%BB%D1%8C" title="Иссык-Куль">Иссык-Куль</a>, где создал устойчивую <a href="/wiki/%D0%9F%D0%BE%D0%BF%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Популяция">популяцию</a>.</li></ul>
<ul><li>Один из подвидов <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%84%D0%BE%D1%80%D0%B5%D0%BB%D1%8C" class="mw-redirect" title="Севанская форель">севанской форели</a> (ишхана)&#160;— зимний бахтак <span lang="la" style="font-style:italic;">(Salmo danilewskii)</span>&#160;— является самым большим представителем <a href="/wiki/%D0%A4%D0%BE%D1%80%D0%B5%D0%BB%D1%8C" title="Форель">форелей</a> на территории стран <a href="/wiki/%D0%A1%D0%9D%D0%93" class="mw-redirect" title="СНГ">СНГ</a>.</li></ul>
<ul><li>Во времена бывшего <a href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="СССР">СССР</a> в <a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Армения">Армении</a> строилась <a href="/wiki/%D0%96%D0%B5%D0%BB%D0%B5%D0%B7%D0%BD%D0%B0%D1%8F_%D0%B4%D0%BE%D1%80%D0%BE%D0%B3%D0%B0" class="mw-redirect" title="Железная дорога">железная дорога</a> для <a href="/wiki/%D0%9C%D0%B0%D0%B3%D0%BB%D0%B5%D0%B2" title="Маглев">поездов на магнитной подушке</a>, она должна была соединить через <a href="/wiki/%D0%90%D0%B1%D0%BE%D0%B2%D1%8F%D0%BD_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Абовян (город)">Абовян</a> города <a href="/wiki/%D0%95%D1%80%D0%B5%D0%B2%D0%B0%D0%BD" title="Ереван">Ереван</a> и <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B0%D0%BD_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Севан (город)">Севан</a>, однако землетрясение 1988 года в <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%82%D0%B0%D0%BA" title="Спитак">Спитаке</a> и <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0%D0%B1%D0%B0%D1%85%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Карабахская война">военные события</a> стали причиной замораживания проекта.</li></ul>
<ul><li>Помимо <a href="/wiki/%D0%93%D0%B0%D1%80%D0%B5%D0%B3%D0%B8%D0%BD_II" title="Гарегин II">Католикосата Всех Армян</a>, заседание которого находится в <a href="/wiki/%D0%AD%D1%87%D0%BC%D0%B8%D0%B0%D0%B4%D0%B7%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D1%84%D0%B5%D0%B4%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%81%D0%BE%D0%B1%D0%BE%D1%80" title="Эчмиадзинский кафедральный собор">Эчмиадзинском кафедральном соборе</a>, существует ещё <a href="/wiki/%D0%9A%D0%B8%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D1%82%D0%BE%D0%BB%D0%B8%D0%BA%D0%BE%D1%81%D0%B0%D1%82" class="mw-redirect" title="Киликийский католикосат">Киликийский католикосат</a>, который заседает в <a href="/wiki/%D0%A1%D0%BE%D0%B1%D0%BE%D1%80_%D0%A1%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%93%D1%80%D0%B8%D0%B3%D0%BE%D1%80%D0%B8%D1%8F_%D0%9F%D1%80%D0%BE%D1%81%D0%B2%D0%B5%D1%82%D0%B8%D1%82%D0%B5%D0%BB%D1%8F_(%D0%90%D0%BD%D1%82%D0%B5%D0%BB%D0%B8%D0%B0%D1%81)" title="Собор Святого Григория Просветителя (Антелиас)">Соборе Святого Григория Просветителя</a> (город <a href="/w/index.php?title=%D0%90%D0%BD%D1%82%D0%B5%D0%BB%D0%B8%D0%B0%D1%81&amp;action=edit&amp;redlink=1" class="new" title="Антелиас (страница отсутствует)">Антелиас</a>, <a href="/wiki/%D0%9B%D0%B8%D0%B2%D0%B0%D0%BD" title="Ливан">Ливан</a>).</li></ul>
<ul><li>Резиденция Каталикоса Всех Армян&#160;— <a href="/wiki/%D0%AD%D1%87%D0%BC%D0%B8%D0%B0%D0%B4%D0%B7%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Эчмиадзинский монастырь">Эчмиадзинский монастырь</a>&#160;— находится под Божией защитой и не подвержена воздействию землетрясений. Но этому есть и научное объяснение.</li></ul>
<ul><li>Существует субэтническая группа <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D0%B5" title="Армяне">армян</a>&#160;— <a href="/wiki/%D0%90%D0%BC%D1%88%D0%B5%D0%BD%D1%86%D1%8B" title="Амшенцы">амшенцы</a>.</li></ul>
<ul><li>Один из храмов <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%9A%D1%80%D0%B5%D0%BC%D0%BB%D1%8C" title="Московский Кремль">Московского Кремля</a> носит название «Храм Григория Армянского».</li></ul>

<ul><li>Одним из немногих людей, успешно прошедших тест <a href="/wiki/%D0%A5%D1%8F%D0%BA%D1%83%D0%BD%D0%B8%D0%BD-%D0%BA%D1%83%D0%BC%D0%B8%D1%82%D1%8D" class="mw-redirect" title="Хякунин-кумитэ">хякунин-кумитэ</a> (100 боёв в <a href="/wiki/%D0%9A%D1%91%D0%BA%D1%83%D1%81%D0%B8%D0%BD%D0%BA%D0%B0%D0%B9" title="Кёкусинкай">кёкусинкай</a>-карате), является <b><a href="/wiki/%D0%9E%D0%B3%D0%B0%D0%BD%D0%B5%D1%81%D1%8F%D0%BD,_%D0%90%D1%80%D1%82%D1%83%D1%80_%D0%AD%D0%B4%D1%83%D0%B0%D1%80%D0%B4%D0%BE%D0%B2%D0%B8%D1%87" title="Оганесян, Артур Эдуардович">Артур Оганесян</a></b>.</li></ul>

<ul><li>В ответ на установленные в мире памятники <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D1%86%D0%B8%D0%B4_%D0%B0%D1%80%D0%BC%D1%8F%D0%BD" title="Геноцид армян">Геноциду армян</a>, в <a href="/wiki/%D0%A2%D1%83%D1%80%D1%86%D0%B8%D1%8F" title="Турция">Турции</a> был установлен <b><a href="/wiki/%D0%9F%D0%B0%D0%BC%D1%8F%D1%82%D0%BD%D0%B8%D0%BA_%D0%B8_%D0%BC%D1%83%D0%B7%D0%B5%D0%B9_%D0%B3%D0%B5%D0%BD%D0%BE%D1%86%D0%B8%D0%B4%D0%B0_%D0%B2_%D0%AB%D0%B3%D0%B4%D1%8B%D1%80%D0%B5" title="Памятник и музей геноцида в Ыгдыре">памятик геноциду турок армянами</a></b>.</li></ul>
<ul><li><a href="/wiki/%D0%A2%D1%83%D1%80%D1%86%D0%B8%D1%8F" title="Турция">Турция</a> в рамках борьбы с «неправильными названиями» <b><a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D0%BE%D1%84%D0%BE%D0%B1%D0%B8%D1%8F#Другое" title="Армянофобия">переименовала</a></b> <a href="/w/index.php?title=%D0%94%D0%B8%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B2%D1%86%D0%B0&amp;action=edit&amp;redlink=1" class="new" title="Дикая армянская овца (страница отсутствует)">дикую армянскую овцу</a> (Ovis armeniana) на «восточноанатолийскую овцу» (Ovis Orientalis Anatolicus), а <a href="/w/index.php?title=%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D0%BE%D1%81%D1%83%D0%BB%D1%8F&amp;action=edit&amp;redlink=1" class="new" title="Армянская косуля (страница отсутствует)">армянскую косулю</a> (Capreolus Capreolus Armenus) на «косулю медноцветную».</li></ul>
<ul><li>Расположенное в <a href="/wiki/%D0%90%D0%B7%D0%B5%D1%80%D0%B1%D0%B0%D0%B9%D0%B4%D0%B6%D0%B0%D0%BD" title="Азербайджан">Азербайджане</a> село <b><a href="/wiki/%D0%A7%D0%B0%D1%80%D0%B4%D0%B0%D1%85%D0%BB%D1%8B" class="mw-redirect" title="Чардахлы">Чардахлы</a></b>&#160;— родина военачальников-армян: <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%88%D0%B0%D0%BB_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%A1%D0%BE%D1%8E%D0%B7%D0%B0" title="Маршал Советского Союза">Маршала Советского Союза</a> <a href="/wiki/%D0%91%D0%B0%D0%B3%D1%80%D0%B0%D0%BC%D1%8F%D0%BD,_%D0%98%D0%B2%D0%B0%D0%BD_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%BE%D1%84%D0%BE%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Баграмян, Иван Христофорович">И.&#160;Х.&#160;Баграмяна</a>, <a href="/wiki/%D0%93%D0%BB%D0%B0%D0%B2%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B0%D1%80%D1%88%D0%B0%D0%BB_%D1%80%D0%BE%D0%B4%D0%B0_%D0%B2%D0%BE%D0%B9%D1%81%D0%BA" title="Главный маршал рода войск">Главного маршала</a> <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2%D1%8B%D0%B5_%D0%B2%D0%BE%D0%B9%D1%81%D0%BA%D0%B0" title="Танковые войска">бронетанковых войск</a> <a href="/wiki/%D0%91%D0%B0%D0%B1%D0%B0%D0%B4%D0%B6%D0%B0%D0%BD%D1%8F%D0%BD,_%D0%90%D0%BC%D0%B0%D0%B7%D0%B0%D1%81%D0%BF_%D0%A5%D0%B0%D1%87%D0%B0%D1%82%D1%83%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Бабаджанян, Амазасп Хачатурович">А.&#160;Х.&#160;Бабаджаняна</a>, двенадцати <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB" title="Генерал">генералов</a> и семи <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BE%D0%B9_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%A1%D0%BE%D1%8E%D0%B7%D0%B0" title="Герой Советского Союза">Героев Советского Союза</a>.</li></ul>
<ul><li>После <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">Первой мировой войны</a> рассматривался вариант <b><a href="/wiki/%D0%92%D0%B8%D0%BB%D1%8C%D1%81%D0%BE%D0%BD%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Вильсоновская Армения">передачи Армении под управление США</a></b>.</li></ul>
<h3><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span></h3>
<p>Наиболее вероятная область происхождения культурной пшеницы расположена на территории <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D1%8F%D1%8F_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" class="mw-redirect" title="Древняя Армения">исторической Армении</a>, близ современного города <a href="/wiki/%D0%94%D0%B8%D1%8F%D1%80%D0%B1%D0%B0%D0%BA%D1%8B%D1%80" title="Диярбакыр">Диярбакыр</a> в юго-восточной Турции.
</p>
<ul><li>Средневековый город <a href="/wiki/%D0%90%D0%BD%D0%B8_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Ани (город)">Ани</a> в XIII веке был разграблен войсками монголов, а теперь стоит в <a href="/wiki/%D0%A0%D1%83%D0%B8%D0%BD%D1%8B" title="Руины">руинах</a>.</li></ul>
<ul><li>Опера армянского композитора <a href="/wiki/%D0%A2%D0%B8%D0%B3%D1%80%D0%B0%D0%BD_%D0%A7%D1%83%D1%85%D0%B0%D0%B4%D0%B6%D1%8F%D0%BD" class="mw-redirect" title="Тигран Чухаджян">Тиграна Чухаджяна</a> <b>«<a href="/wiki/%D0%90%D1%80%D1%88%D0%B0%D0%BA_II_(%D0%BE%D0%BF%D0%B5%D1%80%D0%B0)" title="Аршак II (опера)">Аршак II</a>»</b> являлось первой оперой в музыкальной истории всего <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA_(%D0%BC%D0%B0%D0%BA%D1%80%D0%BE%D1%80%D0%B5%D0%B3%D0%B8%D0%BE%D0%BD)" title="Восток (макрорегион)">Востока</a>.</li></ul>
<ul><li>Согласно <a href="/w/index.php?title=%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%8F_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8_(%D0%BA%D0%BD%D0%B8%D0%B3%D0%B0)&amp;action=edit&amp;redlink=1" class="new" title="История Армении (книга) (страница отсутствует)">«Истории Армении»</a>, потомком <a href="/wiki/%D0%98%D0%B0%D1%84%D0%B5%D1%82" title="Иафет">Иафета</a>, сына <a href="/wiki/%D0%9D%D0%BE%D0%B9" title="Ной">Ноя</a> был <a href="/wiki/%D0%A5%D0%B0%D0%B9%D0%BA" class="mw-redirect" title="Хайк">Хайк</a>, а одним из сыновей Хайка&#160;— <a href="/w/index.php?title=%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B0%D0%BA&amp;action=edit&amp;redlink=1" class="new" title="Арменак (страница отсутствует)">Арменак</a>.</li></ul>
<ul><li>В <a href="/wiki/%D0%A2%D1%83%D1%80%D1%86%D0%B8%D1%8F" title="Турция">Турции</a> сохранилась только одна <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D0%B5" title="Армяне">армянская</a> <b><a href="/wiki/%D0%92%D0%B0%D0%BA%D0%B8%D1%84%D0%BB%D0%B8" class="mw-redirect" title="Вакифли">деревня</a></b>.</li></ul>
