<div style="display:block; border:1px solid #FF0000; vertical-align:top; background:#ffffff; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">        
<div class="floatleft"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Belgium_stub.svg" class="image"><img alt="Belgium stub.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Belgium_stub.svg/140px-Belgium_stub.svg.png" decoding="async" width="140" height="109" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Belgium_stub.svg/210px-Belgium_stub.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Belgium_stub.svg/280px-Belgium_stub.svg.png 2x" data-file-width="930" data-file-height="726" /></a></div><b><a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгия</a></b> или <b>Королевство Бельгия</b> — <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a> в Северо-Западной Европе, член <a href="/wiki/%D0%95%D0%A1" class="mw-redirect" title="ЕС">ЕС</a>, <a href="/wiki/%D0%9E%D0%9E%D0%9D" class="mw-redirect" title="ООН">ООН</a> и <a href="/wiki/%D0%9D%D0%90%D0%A2%D0%9E" title="НАТО">НАТО</a>. Площадь — 30 528 км², численность населения — 11 млн чел. Граничит на севере с <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Нидерланды">Нидерландами</a>, на востоке с <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">Германией</a>, юго-востоке с <a href="/wiki/%D0%9B%D1%8E%D0%BA%D1%81%D0%B5%D0%BC%D0%B1%D1%83%D1%80%D0%B3" title="Люксембург">Люксембургом</a> и с <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Францией</a> на юге и западе. Омывается <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Северное море">Северным морем</a> на северо-западе.
<p>Форма правления — <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BC%D0%BE%D0%BD%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Конституционная монархия">конституционная монархия</a>. Форма административно-территориального устройства — <a href="/wiki/%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" class="mw-redirect" title="Федерация">федерация</a>. Бельгия разделена на три региона: <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%BC%D0%B0%D0%BD%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B5%D0%B3%D0%B8%D0%BE%D0%BD" title="Фламандский регион">Фламандский регион</a>, <a href="/wiki/%D0%92%D0%B0%D0%BB%D0%BB%D0%BE%D0%BD%D0%B8%D1%8F" title="Валлония">Валлонский регион</a> и <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%BD%D1%8B%D0%B9_%D1%80%D0%B5%D0%B3%D0%B8%D0%BE%D0%BD" title="Брюссельский столичный регион">Брюссельский столичный регион</a>, также, параллельно разделена ещё и на три языковых сообщества: <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%BE%D0%B5_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B8" title="Французское сообщество Бельгии">французское</a>, <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%BC%D0%B0%D0%BD%D0%B4%D1%81%D0%BA%D0%BE%D0%B5_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Фламандское сообщество">фламандское</a> и <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%BE%D1%8F%D0%B7%D1%8B%D1%87%D0%BD%D0%BE%D0%B5_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B8" title="Немецкоязычное сообщество Бельгии">немецкоязычное</a>, они же определяют три <a href="/wiki/%D0%9E%D1%84%D0%B8%D1%86%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Официальный язык">официальных языка</a> Бельгии. Столицей государства является <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C" title="Брюссель">Брюссель</a>.

<p><b><a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D1%85%D0%BD%D1%8F" title="Бельгийская кухня">Бельгийская кухня</a></b> — национальная кухня <a href="/wiki/%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D1%8F" title="Бельгия">Бельгии</a>. Бельгийская кухня сочетает в себе средневековые кулинарные традиции с традициями соседних стран — <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a>, <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">Германии</a>, <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Нидерланды">Голландии</a>. В то время как домашние блюда более простые и сытные, как в <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D1%85%D0%BD%D1%8F" title="Немецкая кухня">немецкой</a> или <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D1%85%D0%BD%D1%8F" title="Нидерландская кухня">нидерландской кухнях</a>, на <a href="/wiki/%D0%A0%D0%B5%D1%81%D1%82%D0%BE%D1%80%D0%B0%D0%BD" title="Ресторан">ресторанную</a> кухню большое влияние оказала <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B0%D1%8F_%D0%BA%D1%83%D1%85%D0%BD%D1%8F" title="Французская кухня">французская кухня</a>.
</p><p>Бельгийцы предпочитают использовать сезонные продукты и продукты из региона, поэтому даже в таком небольшом королевстве заметны различия в региональных кухнях. В прибрежных регионах преобладают блюда из <a href="/wiki/%D0%A0%D1%8B%D0%B1%D0%B0" class="mw-redirect" title="Рыба">рыбы</a> и <a href="/wiki/%D0%9C%D0%BE%D1%80%D0%B5%D0%BF%D1%80%D0%BE%D0%B4%D1%83%D0%BA%D1%82%D1%8B" title="Морепродукты">морепродуктов</a>, в <a href="/wiki/%D0%90%D1%80%D0%B4%D0%B5%D0%BD%D0%BD%D1%8B_(%D0%B3%D0%BE%D1%80%D1%8B)" title="Арденны (горы)">Арденнах</a> чаще используют <a href="/wiki/%D0%94%D0%B8%D1%87%D1%8C" title="Дичь">дичь</a>. Помимо этого с давних пор конфликтующие между собой языковые сообщества — <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%BC%D0%B0%D0%BD%D0%B4%D1%81%D0%BA%D0%BE%D0%B5_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" title="Фламандское сообщество">Фламандское</a> и <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%BE%D0%B5_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B8" title="Французское сообщество Бельгии">Валлонское</a> — также имеют каждое свою региональную кухню.
</p><p>Бельгийская кухня знаменита в первую очередь <a href="/wiki/%D0%A8%D0%BE%D0%BA%D0%BE%D0%BB%D0%B0%D0%B4" title="Шоколад">шоколадом</a>, <a href="/wiki/%D0%92%D0%B0%D1%84%D0%BB%D1%8F" title="Вафля">вафлями</a>, <a href="/wiki/%D0%9A%D0%B0%D1%80%D1%82%D0%BE%D1%84%D0%B5%D0%BB%D1%8C_%D1%84%D1%80%D0%B8" title="Картофель фри">картофелем фри</a> и <a href="/wiki/%D0%9F%D0%B8%D0%B2%D0%BE" title="Пиво">пивом</a>.

<p><b><a href="/wiki/%D0%90%D0%BB%D1%8C%D0%B1%D0%B5%D1%80%D1%82_I_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B8)" title="Альберт I (король Бельгии)">Альберт I</a></b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">Albert I</span>; <a href="/wiki/8_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="8 апреля">8 апреля</a> <a href="/wiki/1875" class="mw-redirect" title="1875">1875</a>, <a href="/wiki/%D0%91%D1%80%D1%8E%D1%81%D1%81%D0%B5%D0%BB%D1%8C" title="Брюссель">Брюссель</a> — <a href="/wiki/17_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="17 февраля">17 февраля</a> <a href="/wiki/1934" class="mw-redirect" title="1934">1934</a>, близ Марш-ле-Дам), <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B8_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B8" class="mw-redirect" title="Короли Бельгии">король бельгийцев</a> с <a href="/wiki/17_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="17 декабря">17 декабря</a> <a href="/wiki/1909" class="mw-redirect" title="1909">1909</a>. Из <a href="/wiki/%D0%A1%D0%B0%D0%BA%D1%81%D0%B5%D0%BD-%D0%9A%D0%BE%D0%B1%D1%83%D1%80%D0%B3-%D0%93%D0%BE%D1%82%D1%81%D0%BA%D0%B0%D1%8F_%D0%B4%D0%B8%D0%BD%D0%B0%D1%81%D1%82%D0%B8%D1%8F" title="Саксен-Кобург-Готская династия">Саксен-Кобург-Готской династии</a>. Сын графа <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9,_%D0%B3%D1%80%D0%B0%D1%84_%D0%A4%D0%BB%D0%B0%D0%BD%D0%B4%D1%80%D1%81%D0%BA%D0%B8%D0%B9" class="mw-redirect" title="Филипп Бельгийский, граф Фландрский">Филиппа Фландрского</a> и принцессы Марии Гогенцоллерн-Зигмаринген, племянник бельгийского короля <a href="/wiki/%D0%9B%D0%B5%D0%BE%D0%BF%D0%BE%D0%BB%D1%8C%D0%B4_II_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%91%D0%B5%D0%BB%D1%8C%D0%B3%D0%B8%D0%B8)" title="Леопольд II (король Бельгии)">Леопольда II</a>.
</p><p>В <a href="/wiki/1891_%D0%B3%D0%BE%D0%B4" title="1891 год">1891 году</a> после смерти своего старшего брата Бодуэна объявлен наследником престола. Получил частное образование; окончил в <a href="/wiki/1892_%D0%B3%D0%BE%D0%B4" title="1892 год">1892 году</a> Королевскую военную школу. До вступления на престол он носил титул <a href="/wiki/%D0%93%D1%80%D0%B0%D1%84_(%D1%82%D0%B8%D1%82%D1%83%D0%BB)" title="Граф (титул)">графа</a> <a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%BD%D0%B4%D1%80%D0%B8%D1%8F_(%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Фландрия (историческая область)">Фландрского</a>. 2 октября 1900 женился на герцогине Елизавете Баварской, дочери <a href="/wiki/%D0%93%D0%B5%D1%80%D1%86%D0%BE%D0%B3" title="Герцог">герцога</a> Карла Фридриха.
