<p><b>Белгоро́дская о́бласть</b>&#160;(неофиц. <i>Белого́рье, Святое Белого́рье, Белгоро́дчина, Белгоро́дщина</i>) — <a href="/wiki/%D0%90%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE-%D1%82%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" class="mw-redirect" title="Административно-территориальное деление России">субъект</a> <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" class="mw-redirect" title="Российская Федерация">Российской Федерации</a>, расположен в юго-западной части России в 500—700&#160;км к югу от <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москвы</a>, на границе с <a href="/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0" title="Украина">Украиной</a>. Крупнейшие города: <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Белгород">Белгород</a>&#160;— 384 425 чел., <a href="/wiki/%D0%A1%D1%82%D0%B0%D1%80%D1%8B%D0%B9_%D0%9E%D1%81%D0%BA%D0%BE%D0%BB" title="Старый Оскол">Старый Оскол</a> — 221 254 чел., <a href="/wiki/%D0%93%D1%83%D0%B1%D0%BA%D0%B8%D0%BD_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" title="Губкин (город)">Губкин</a> — 87 405 чел.
</p><p>Белгородская область образована <a href="/wiki/6_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="6 января">6 января</a> <a href="/wiki/1954_%D0%B3%D0%BE%D0%B4" title="1954 год">1954 года</a>. С этого момента её границы не изменялись.
</p><p><i>Областной центр</i>&#160;— <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B3%D0%BE%D1%80%D0%BE%D0%B4" title="Белгород">Белгород</a>.

<p><b>Ала́ны</b> (<a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B3%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Древнегреческий язык">др.-греч.</a> <span lang="grc"><span style="font-family: palatino linotype, new athena unicode, athena, gentium, code2000, serif; font-size: 105%;">Ἀλανοί</span></span>, <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a>&#160;<span lang="la" style="font-style:italic;">Alani, Halani</span>) — кочевые <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B5_%D1%8F%D0%B7%D1%8B%D0%BA%D0%B8" title="Иранские языки">ираноязычные</a> племена <a href="/wiki/%D0%A1%D0%BA%D0%B8%D1%84%D1%8B" title="Скифы">скифо</a>-<a href="/wiki/%D0%A1%D0%B0%D1%80%D0%BC%D0%B0%D1%82%D1%8B" title="Сарматы">сарматского</a> происхождения, в письменных источниках упоминаются с <a href="/wiki/I_%D0%B2%D0%B5%D0%BA" title="I век">I века</a> н. э. — времени их появления в <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%B0%D0%B7%D0%BE%D0%B2%D1%8C%D0%B5" class="mw-redirect" title="Приазовье">Приазовье</a> и <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B4%D0%BA%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7%D1%8C%D0%B5" title="Предкавказье">Предкавказье</a>.
</p><p>Часть аланов с конца <a href="/wiki/IV_%D0%B2%D0%B5%D0%BA" title="IV век">IV века</a> приняла участие в <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B5_%D0%BF%D0%B5%D1%80%D0%B5%D1%81%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BE%D0%B2" title="Великое переселение народов">Великом переселении народов</a>, в то время как другие остались на территориях, прилегающих к предгорьям <a href="/wiki/%D0%9A%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7" title="Кавказ">Кавказа</a>. Племенной союз аланов стал основой объединения аланских и местных <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%BE%D0%B4%D1%8B_%D0%9A%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7%D0%B0" title="Народы Кавказа">кавказских племён</a>, известного под названием <a href="/wiki/%D0%90%D0%BB%D0%B0%D0%BD%D0%B8%D1%8F" title="Алания">Алания</a>, и образования в центральном Предкавказье раннефеодального государства, просуществовавшего до похода монголов.
</p><p>Татаро-монголы, разгромившие Аланию и захватившие к концу 1230-х годов плодородные равнинные районы Предкавказья, вынудили уцелевших аланов укрыться в горах Центрального Кавказа и в Закавказье. Там одна из групп аланов, при участии местных племён, дала начало современным <a href="/wiki/%D0%9E%D1%81%D0%B5%D1%82%D0%B8%D0%BD%D1%8B" title="Осетины">осетинам</a>.
</p><p>(<i><a href="/wiki/%D0%90%D0%BB%D0%B0%D0%BD%D1%8B" title="Аланы">далее…</a></i>)

<div style="display:block; border:1px solid LightBlue; vertical-align:top; background:white; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">        
<p><b>Васи́лий Я́ковлевич Го́рин</b> (<a href="/wiki/9_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="9 января">9 января</a> <a href="/wiki/1922" class="mw-redirect" title="1922">1922</a>, с. <a href="/wiki/%D0%91%D0%B5%D1%81%D1%81%D0%BE%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0_(%D0%91%D0%B5%D0%BB%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Бессоновка (Белгородская область)">Бессоновка</a>, <a href="/wiki/%D0%9A%D1%83%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Курская губерния">Курская губерния</a>&#160;— <a href="/wiki/4_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="4 апреля">4 апреля</a> <a href="/wiki/2014" class="mw-redirect" title="2014">2014</a>, с. Бессоновка, <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Белгородская область">Белгородская область</a>)&#160;— организатор колхозного производства, председатель ордена Трудового Красного Знамени колхоза имени Фрунзе Белгородской области, дважды <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BE%D0%B9_%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%A2%D1%80%D1%83%D0%B4%D0%B0" title="Герой Социалистического Труда">Герой Социалистического Труда</a> (<a href="/wiki/1971" class="mw-redirect" title="1971">1971</a>, <a href="/wiki/1985" class="mw-redirect" title="1985">1985</a>), почётный гражданин Белгородской области.
</p><p>Родился <a href="/wiki/9_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="9 января">9 января</a> <a href="/wiki/1922_%D0%B3%D0%BE%D0%B4" title="1922 год">1922&#160;года</a> в селе <a href="/wiki/%D0%91%D0%B5%D1%81%D1%81%D0%BE%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0_(%D0%91%D0%B5%D0%BB%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Бессоновка (Белгородская область)">Бессоновка</a> <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%B5%D0%B7%D0%B4" title="Белгородский уезд">Белгородского уезда</a> <a href="/wiki/%D0%9A%D1%83%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Курская губерния">Курской губернии</a> (ныне <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Белгородский район">Белгородского района</a>, Белгородская область) в семье кузнеца.
</p><p>После окончания средней школы поступил в Харьковское военно-медицинское училище, которое окончил летом <a href="/wiki/1941_%D0%B3%D0%BE%D0%B4" title="1941 год">1941&#160;года</a>.
</p><p>Участник <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Великая Отечественная война">Великой Отечественной войны</a>, сражался на Западном и Калининском фронтах. После ранения попал в плен, а когда лагерь освободили, продолжал службу в Красной Армии в качестве <a href="/wiki/%D0%A4%D0%B5%D0%BB%D1%8C%D0%B4%D1%88%D0%B5%D1%80" title="Фельдшер">фельдшера</a>.
</p><p>В декабре <a href="/wiki/1945_%D0%B3%D0%BE%D0%B4" title="1945 год">1945&#160;года</a> демобилизовался и работал заведующим Черемошанским, а затем Краснооктябрьским, Бессоновским фельдшерско-акушерскими пунктами. Принимал активнейшее участие в работе сельсоветов. Член <a href="/wiki/%D0%9A%D0%9F%D0%A1%D0%A1" class="mw-redirect" title="КПСС">КПСС</a> с <a href="/wiki/1958_%D0%B3%D0%BE%D0%B4" title="1958 год">1958&#160;года</a>.
