<p><b>Тиро́ль</b> (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a>&#160;<span lang="de" style="font-style:italic;">Tirol</span>; <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a>&#160;<span lang="it" style="font-style:italic;">Tirolo</span>; <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a>&#160;<span lang="la" style="font-style:italic;">Tirolis</span>)&#160;— <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Историческая область">историческая область</a> в <a href="/wiki/%D0%A6%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Центральная Европа">Центральной Европе</a> в восточной части <a href="/wiki/%D0%90%D0%BB%D1%8C%D0%BF%D1%8B" title="Альпы">Альпийских гор</a>, включающая федеральную землю <a href="/wiki/%D0%A2%D0%B8%D1%80%D0%BE%D0%BB%D1%8C_(%D0%B7%D0%B5%D0%BC%D0%BB%D1%8F)" title="Тироль (земля)">Тироль</a> в составе <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B8%D1%8F" title="Австрия">Австрии</a> и автономные провинции <a href="/wiki/%D0%AE%D0%B6%D0%BD%D1%8B%D0%B9_%D0%A2%D0%B8%D1%80%D0%BE%D0%BB%D1%8C" class="mw-redirect" title="Южный Тироль">Южный Тироль</a> и <a href="/wiki/%D0%A2%D1%80%D0%B5%D0%BD%D1%82%D0%BE_(%D0%BF%D1%80%D0%BE%D0%B2%D0%B8%D0%BD%D1%86%D0%B8%D1%8F)" title="Тренто (провинция)">Трентино</a> в <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италии</a>.
</p><p>Тироль населён со времён мезолита, однако особенно массовый приток населения начался в эпоху неолита. Тирольские земли вошли в состав <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B9_%D0%A0%D0%B8%D0%BC" title="Древний Рим">Римской империи</a> в 15 году до&#160;н.&#160;э. в результате походов <a href="/wiki/%D0%94%D0%B5%D1%86%D0%B8%D0%BC_%D0%9A%D0%BB%D0%B0%D0%B2%D0%B4%D0%B8%D0%B9_%D0%9D%D0%B5%D1%80%D0%BE%D0%BD" class="mw-redirect" title="Децим Клавдий Нерон">Друза</a> и <a href="/wiki/%D0%A2%D0%B8%D0%B1%D0%B5%D1%80%D0%B8%D0%B9" title="Тиберий">Тиберия</a>. В период римского господства Тироль оставался достаточно слабо затронут цивилизацией из-за горного характера региона. Однако и здесь были основаны римские поселения: Тридентум (<a href="/wiki/%D0%A2%D1%80%D0%B5%D0%BD%D1%82%D0%BE" title="Тренто">Тренто</a>), Энипонс (<a href="/wiki/%D0%98%D0%BD%D1%81%D0%B1%D1%80%D1%83%D0%BA" title="Инсбрук">Инсбрук</a>). Большое значение имела дорога через <a href="/wiki/%D0%91%D1%80%D0%B5%D0%BD%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D0%B2%D0%B0%D0%BB" class="mw-redirect" title="Бреннерский перевал">Бреннерский перевал</a> из <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италии</a> в долину <a href="/wiki/%D0%94%D1%83%D0%BD%D0%B0%D0%B9" title="Дунай">Дуная</a> и Августу Винделикорум (<a href="/wiki/%D0%90%D1%83%D0%B3%D1%81%D0%B1%D1%83%D1%80%D0%B3" title="Аугсбург">Аугсбург</a>). В этот период в Тироле началась разработка месторождений металлов в Альпах. В 233&#160;году в Рецию вторглись <a href="/wiki/%D0%90%D0%BB%D0%B5%D0%BC%D0%B0%D0%BD%D0%BD%D1%8B" title="Алеманны">алеманны</a>, которые расселились на правах <a href="/wiki/%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%82%D1%8B_(%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B9_%D0%A0%D0%B8%D0%BC)" title="Федераты (Древний Рим)">федератов</a> в северной части Тироля. В IV веке на территории Тироля начало распространяться христианство; тогда же были основаны епископства. В <a href="/wiki/IV_%D0%B2%D0%B5%D0%BA" title="IV век">IV веке</a> войны римлян с <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B5_%D0%B3%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D1%86%D1%8B" class="mw-redirect" title="Древние германцы">германцами</a> возобновились. Алеманны разорили Рецию, а после падения в 476&#160;году Римской империи Тироль был включен сначала в государство <a href="/wiki/%D0%9E%D0%B4%D0%BE%D0%B0%D0%BA%D1%80" title="Одоакр">Одоакра</a>, а с 493&#160;года&#160;— <a href="/wiki/%D0%9E%D1%81%D1%82%D0%B3%D0%BE%D1%82%D1%8B" title="Остготы">Остготского королевства</a> <a href="/wiki/%D0%A2%D0%B5%D0%BE%D0%B4%D0%BE%D1%80%D0%B8%D1%85_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" title="Теодорих Великий">Теодориха Великого</a>. С середины <a href="/wiki/VI_%D0%B2%D0%B5%D0%BA" title="VI век">VI века</a> Тироль перешёл под власть <a href="/wiki/%D0%9B%D0%B0%D0%BD%D0%B3%D0%BE%D0%B1%D0%B0%D1%80%D0%B4%D1%8B" title="Лангобарды">лангобардов</a>. Однако власть этих государств в регионе была крайне слаба. Этим воспользовались <a href="/wiki/%D0%91%D0%B0%D0%B2%D0%B0%D1%80%D1%8B" title="Бавары">бавары</a> и отчасти славяне (<a href="/wiki/%D0%A1%D0%BB%D0%BE%D0%B2%D0%B5%D0%BD%D1%86%D1%8B" title="Словенцы">словенцы</a>), которые заселили север и север-восток Тироля, однако его южная часть оставалась под властью лангобардов.

<p>Герцогиня <b><a href="/wiki/%D0%95%D0%BB%D0%B8%D0%B7%D0%B0%D0%B2%D0%B5%D1%82%D0%B0_%D0%91%D0%B0%D0%B2%D0%B0%D1%80%D1%81%D0%BA%D0%B0%D1%8F_(%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B0%D1%82%D1%80%D0%B8%D1%86%D0%B0_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B8%D0%B8)" title="Елизавета Баварская (императрица Австрии)">Амалия Евгения Елизавета Баварская</a></b>  (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a>&#160;<span lang="de" style="font-style:italic;">Elisabeth Amalie Eugenie, Herzogin in Bayern</span>; <a href="/wiki/24_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="24 декабря">24 декабря</a> <a href="/wiki/1837_%D0%B3%D0%BE%D0%B4" title="1837 год">1837 года</a>&#160;— <a href="/wiki/10_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="10 сентября">10 сентября</a> <a href="/wiki/1898_%D0%B3%D0%BE%D0%B4" title="1898 год">1898 года</a>)&#160;— баварская принцесса, супруга <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86_%D0%98%D0%BE%D1%81%D0%B8%D1%84_I" title="Франц Иосиф I">императора Франца Иосифа I</a>. Императрица <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B8%D1%8F" title="Австрия">Австрии</a> с <a href="/wiki/24_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="24 апреля">24 апреля</a> <a href="/wiki/1854_%D0%B3%D0%BE%D0%B4" title="1854 год">1854 года</a> (дня заключения брака), королева-консорт <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B3%D1%80%D0%B8%D1%8F" title="Венгрия">Венгрии</a> с <a href="/wiki/8_%D0%B8%D1%8E%D0%BD%D1%8F" title="8 июня">8 июня</a> <a href="/wiki/1867_%D0%B3%D0%BE%D0%B4" title="1867 год">1867 года</a> (дня образования двуединой монархии <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%BE-%D0%92%D0%B5%D0%BD%D0%B3%D1%80%D0%B8%D1%8F" title="Австро-Венгрия">Австро-Венгрии</a>). Известна под уменьшительно-ласкательным именем <b>Сиси</b> (<a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Немецкий язык">нем.</a>&#160;<span lang="de" style="font-style:italic;">Sissi</span>), которым её называли родные и друзья (в художественной литературе и кино употребляется вариант написания <i>Сисси</i>).
