<p><b>Мужская сборная Болгарии по волейболу</b> — национальная команда, представляющая <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D0%B8%D1%8F" title="Болгария">Болгарию</a> на международных соревнованиях по <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB" title="Волейбол">волейболу</a>.
</p><p>Первый выход на международную арену болгарских волейболистов состоялся в 1934 году, когда команда Американского колледжа из <a href="/wiki/%D0%A1%D0%BE%D1%84%D0%B8%D1%8F" title="София">Софии</a> сыграли в гостях с командой Колледжа Роберта из <a href="/wiki/%D0%A1%D1%82%D0%B0%D0%BC%D0%B1%D1%83%D0%BB" title="Стамбул">Царьграда</a> и уступила в трёх партиях. В <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D0%B8%D1%8F" title="Болгария">Болгарии</a> же первый международный товарищеский матч прошёл 22 июня 1934 года против сборной <a href="/wiki/%D0%A0%D1%83%D0%BC%D1%8B%D0%BD%D0%B8%D1%8F" title="Румыния">Румынии</a> во время проведения футбольного Кубка Балкан в <a href="/wiki/%D0%A1%D0%BE%D1%84%D0%B8%D1%8F" title="София">Софии</a> и завершился победой болгар — 3:0.
</p><p>На первом же <a href="/wiki/%D0%A7%D0%B5%D0%BC%D0%BF%D0%B8%D0%BE%D0%BD%D0%B0%D1%82_%D0%BC%D0%B8%D1%80%D0%B0_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83_%D1%81%D1%80%D0%B5%D0%B4%D0%B8_%D0%BC%D1%83%D0%B6%D1%87%D0%B8%D0%BD_1949" title="Чемпионат мира по волейболу среди мужчин 1949">чемпионате мира среди мужчин в Чехословакии в 1949 году</a> болгары завоевали бронзовые медали, уступив лишь сборным <a href="/wiki/%D0%9C%D1%83%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%A1%D0%A1%D0%A1%D0%A0_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83" title="Мужская сборная СССР по волейболу">СССР</a> и Чехословакии. Тогда в составе команды выступали такие игроки, как Костадин Шопов, Борис Гюдеров, Митко Димитров, Драгомир Стоянов, Тодор Симов, Георгий Коматов, Панайот Пондалов, Гёко Петров, Боян Мошелов, Коста Баджаков, Стойчо Карджиев, Борис Владимиров под руководством главного тренера Валентина Анкова.
</p><p>На Олимпиаде болгары впервые выступили в <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB_%D0%BD%D0%B0_%D0%BB%D0%B5%D1%82%D0%BD%D0%B8%D1%85_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85_1964" title="Волейбол на летних Олимпийских играх 1964">1964 году в Токио</a> и заняли 5-е место. Одним из самых значительных достижений болгарской сборной стало «серебро» <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB_%D0%BD%D0%B0_%D0%BB%D0%B5%D1%82%D0%BD%D0%B8%D1%85_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85_1980" title="Волейбол на летних Олимпийских играх 1980">Олимпийских игр в 1980 году в Москве</a>. Тогда под знамёна олимпийской сборной были призваны Стоян Гунчев, Христо Стоянов, Димитр Златанов, Димитр Димитров, Стефан Димитров, Йордан Ангелов, Христо Илиев, Петко Петков, Каспар Симеонов, Эмил Вылчев, Митко Тодоров и Цано Цанов. <small><i>(<a href="/wiki/%D0%9C%D1%83%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D0%B8%D0%B8_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83" title="Мужская сборная Болгарии по волейболу">Подробнее</a>)</i></small>
</p>

<div style="display:block; border:1px solid #699; vertical-align:top; background:white; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<ul><li>Именем <a href="/wiki/%D0%9C%D0%BE%D1%80%D0%B8%D1%82%D0%B0,_%D0%94%D0%B7%D1%8E%D0%BD%D0%B3%D0%BE" title="Морита, Дзюнго">олимпийского чемпиона 1972 года</a> названа комбинация в <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB" title="Волейбол">волейболе</a>.</li>
<li>Первый волейбольный матч состоялся в <a href="/wiki/1896_%D0%B3%D0%BE%D0%B4" title="1896 год">1896 году</a> в <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BB%D0%B5%D0%B4%D0%B6" title="Колледж">колледже</a> <a href="/wiki/%D0%A1%D0%BF%D1%80%D0%B8%D0%BD%D0%B3%D1%84%D0%B8%D0%BB%D0%B4_(%D0%9C%D0%B0%D1%81%D1%81%D0%B0%D1%87%D1%83%D1%81%D0%B5%D1%82%D1%81)" title="Спрингфилд (Массачусетс)">Спрингфилда</a> в <a href="/wiki/%D0%A1%D0%BE%D0%B5%D0%B4%D0%B8%D0%BD%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%A8%D1%82%D0%B0%D1%82%D1%8B_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8" title="Соединённые Штаты Америки">США</a>.</li>
<li><a href="/wiki/%D0%A1%D0%BA%D0%BE%D1%80%D0%BE%D1%81%D1%82%D1%8C" title="Скорость">Скорость</a> полёта <a href="/wiki/%D0%9C%D1%8F%D1%87" title="Мяч">мяча</a> после удара может превысить 130 километров в час.</li>
<li>Соревнования по <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB" title="Волейбол">волейболу</a> впервые появились на <a href="/wiki/%D0%9B%D0%B5%D1%82%D0%BD%D0%B8%D0%B5_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B_1964" title="Летние Олимпийские игры 1964">летних Олимпийских играх 1964</a> в <a href="/wiki/%D0%A2%D0%BE%D0%BA%D0%B8%D0%BE" title="Токио">Токио</a>, а по <a href="/wiki/%D0%9F%D0%BB%D1%8F%D0%B6%D0%BD%D1%8B%D0%B9_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB" title="Пляжный волейбол">пляжному волейболу</a> — на <a href="/wiki/%D0%9B%D0%B5%D1%82%D0%BD%D0%B8%D0%B5_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B_1996" title="Летние Олимпийские игры 1996">Играх 1996 года</a> в <a href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B0" title="Атланта">Атланте</a>.</li>
<li><a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%81%D0%BF%D0%BE%D1%80%D1%82%D1%81%D0%BC%D0%B5%D0%BD%D0%BE%D0%B2_%D1%81_%D0%BD%D0%B0%D0%B8%D0%B1%D0%BE%D0%BB%D1%8C%D1%88%D0%B8%D0%BC_%D0%BA%D0%BE%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE%D0%BC_%D1%83%D1%87%D0%B0%D1%81%D1%82%D0%B8%D0%B9_%D0%B2_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85" title="Список спортсменов с наибольшим количеством участий в Олимпийских играх">Рекорд по количеству участий в олимпийских турнирах</a> по волейболу принадлежит россиянке <a href="/wiki/%D0%AD%D1%81%D1%82%D0%B5%D1%81,_%D0%95%D0%B2%D0%B3%D0%B5%D0%BD%D0%B8%D1%8F_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80%D0%BE%D0%B2%D0%BD%D0%B0" title="Эстес, Евгения Викторовна">Евгении Эстес</a>, выступавшей на шести Олимпийских играх — в 1992, 1996, 2000, 2004, 2008 и 2012 годах.</li>
<li>С 1996 года ни одной команде не удаётся стать победителем <a href="/wiki/%D0%9C%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BB%D0%B8%D0%B3%D0%B0_(%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB)" title="Мировая лига (волейбол)">Мировой лиги</a>, если она является организатором финального турнира этого соревнования. Последними добивались успеха на глазах у своих болельщиков волейболисты Нидерландов.</li>
<li><a href="/wiki/%D0%A7%D1%83%D0%B4%D0%B8%D0%BD%D0%B0,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%B0_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%BD%D0%B0" title="Чудина, Александра Георгиевна">Трёхкратной чемпионке мира и четырёхкратной чемпионке Европы по волейболу</a> принадлежали два мировых рекорда в <a href="/wiki/%D0%9B%D1%91%D0%B3%D0%BA%D0%B0%D1%8F_%D0%B0%D1%82%D0%BB%D0%B5%D1%82%D0%B8%D0%BA%D0%B0" title="Лёгкая атлетика">легкоатлетическом</a> пятиборье и одно высшее достижение в прыжках в высоту.</li>
<li>12 августа 2012 года в <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондоне</a> <a href="/wiki/%D0%9C%D1%83%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83" title="Мужская сборная России по волейболу">сборная России</a> стала первой командой в мире, которая смогла одержать победу в <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB_%D0%BD%D0%B0_%D0%BB%D0%B5%D1%82%D0%BD%D0%B8%D1%85_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85_2012_(%D1%84%D0%B8%D0%BD%D0%B0%D0%BB,_%D0%BC%D1%83%D0%B6%D1%87%D0%B8%D0%BD%D1%8B)" title="Волейбол на летних Олимпийских играх 2012 (финал, мужчины)">финальном матче мужского турнира Олимпийских игр</a>, проигрывая по его ходу со счётом 0:2.</li>
<li>Самого эмоционального <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%80%D0%B8%D0%B4%D0%BE%D0%BD%D0%BE%D0%B2,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B5%D0%B9_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Спиридонов, Алексей Александрович">игрока</a> <a href="/wiki/%D0%9C%D1%83%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83" title="Мужская сборная России по волейболу">сборной России по волейболу</a> зарубежные болельщики за характерную внешность сравнивают с <a href="/wiki/%D0%A2%D0%B8%D0%BD%D1%82%D0%B8%D0%BD" title="Тинтин">героем комиксов</a>.</li>

<p><b>Максим Михайлович Михайлов</b> (родился <a href="/wiki/19_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="19 марта">19 марта</a> <a href="/wiki/1988_%D0%B3%D0%BE%D0%B4" title="1988 год">1988 года</a> в посёлке <a href="/wiki/%D0%9A%D1%83%D0%B7%D1%8C%D0%BC%D0%BE%D0%BB%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9" title="Кузьмоловский">Кузьмоловском</a> <a href="/wiki/%D0%9B%D0%B5%D0%BD%D0%B8%D0%BD%D0%B3%D1%80%D0%B0%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Ленинградская область">Ленинградской области</a>) — <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">российский</a> <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB" title="Волейбол">волейболист</a>, диагональный, капитан <a href="/wiki/%D0%97%D0%B5%D0%BD%D0%B8%D1%82_(%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BB%D1%83%D0%B1)" class="mw-redirect mw-disambig" title="Зенит (волейбольный клуб)">казанского «Зенита»</a>, игрок <a href="/wiki/%D0%9C%D1%83%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83" title="Мужская сборная России по волейболу">сборной России</a>. Бронзовый призёр <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB_%D0%BD%D0%B0_%D0%BB%D0%B5%D1%82%D0%BD%D0%B8%D1%85_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85_2008_(%D0%BC%D1%83%D0%B6%D1%87%D0%B8%D0%BD%D1%8B)" title="Волейбол на летних Олимпийских играх 2008 (мужчины)">Игр XXIX Олимпиады в Пекине</a>, чемпион <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB_%D0%BD%D0%B0_%D0%BB%D0%B5%D1%82%D0%BD%D0%B8%D1%85_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85_2012_(%D0%BC%D1%83%D0%B6%D1%87%D0%B8%D0%BD%D1%8B)" title="Волейбол на летних Олимпийских играх 2012 (мужчины)">Игр ХХХ Олимпиады в Лондоне</a>, <a href="/wiki/%D0%97%D0%B0%D1%81%D0%BB%D1%83%D0%B6%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B0%D1%81%D1%82%D0%B5%D1%80_%D1%81%D0%BF%D0%BE%D1%80%D1%82%D0%B0_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Заслуженный мастер спорта России">заслуженный мастер спорта России</a>.
</p><p>Начинал заниматься волейболом с 8 лет в Кузьмоловском у тренера Валерия Емельяновича Беспрозванных, затем тренировался в спортшколе олимпийского резерва «Экран» в <a href="/wiki/%D0%A1%D0%B0%D0%BD%D0%BA%D1%82-%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3" title="Санкт-Петербург">Санкт-Петербурге</a>, в 15 лет переехал в <a href="/wiki/%D0%AF%D1%80%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BB%D1%8C" title="Ярославль">Ярославль</a>. В сезоне 2004/05 годов выступал в высшей лиге «Б» чемпионата России за ярославский «Строитель» — фарм-команду <a href="/wiki/%D0%AF%D1%80%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%B8%D1%87" title="Ярославич">«Нефтяника»</a>, являвшуюся базовой командой юниорской сборной России. С 2005 года играл за «Нефтяник» (ныне — «Ярославич»), с 2010 года защищает цвета казанского «Зенита».
</p><p><a href="/wiki/14_%D0%B8%D1%8E%D0%BD%D1%8F" title="14 июня">14 июня</a> <a href="/wiki/2008_%D0%B3%D0%BE%D0%B4" title="2008 год">2008 года</a> в <a href="/wiki/%D0%A1%D1%83%D0%B2%D0%BE%D0%BD" title="Сувон">Сувоне</a> провёл первый матч за национальную сборную России. В 2010—2011 годах завоёвывал индивидуальные призы на всех официальных соревнованиях, признавался самым ценным игроком <a href="/wiki/%D0%9C%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%BB%D0%B8%D0%B3%D0%B0_2011_(%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB)" title="Мировая лига 2011 (волейбол)">Мировой лиги</a> и <a href="/wiki/%D0%9A%D1%83%D0%B1%D0%BE%D0%BA_%D0%BC%D0%B8%D1%80%D0%B0_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83_%D1%81%D1%80%D0%B5%D0%B4%D0%B8_%D0%BC%D1%83%D0%B6%D1%87%D0%B8%D0%BD_2011" title="Кубок мира по волейболу среди мужчин 2011">Кубка мира</a> 2011 года, завершившихся победами сборной России. В заявке команды на <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB_%D0%BD%D0%B0_%D0%BB%D0%B5%D1%82%D0%BD%D0%B8%D1%85_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D1%85_%D0%B8%D0%B3%D1%80%D0%B0%D1%85_2012_(%D0%BC%D1%83%D0%B6%D1%87%D0%B8%D0%BD%D1%8B)" title="Волейбол на летних Олимпийских играх 2012 (мужчины)">Олимпийские игры в Лондоне</a> являлся единственным диагональным и несмотря на травму голеностопа, полученную в контрольном матче со <a href="/wiki/%D0%9C%D1%83%D0%B6%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D0%B8%D0%B8_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9%D0%B1%D0%BE%D0%BB%D1%83" title="Мужская сборная Болгарии по волейболу">сборной Болгарии</a> за два дня до первой игры на Олимпиаде, сыграл во всех матчах сборной России, завоевал золотую медаль и со 148 очками стал самым результативным волейболистом на лондонских Играх. <i><small>(<a href="/wiki/%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2,_%D0%9C%D0%B0%D0%BA%D1%81%D0%B8%D0%BC_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Михайлов, Максим Михайлович">Подробнее</a>)</small></i>
</p>
