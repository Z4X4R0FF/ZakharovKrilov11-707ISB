<p>Во всех странах мира автомобилям присваиваются уникальные номера, которые должны быть показаны на заметных светоотражающих плашках впереди и/или сзади автомобиля.
</p><p>В России большинство <a href="/wiki/%D0%90%D0%B2%D1%82%D0%BE%D0%BC%D0%BE%D0%B1%D0%B8%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%B4" class="mw-redirect" title="Автомобильный код">номерных знаков</a> имеют следующий вид:
</p>
<div class="floatright"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Russian_numberplate.png" class="image"><img alt="Russian numberplate.png" src="//upload.wikimedia.org/wikipedia/commons/8/82/Russian_numberplate.png" decoding="async" width="312" height="120" data-file-width="312" data-file-height="120" /></a></div>
<p>Все используемые номера зарегистрированы. Для каждого административного района есть свой номер, общий для всех автомобилей, зарегистрированных в этом округе. <a href="/wiki/%D0%93%D0%9E%D0%A1%D0%A2" class="mw-redirect" title="ГОСТ">ГОСТ</a> предусматривает изготовление лишь 1 млн 726 тыс. 272 комплектов регистрационных знаков для каждого субъекта Федерации - именно столько 
возможно комбинаций с использованием разрешенных <a href="/wiki/%D0%93%D0%9E%D0%A1%D0%A2" class="mw-redirect" title="ГОСТ">ГОСТом</a> символов - это цифры 
и 12 букв кириллицы, имеющих графические аналоги в латинском алфавите (А, В, 
Е, К, М, Н, О, Р, С, Т, X и У). Между тем понятно, что в некоторых регионах куда как больше, чем 1,7 млн автомобилей. Например, на трехзначный код, помимо Москвы, были вынуждены перейти республика <a href="/wiki/%D0%A2%D0%B0%D1%82%D0%B0%D1%80%D1%81%D1%82%D0%B0%D0%BD" title="Татарстан">Татарстан</a> (16, 116), <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Московская область">Московская область</a> (50, 90, 150), <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%B0%D1%80%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Самарская область">Самарская область</a> (63, 163), <a href="/wiki/%D0%A7%D0%B5%D0%BB%D1%8F%D0%B1%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Челябинская область">Челябинская область</a> (74, 174) и <a href="#Дополнительные_номера">другие регионы</a>. Миллионами исчисляются парки и других областей, где действует несколько серийных кодов: <a href="/wiki/%D0%A1%D0%B0%D0%BD%D0%BA%D1%82-%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3" title="Санкт-Петербург">Санкт-Петербург</a> (78, 98,178), 
<a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D0%BE%D0%B4%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B0%D0%B9" title="Краснодарский край">Краснодарский край</a> (23, 93), <a href="/wiki/%D0%A1%D0%B2%D0%B5%D1%80%D0%B4%D0%BB%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Свердловская область">Свердловская область</a> (66, 96), <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B8%D0%BD%D0%B8%D0%BD%D0%B3%D1%80%D0%B0%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Калининградская область">Калининградская область</a> (39, 91).
