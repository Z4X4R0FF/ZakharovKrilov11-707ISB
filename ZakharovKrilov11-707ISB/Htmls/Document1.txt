<p><b><a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Вторая мировая война</a></b> (<a href="/wiki/1939" class="mw-redirect" title="1939">1939</a>—<a href="/wiki/1945" class="mw-redirect" title="1945">1945</a>) — крупнейшая <a href="/wiki/%D0%92%D0%BE%D0%B9%D0%BD%D0%B0" title="Война">война</a> в истории человечества. В ней участвовали 62 государства и свыше 80&#160;% населения земного шара, военные действия охватили территории 40 государств, погибло приблизительно 58 млн человек.

Японское соединение осуществляло прикрытие <a href="/wiki/%D0%9A%D0%BE%D0%BD%D0%B2%D0%BE%D0%B9_%D1%81%D1%83%D0%B4%D0%BE%D0%B2" class="mw-redirect" title="Конвой судов">конвоя</a>, доставляющего подкрепления на остров <a href="/wiki/%D0%93%D1%83%D0%B0%D0%B4%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D0%B0%D0%BB_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" class="mw-redirect" title="Гуадалканал (остров)">Гуадалканал</a> в рамках операции «„Ка“ Сакусэн». Американское соединение выполняло функции по прикрытию своих высадившихся войск. ... <b><a href="/wiki/%D0%A1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_%D1%83_%D0%B2%D0%BE%D1%81%D1%82%D0%BE%D1%87%D0%BD%D1%8B%D1%85_%D0%A1%D0%BE%D0%BB%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D0%B2%D1%8B%D1%85_%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2%D0%BE%D0%B2" title="Сражение у восточных Соломоновых островов">Дальше</a></b>

<p><b>Иван Данилович Черняховский</b> (<a href="/wiki/29_%D0%B8%D1%8E%D0%BD%D1%8F" title="29 июня">29 июня</a> <a href="/wiki/1906_%D0%B3%D0%BE%D0%B4" title="1906 год">1906 года</a>&#160;— <a href="/wiki/18_%D1%84%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8F" title="18 февраля">18 февраля</a> <a href="/wiki/1945_%D0%B3%D0%BE%D0%B4" title="1945 год">1945 года</a>)&#160;— выдающийся полководец <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Великая Отечественная война">Великой Отечественной войны</a>, <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB_%D0%B0%D1%80%D0%BC%D0%B8%D0%B8" title="Генерал армии">генерал армии</a> (<a href="/wiki/1944" class="mw-redirect" title="1944">1944</a>), дважды <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BE%D0%B9_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%A1%D0%BE%D1%8E%D0%B7%D0%B0" title="Герой Советского Союза">Герой Советского Союза</a>...
</p><p>Родился на Украине в селе Оксанине Уманьского уезда <a href="/wiki/%D0%9A%D0%B8%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Киевская губерния">Киевской губернии</a>. В <a href="/wiki/1924_%D0%B3%D0%BE%D0%B4" title="1924 год">1924 году</a> добровольцем вступил в <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D0%B0%D1%8F_%D0%90%D1%80%D0%BC%D0%B8%D1%8F" class="mw-redirect" title="Красная Армия">Красную Армию</a>.
</p><p>В начале <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Великая Отечественная война">Великой Отечественной войны</a> командовал 28-й танковой дивизией. С июля <a href="/wiki/1942_%D0%B3%D0%BE%D0%B4" title="1942 год">1942 года</a>&#160;— командующий 60-й армией ...
</p><p><a href="/wiki/28_%D0%B8%D1%8E%D0%BD%D1%8F" title="28 июня">28 июня</a> <a href="/wiki/1944_%D0%B3%D0%BE%D0%B4" title="1944 год">1944 года</a> ему присвоено воинское звание «<a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB_%D0%B0%D1%80%D0%BC%D0%B8%D0%B8_(%D0%A1%D0%A1%D0%A1%D0%A0)" title="Генерал армии (СССР)">генерал армии</a>». Таким образом И.Д. Черняховский стал самым молодым генералом армии в <a href="/wiki/%D0%A0%D0%9A%D0%9A%D0%90" class="mw-redirect" title="РККА">РККА</a> (в 38 лет)... <b><a href="/wiki/%D0%A7%D0%B5%D1%80%D0%BD%D1%8F%D1%85%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9,_%D0%98%D0%B2%D0%B0%D0%BD_%D0%94%D0%B0%D0%BD%D0%B8%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Черняховский, Иван Данилович">Дальше</a></b>

<p><b>152-мм гаубица образца 1938 года</b> — <a href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="СССР">советская</a> <a href="/wiki/%D0%93%D0%B0%D1%83%D0%B1%D0%B8%D1%86%D0%B0" title="Гаубица">гаубица</a> периода <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a>. Это орудие серийно выпускалось с <a href="/wiki/1939_%D0%B3%D0%BE%D0%B4" title="1939 год">1939</a> по <a href="/wiki/1941_%D0%B3%D0%BE%D0%B4" title="1941 год">1941 год</a> включительно, использовалось в течение всей <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Великая Отечественная война">Великой Отечественной войны</a> и состояло на вооружении <a href="/wiki/%D0%A0%D0%9A%D0%9A%D0%90" class="mw-redirect" title="РККА">Рабоче-крестьянской Красной армии</a> (позже <a href="/wiki/%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F" title="Советская армия">Советской армии</a>) до конца <a href="/wiki/1950-%D0%B5" class="mw-redirect" title="1950-е">1950-х годов</a>. Трофейные гаубицы М-10 были приняты на вооружение немецкого <a href="/wiki/%D0%92%D0%B5%D1%80%D0%BC%D0%B0%D1%85%D1%82" title="Вермахт">вермахта</a> и армии <a href="/wiki/%D0%A4%D0%B8%D0%BD%D0%BB%D1%8F%D0%BD%D0%B4%D0%B8%D1%8F" title="Финляндия">Финляндии</a>; захваченные орудия активно использовались в боях Второй мировой войны. Финские военные были настолько удовлетворены возможностями М-10, что в послевоенное время не снимали её с вооружения вплоть до <a href="/wiki/2000_%D0%B3%D0%BE%D0%B4" title="2000 год">2000 года</a><b>… <a href="/wiki/152-%D0%BC%D0%BC_%D0%B3%D0%B0%D1%83%D0%B1%D0%B8%D1%86%D0%B0_%D0%BE%D0%B1%D1%80%D0%B0%D0%B7%D1%86%D0%B0_1938_%D0%B3%D0%BE%D0%B4%D0%B0" class="mw-redirect" title="152-мм гаубица образца 1938 года">Дальше</a></b>
</p>
