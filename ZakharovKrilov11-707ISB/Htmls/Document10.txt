<div class="floatleft"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Coat_of_arms_of_Vologda_oblast.svg" class="image"><img alt="Coat of arms of Vologda oblast.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/7a/Coat_of_arms_of_Vologda_oblast.svg/100px-Coat_of_arms_of_Vologda_oblast.svg.png" decoding="async" width="100" height="126" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/7a/Coat_of_arms_of_Vologda_oblast.svg/150px-Coat_of_arms_of_Vologda_oblast.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/7a/Coat_of_arms_of_Vologda_oblast.svg/200px-Coat_of_arms_of_Vologda_oblast.svg.png 2x" data-file-width="716" data-file-height="899" /></a></div>
<p><b><a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Вологодская область">Вологодская область</a></b> — субъект <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Российской Федерации</a>, образована <a href="/wiki/23_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="23 сентября">23 сентября</a> <a href="/wiki/1937" class="mw-redirect" title="1937">1937</a>. Расположена на северо-западе Европейской части России. Входит в состав <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BE-%D0%B7%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BE%D0%BA%D1%80%D1%83%D0%B3" class="mw-redirect" title="Северо-западный федеральный округ">Северо-Западного федерального округа</a>. Административный центр — <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BE%D0%B4" title="Город">город</a> <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%B4%D0%B0" title="Вологда">Вологда</a> (288,4 тыс. чел. (<a href="/wiki/2005" class="mw-redirect" title="2005">2005</a>)).
</p><p>Вологодская область по территории (145,7 тыс. км²) занимает 5-е место из 6-ти регионов <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D1%8D%D0%BA%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Северный экономический район">Северного экономического района</a> и 27-е по <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a>, а по населению (1245,5 тыс. чел. на 1.01.2005 г.) 2-е место (42-е в России). По плотности населения (8,5 чел/км²) - 56-й регион в РФ, а по уровню урбанизации (уд. вес городского населения — 68&#160;%) - 39-е место (<a href="/wiki/2005" class="mw-redirect" title="2005">2005</a>).
</p><p>Ряд городов и населённых пунктов области: <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D0%B7%D0%B5%D1%80%D1%81%D0%BA" title="Белозерск">Белозерск</a>, <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9_%D0%A3%D1%81%D1%82%D1%8E%D0%B3" title="Великий Устюг">Великий Устюг</a>, <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%B4%D0%B0" title="Вологда">Вологда</a>, <a href="/wiki/%D0%A3%D1%81%D1%82%D1%8E%D0%B6%D0%BD%D0%B0" title="Устюжна">Устюжна</a>, <a href="/wiki/%D0%A2%D0%BE%D1%82%D1%8C%D0%BC%D0%B0" title="Тотьма">Тотьма</a> и другие, — имеют статус <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B5_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" class="mw-redirect" title="Исторические города России">исторических</a> и являются музеями под открытым небом. Великий Устюг с <a href="/wiki/1999" class="mw-redirect" title="1999">1999</a> года считается родиной российского <a href="/wiki/%D0%94%D0%B5%D0%B4_%D0%9C%D0%BE%D1%80%D0%BE%D0%B7" title="Дед Мороз">Деда Мороза</a>.
</p>

<p><b><a href="/wiki/%D0%9E%D0%BD%D0%B5%D0%B6%D1%81%D0%BA%D0%BE%D0%B5_%D0%BE%D0%B7%D0%B5%D1%80%D0%BE" title="Онежское озеро">Оне́жское о́зеро</a></b> (<b>Оне́го</b>, <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Карельский язык">карел.</a> <span lang="krl" style="font-style:italic;">Oniegu</span>, <a href="/wiki/%D0%A4%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Финский язык">фин.</a> <span lang="fi" style="font-style:italic;"><i>Ääninen</i>, <i>Äänisjärvi</i></span>) — <a href="/wiki/%D0%9E%D0%B7%D0%B5%D1%80%D0%BE" title="Озеро">озеро</a> на северо-западе Европейской части <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Российской Федерации</a>, расположено на территории <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D0%B8%D1%8F" title="Карелия">Карелии</a>, <a href="/wiki/%D0%9B%D0%B5%D0%BD%D0%B8%D0%BD%D0%B3%D1%80%D0%B0%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Ленинградская область">Ленинградской</a> и <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Вологодская область">Вологодской</a> областей. Второе по величине озеро в <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">Европе</a> после <a href="/wiki/%D0%9B%D0%B0%D0%B4%D0%BE%D0%B6%D1%81%D0%BA%D0%BE%D0%B5_%D0%BE%D0%B7%D0%B5%D1%80%D0%BE" title="Ладожское озеро">Ладожского</a>. Относится к бассейну <a href="/wiki/%D0%91%D0%B0%D0%BB%D1%82%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Балтийское море">Балтийского моря</a> <a href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Атлантический океан">Атлантического океана</a>.
</p><p>Площадь озера без островов составляет от 9,7 тыс. км²; объём водной массы — 285 км³; длина с юга на север — 245 км, наибольшая ширина — 91,6 км. Средняя глубина — 30&#160;м, а максимальная — 127&#160;м.
</p><p>На берегах Онежского озера расположены города <a href="/wiki/%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B7%D0%B0%D0%B2%D0%BE%D0%B4%D1%81%D0%BA" title="Петрозаводск">Петрозаводск</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BD%D0%B4%D0%BE%D0%BF%D0%BE%D0%B3%D0%B0" title="Кондопога">Кондопога</a> и <a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%B2%D0%B5%D0%B6%D1%8C%D0%B5%D0%B3%D0%BE%D1%80%D1%81%D0%BA" title="Медвежьегорск">Медвежьегорск</a>. В Онежское озеро впадают около 50 рек, а вытекает только одна — <a href="/wiki/%D0%A1%D0%B2%D0%B8%D1%80%D1%8C_(%D1%80%D0%B5%D0%BA%D0%B0)" class="mw-redirect" title="Свирь (река)">Свирь</a>.
</p>

<div style="display:block; border:1px solid red; vertical-align:top; background:white; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<p><b><a href="/wiki/%D0%9D%D0%B5%D0%BA%D1%80%D0%B0%D1%81%D0%BE%D0%B2,_%D0%AE%D1%80%D0%B8%D0%B9_%D0%9A%D0%BB%D0%B0%D0%B2%D0%B4%D0%B8%D0%B5%D0%B2%D0%B8%D1%87" title="Некрасов, Юрий Клавдиевич">Юрий Клавдиевич Некрасов</a></b> (<a href="/wiki/10_%D0%BC%D0%B0%D1%8F" title="10 мая">10 мая</a> <a href="/wiki/1935" class="mw-redirect" title="1935">1935</a>, <a href="/wiki/%D0%90%D1%80%D1%85%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D1%8C%D1%81%D0%BA" title="Архангельск">Архангельск</a> - <a href="/wiki/27_%D0%B8%D1%8E%D0%BD%D1%8F" title="27 июня">27 июня</a> <a href="/wiki/2006" class="mw-redirect" title="2006">2006</a>, <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%B4%D0%B0" title="Вологда">Вологда</a>) - российский историк, политический деятель.
</p><p>Учился на историческом факультете Ленинградского государственного университета, окончил историко-филологический факультет Вологодского государственного педагогического института (ВГПИ, ныне <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B4%D1%81%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BF%D0%B5%D0%B4%D0%B0%D0%B3%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82" title="Вологодский государственный педагогический университет">ВГПУ</a>). Кандидат исторических наук (с <a href="/wiki/1968" class="mw-redirect" title="1968">1968</a>), доктор исторических наук (с <a href="/wiki/1982" class="mw-redirect" title="1982">1982</a>), профессор (с <a href="/wiki/1985" class="mw-redirect" title="1985">1985</a>). С <a href="/wiki/1978" class="mw-redirect" title="1978">1978</a> - по <a href="/wiki/2006" class="mw-redirect" title="2006">2006</a> гг. - заведующий кафедрой всеобщей истории ВГПУ.
</p><p>Основная проблематика исследований: история средневековой цивилизации, ее переход к прединдустриальной цивилизации и возникновение явлений раннего капитализма; социально-экономическая история германских городов; изучение Реформации и Крестьянской войны в <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">Германии</a>.
</p>

<div style="display:block; border:1px solid red; vertical-align:top; background:white; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<ul><li><b><a href="/wiki/%D0%A4%D0%B5%D1%80%D0%B0%D0%BF%D0%BE%D0%BD%D1%82%D0%BE%D0%B2_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" class="mw-redirect" title="Ферапонтов монастырь">Ферапонтов монастырь</a></b>, являющийся редким образцом сохранности и стилевого единства русского северного монастырского ансамбля <a href="/wiki/XV_%D0%B2%D0%B5%D0%BA" title="XV век">XV</a> — <a href="/wiki/XVII_%D0%B2%D0%B5%D0%BA" title="XVII век">XVII</a> веков, был включён в список <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D0%B8%D0%B5" title="Всемирное наследие">Всемирного наследия</a> <a href="/wiki/%D0%AE%D0%9D%D0%95%D0%A1%D0%9A%D0%9E" title="ЮНЕСКО">ЮНЕСКО</a> в <a href="/wiki/2000" class="mw-redirect" title="2000">2000</a> году</li></ul>
<ul><li>На вкус и аромат <b><a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B4%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%B0%D1%81%D0%BB%D0%BE" title="Вологодское масло">Вологодского масла</a></b> влияет множество факторов: это состав трав на лугах, рацион коров, породность стада, жирность и кислотность <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%BE%D0%BA%D0%BE" title="Молоко">молока</a>, но самое главное — его чистота и свежесть. Для выработки вологодского масла требуется молоко с особыми свойствами, только свежие <a href="/wiki/%D0%A1%D0%BB%D0%B8%D0%B2%D0%BA%D0%B8" title="Сливки">сливки</a> с чистым, свежим, сладковатым вкусом без посторонних привкусов и запахов, определённой жирности, нужна определённая температура их пастеризации в закрытых емкостях, чтобы удержать в них ароматические вещества, создающие вкус масла, быстрое охлаждение, режим физического созревания и сбивание, а затем промывка готового масла чистой, а ещё лучше — кипячёной водой.</li></ul>

<ul><li><b><a href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%9F%D0%BE%D0%BA%D1%80%D0%BE%D0%B2%D0%B0_%D0%9F%D1%80%D0%B5%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B9_%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D1%8B_(%D0%92%D1%8B%D1%82%D0%B5%D0%B3%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%B3%D0%BE%D1%81%D1%82)" title="Церковь Покрова Пресвятой Богородицы (Вытегорский погост)">Церковь Покрова Пресвятой Богородицы</a></b>, построенная в <a href="/wiki/1708" class="mw-redirect" title="1708">1708</a> г. и сгоревшая в <a href="/wiki/1963" class="mw-redirect" title="1963">1963</a> г., являлась выдающимся образцом русской деревянной архитектуры. Располагалась в селе Анхимово (Вытегорский Погост, ныне <a href="/wiki/%D0%92%D1%8B%D1%82%D0%B5%D0%B3%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD_%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B4%D1%81%D0%BA%D0%BE%D0%B9_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D0%B8" class="mw-redirect" title="Вытегорский район Вологодской области">Вытегорский район</a> <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Вологодская область">Вологодской области</a>). Этот 24-главый храм был одним из самых ярких памятников русского <a href="/wiki/%D0%94%D0%B5%D1%80%D0%B5%D0%B2%D1%8F%D0%BD%D0%BD%D0%BE%D0%B5_%D0%B7%D0%BE%D0%B4%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%BE" class="mw-redirect" title="Деревянное зодчество">деревянного зодчества</a>, непосредственным предшественником знаменитого Преображенского храма <a href="/wiki/%D0%9A%D0%B8%D0%B6%D0%B8" title="Кижи">Кижского погоста</a> (<a href="/wiki/1714" class="mw-redirect" title="1714">1714</a>). Сходство двух храмов вызвало к жизни предположение об их строительстве одними и теми же мастерами.</li></ul>
