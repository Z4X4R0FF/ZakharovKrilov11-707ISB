<p><b><a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Австралия">Австрали́йский Сою́з</a></b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">Commonwealth of Australia</span>), <b>Австра́лия</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">Australia</span>, от <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a>&#160;<span lang="la" style="font-style:italic;">austrālis</span>&#160;«южный»)&#160;— государство в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%BE%D0%B5_%D0%BF%D0%BE%D0%BB%D1%83%D1%88%D0%B0%D1%80%D0%B8%D0%B5" title="Южное полушарие">Южном полушарии</a>. Население, по состоянию на <a href="/wiki/2011_%D0%B3%D0%BE%D0%B4" title="2011 год">2011 год</a>&#160;— 22 858 157 человек, территория&#160;— 7 692 024 км², по обоим этим показателем является крупнейшей страной <a href="/wiki/%D0%9E%D0%BA%D0%B5%D0%B0%D0%BD%D0%B8%D1%8F" title="Океания">региона</a>. Занимает <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%81%D1%82%D1%80%D0%B0%D0%BD,_%D1%81%D0%BE%D1%80%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%BA%D0%B0_%D0%BF%D0%BE_%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D0%B8_%D0%BD%D0%B0%D1%81%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" class="mw-redirect" title="Список стран, сортировка по численности населения">пятьдесят первое место в мире по населению</a> и <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%81%D1%82%D1%80%D0%B0%D0%BD_%D0%B8_%D0%B7%D0%B0%D0%B2%D0%B8%D1%81%D0%B8%D0%BC%D1%8B%D1%85_%D1%82%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D0%B9_%D0%BF%D0%BE_%D0%BF%D0%BB%D0%BE%D1%89%D0%B0%D0%B4%D0%B8" class="mw-redirect" title="Список стран и зависимых территорий по площади">шестое&#160;— по территории</a>. Большинство австралийцев проживают на восточном побережье континента.
</p><p>Государство расположено на <b><a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F_(%D0%BA%D0%BE%D0%BD%D1%82%D0%B8%D0%BD%D0%B5%D0%BD%D1%82)" title="Австралия (континент)">одноименном континенте</a></b> и на многочисленных островах в <a href="/wiki/%D0%A2%D0%B8%D1%85%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Тихий океан">Тихом</a> и <a href="/wiki/%D0%98%D0%BD%D0%B4%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Индийский океан">Индийском</a> <a href="/wiki/%D0%9E%D0%BA%D0%B5%D0%B0%D0%BD" title="Океан">океанах</a>. Сухопутных границ не имеет, существует морская граница с <a href="/wiki/%D0%9F%D0%B0%D0%BF%D1%83%D0%B0-%D0%9D%D0%BE%D0%B2%D0%B0%D1%8F_%D0%93%D0%B2%D0%B8%D0%BD%D0%B5%D1%8F" class="mw-redirect" title="Папуа-Новая Гвинея">Папуа-Новой Гвинеей</a>.
</p><p>Столица&#160;— <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%B1%D0%B5%D1%80%D1%80%D0%B0" title="Канберра">Канберра</a>. Государственный язык&#160;— <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">английский</a> (особый вариант, известный как <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%B0%D1%80%D0%B8%D0%B0%D0%BD%D1%82_%D0%B0%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D1%8F%D0%B7%D1%8B%D0%BA%D0%B0" title="Австралийский вариант английского языка">австралийский английский</a>; на острове <a href="/wiki/%D0%9D%D0%BE%D1%80%D1%84%D0%BE%D0%BB%D0%BA_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" class="mw-redirect" title="Норфолк (остров)">Норфолк</a> государственным наряду с английским является <a href="/wiki/%D0%9D%D0%BE%D1%80%D1%84%D0%BE%D0%BB%D0%BA%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Норфолкский язык">норфолкский язык</a>), также на территории страны используются <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D1%8F%D0%B7%D1%8B%D0%BA%D0%B8" title="Австралийские языки">австралийские языки</a>.
</p><p>Является <a href="/wiki/%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F" class="mw-redirect" title="Федерация">федерацией</a>, <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BC%D0%BE%D0%BD%D0%B0%D1%80%D1%85%D0%B8%D1%8F" title="Конституционная монархия">конституционной монархией</a> во главе с <a href="/wiki/%D0%95%D0%BB%D0%B8%D0%B7%D0%B0%D0%B2%D0%B5%D1%82%D0%B0_II" title="Елизавета II">Елизаветой II</a>. <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%BC%D1%8C%D0%B5%D1%80-%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B8" title="Премьер-министр Австралии">Премьер-министр</a>&#160;— <a href="/wiki/%D0%AD%D0%B1%D0%B1%D0%BE%D1%82%D1%82,_%D0%A2%D0%BE%D0%BD%D0%B8" class="mw-redirect" title="Эбботт, Тони">Тони Эбботт</a>, <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D0%BB-%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B0%D1%82%D0%BE%D1%80_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B8" title="Генерал-губернатор Австралии">генерал-губернатор</a>&#160;— <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B9%D1%81,_%D0%9A%D0%B2%D0%B5%D0%BD%D1%82%D0%B8%D0%BD" title="Брайс, Квентин">Квентин Брайс</a>. Состоит <a href="/wiki/%D0%90%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B8" title="Административное деление Австралии">из 6 штатов и 2 территорий</a>, которые в свою очередь состоят из 600 графств, 544 сотен и по крайней мере 15 692 приходов. Также Австралии принадлежит <a href="/wiki/%D0%90%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B8#Внешние_территории" title="Административное деление Австралии">ряд внешних территорий</a>.
</p><p>Австралийский Союз образован <a href="/wiki/1_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="1 января">1 января</a> <a href="/wiki/1901_%D0%B3%D0%BE%D0%B4" title="1901 год">1901 года</a> и до <a href="/wiki/1931_%D0%B3%D0%BE%D0%B4" title="1931 год">1931 года</a> был <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B8%D0%BD%D0%B8%D0%BE%D0%BD" title="Доминион">доминионом</a> <a href="/wiki/%D0%91%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Британская империя">Британской империи</a>. Предшествовавшие этому два века на территории Австралии существовали шесть колоний, впоследствии вошедшие в Союз на правах штатов. <i>(<a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Австралия">Далее…</a>)</i>
</p><p><br />

<p><a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%A5%D0%BE%D1%80%D0%BE%D1%88%D0%B8%D0%B5_%D1%81%D1%82%D0%B0%D1%82%D1%8C%D0%B8" title="Хорошая статья"><img alt="✰" src="//upload.wikimedia.org/wikipedia/commons/thumb/2/20/Blue_star_unboxed.svg/13px-Blue_star_unboxed.svg.png" decoding="async" width="13" height="13" style="vertical-align: baseline" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/2/20/Blue_star_unboxed.svg/20px-Blue_star_unboxed.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/20/Blue_star_unboxed.svg/26px-Blue_star_unboxed.svg.png 2x" data-file-width="13" data-file-height="13" /></a><b>BHP Billiton</b>, <b>«Би-Эйч-Пи Биллитон»</b> — одна из крупнейших в мире <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%BF%D1%80%D0%BE%D0%BC%D1%8B%D1%88%D0%BB%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Горная промышленность">горнодобывающих</a> компаний. Основная штаб-квартира находится в <a href="/wiki/%D0%9C%D0%B5%D0%BB%D1%8C%D0%B1%D1%83%D1%80%D0%BD" title="Мельбурн">Мельбурне</a> (<a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Австралия">Австралия</a>) дополнительная — в <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондоне</a> (<a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">Великобритания</a>). Основными направлениями деятельности являются добыча <a href="/wiki/%D0%9D%D0%B5%D1%84%D1%82%D1%8C" title="Нефть">нефти</a> и <a href="/wiki/%D0%9F%D1%80%D0%B8%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D0%B3%D0%B0%D0%B7" title="Природный газ">газа</a> (в том числе <a href="/wiki/%D0%A1%D0%BB%D0%B0%D0%BD%D1%86%D0%B5%D0%B2%D0%B0%D1%8F_%D0%BD%D0%B5%D1%84%D1%82%D1%8C" title="Сланцевая нефть">сланцевых</a>), <a href="/wiki/%D0%9C%D0%B5%D0%B4%D1%8C" title="Медь">меди</a>, <a href="/wiki/%D0%96%D0%B5%D0%BB%D0%B5%D0%B7%D0%BD%D0%B0%D1%8F_%D1%80%D1%83%D0%B4%D0%B0" title="Железная руда">железной руды</a> и <a href="/wiki/%D0%A3%D0%B3%D0%BE%D0%BB%D1%8C" class="mw-redirect" title="Уголь">угля</a>. Основные мощности расположены в Австралии, а также в <a href="/wiki/%D0%A1%D0%A8%D0%90" class="mw-redirect" title="США">США</a>, <a href="/wiki/%D0%A7%D0%B8%D0%BB%D0%B8" title="Чили">Чили</a>, <a href="/wiki/%D0%9F%D0%B5%D1%80%D1%83" title="Перу">Перу</a> и других странах.
</p>
Образована в <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 году</a> путём объединения австралийской <b>Broken Hill Proprietary Company</b> (BHP), основанной в <a href="/wiki/1885_%D0%B3%D0%BE%D0%B4" title="1885 год">1885 году</a>, и англо-южноафриканской <b>Billiton</b>, основанной в <a href="/wiki/%D0%9D%D0%B8%D0%B4%D0%B5%D1%80%D0%BB%D0%B0%D0%BD%D0%B4%D1%8B" title="Нидерланды">Нидерландах</a> в <a href="/wiki/1860_%D0%B3%D0%BE%D0%B4" title="1860 год">1860 году</a>. <i>(<a href="/wiki/BHP_Billiton" title="BHP Billiton">Далее…</a>)</i><div style="clear:both"></div>

<p>Австралия состоит из шести <a href="/wiki/%D0%90%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B8" title="Административное деление Австралии">штатов</a>, двух материковых <a href="/wiki/%D0%A2%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D1%8F" title="Территория">территорий</a> и других более мелких территорий. Штатами являются <a href="/wiki/%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80%D0%B8%D1%8F_(%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F)" title="Виктория (Австралия)">Виктория</a> <i>(зеленым на карте)</i>, <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%B0%D1%8F_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Западная Австралия">Западная Австралия</a> <i>(оранжевым)</i>, <a href="/wiki/%D0%9A%D0%B2%D0%B8%D0%BD%D1%81%D0%BB%D0%B5%D0%BD%D0%B4" title="Квинсленд">Квинсленд</a> <i>(голубым)</i>, <a href="/wiki/%D0%9D%D0%BE%D0%B2%D1%8B%D0%B9_%D0%AE%D0%B6%D0%BD%D1%8B%D0%B9_%D0%A3%D1%8D%D0%BB%D1%8C%D1%81" title="Новый Южный Уэльс">Новый Южный Уэльс</a> <i>(красным)</i>, <a href="/wiki/%D0%A2%D0%B0%D1%81%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Тасмания">Тасмания</a> <i>(жёлтый остров на юге)</i> и <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F" title="Южная Австралия">Южная Австралия</a> <i>(фиолетовым)</i>. Двумя главными материковыми территориями являются <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D1%82%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D1%8F_(%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D1%8F)" class="mw-redirect" title="Северная территория (Австралия)">Северная территория</a> <i>(жёлтым)</i> и <a href="/wiki/%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D1%82%D0%BE%D0%BB%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D1%82%D0%B5%D1%80%D1%80%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D1%8F" title="Австралийская столичная территория">Территория федеральной столицы</a> <i>(небольшая красная точка)</i>. Статус территорий во многом аналогичен статусу штатов, за исключением того, что федеральный парламент может отменить любое решение парламента территории, в то время как по отношению к штатам федеральное законодательство имеет верховенство над законодательством штатов только в тех случаях, которые указаны в параграфе 51 <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%86%D0%B8%D1%8F_%D0%90%D0%B2%D1%81%D1%82%D1%80%D0%B0%D0%BB%D0%B8%D0%B8" title="Конституция Австралии">Конституции</a>. Все остальные вопросы остаются в ведении штата, например, <a href="/wiki/%D0%97%D0%B4%D1%80%D0%B0%D0%B2%D0%BE%D0%BE%D1%85%D1%80%D0%B0%D0%BD%D0%B5%D0%BD%D0%B8%D0%B5" title="Здравоохранение">здравоохранение</a>, <a href="/wiki/%D0%9E%D0%B1%D1%80%D0%B0%D0%B7%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Образование">образование</a>, <a href="/wiki/%D0%9F%D1%80%D0%B0%D0%B2%D0%BE%D0%BF%D0%BE%D1%80%D1%8F%D0%B4%D0%BE%D0%BA" title="Правопорядок">правопорядок</a>, <a href="/wiki/%D0%9E%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BF%D0%BE%D1%80%D1%82" title="Общественный транспорт">общественный транспорт</a>, дороги, <a href="/wiki/%D0%A1%D1%83%D0%B4%D0%BE%D1%83%D1%81%D1%82%D1%80%D0%BE%D0%B9%D1%81%D1%82%D0%B2%D0%BE_(%D0%BD%D0%BE%D1%80%D0%BC%D1%8B)" title="Судоустройство (нормы)">судоустройство</a> и <a href="/wiki/%D0%9C%D0%B5%D1%81%D1%82%D0%BD%D0%BE%D0%B5_%D1%81%D0%B0%D0%BC%D0%BE%D1%83%D0%BF%D1%80%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5" title="Местное самоуправление">местное самоуправление</a>.
