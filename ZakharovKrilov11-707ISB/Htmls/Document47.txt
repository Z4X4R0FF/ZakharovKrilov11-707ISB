<p><b>Бургундское королевство</b>  (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">Royaume de Bourgogne</span>) — <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%BD%D0%B8%D0%B5_%D0%B2%D0%B5%D0%BA%D0%B0" title="Средние века">средневековое</a> государственное образование, существовавшее в <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X</a>—<a href="/wiki/XIV_%D0%B2%D0%B5%D0%BA" title="XIV век">XIV веках</a> на территории современной юго-восточной <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a> (области <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D1%81" title="Прованс">Прованс</a>, <a href="/wiki/%D0%94%D0%BE%D1%84%D0%B8%D0%BD%D0%B5" title="Дофине">Дофине</a>, <a href="/wiki/%D0%A1%D0%B0%D0%B2%D0%BE%D0%B9%D1%8F" title="Савойя">Савойя</a>, <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%88-%D0%9A%D0%BE%D0%BD%D1%82%D0%B5" title="Франш-Конте">Франш-Конте</a>) и западной <a href="/wiki/%D0%A8%D0%B2%D0%B5%D0%B9%D1%86%D0%B0%D1%80%D0%B8%D1%8F" title="Швейцария">Швейцарии</a>. Возникнув после объединения в <a href="/wiki/933" class="mw-redirect" title="933">933</a> г. королевств <a href="/wiki/%D0%92%D0%B5%D1%80%D1%85%D0%BD%D1%8F%D1%8F_%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D0%B8%D1%8F" title="Верхняя Бургундия">Верхней</a> и <a href="/wiki/%D0%9D%D0%B8%D0%B6%D0%BD%D1%8F%D1%8F_%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D0%B8%D1%8F" title="Нижняя Бургундия">Нижней Бургундии</a>, государство в <a href="/wiki/1032" class="mw-redirect" title="1032">1032</a>—<a href="/wiki/1034" class="mw-redirect" title="1034">1034</a> г. вошло в состав <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%A0%D0%B8%D0%BC%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Священная Римская империя">Священной Римской империи</a>. Позднее королевство распалось на ряд более мелких <a href="/wiki/%D0%A4%D0%B5%D0%BE%D0%B4%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Феодализм">феодальных</a> владений (наиболее известные из них — графства <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D1%81" title="Прованс">Прованс</a> и <a href="/wiki/%D0%A1%D0%B0%D0%B2%D0%BE%D0%B9%D1%8F" title="Савойя">Савойя</a>, <a href="/wiki/%D0%93%D1%80%D0%B0%D1%84%D1%81%D1%82%D0%B2%D0%BE_%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D0%B8%D1%8F" class="mw-redirect" title="Графство Бургундия">пфальцграфство Бургундское</a>), бо́льшая часть которых была постепенно присоединена к <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a>. По <a href="/wiki/%D0%9B%D0%B0%D1%82%D1%8B%D0%BD%D1%8C" class="mw-redirect" title="Латынь">латинскому</a> названию своей столицы — <a href="/wiki/%D0%90%D1%80%D0%BB%D1%8C" title="Арль">Арля</a>, Бургундское королевство также получило известность под именем <b>Арелат</b> или <b>Арелатское государство</b>.
Современная <a href="/wiki/%D0%98%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B5_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D0%B8_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8" title="Исторические области Франции">историческая область</a> и <a href="/wiki/%D0%A0%D0%B5%D0%B3%D0%B8%D0%BE%D0%BD%D1%8B_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8" title="Регионы Франции">регион</a> <a href="/wiki/%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D0%B8%D1%8F" title="Бургундия">Бургундия</a> никогда не входила в состав Бургундского королевства, оставаясь <a href="/wiki/%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D0%B8%D1%8F_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3%D1%81%D1%82%D0%B2%D0%BE)" class="mw-redirect" title="Бургундия (герцогство)">герцогством</a> под <a href="/wiki/%D0%A1%D1%8E%D0%B7%D0%B5%D1%80%D0%B5%D0%BD" title="Сюзерен">сюзеренитетом</a> королей <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a>.

<div style="display:block; border:1px solid #699; vertical-align:top; background:lightyellow; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<ul><li>что, вероятно, <b><a href="/wiki/%D0%AD%D0%B4_II_(%D0%B3%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D0%B8%D0%B8)" title="Эд II (герцог Бургундии)">Эд II</a></b> был первым герцогом Бургундии, носившим флаг герцогства на своем оружии.</li>
<li>что <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_III_%D0%94%D0%BE%D0%B1%D1%80%D1%8B%D0%B9" title="Филипп III Добрый">Филипп III Добрый</a> имел 17 внебрачных детей</li>
<li>что в 1454 г. <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_III_%D0%94%D0%BE%D0%B1%D1%80%D1%8B%D0%B9" title="Филипп III Добрый">Филипп III Добрый</a> хотел устроить крестовый поход против Османской империи, однако замысел этот так и не был осуществлён.</li>
<li>что есть небольшая часть герцогства Бургундии (область Бургундии), который является теперь в области Шампанского-Арденн.</li></ul>

<p><b>Филипп <i>Добрый</i></b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">Philippe le Bon</span>) или <b>Филипп III Бургундский</b> (<span class="nowrap"><span data-wikidata-property-id="P569" class="no-wikidata"><a href="/wiki/31_%D0%B8%D1%8E%D0%BB%D1%8F" title="31 июля">31&#160;июля</a> <a href="/wiki/1396_%D0%B3%D0%BE%D0%B4" title="1396 год">1396</a></span><span class="noprint" style="display:none"> (<span class="bday">1396-07-31</span>)</span></span>, <a href="/wiki/%D0%94%D0%B8%D0%B6%D0%BE%D0%BD" title="Дижон">Дижон</a> — <span data-wikidata-property-id="P570" class="no-wikidata"><a href="/wiki/15_%D0%B8%D1%8E%D0%BD%D1%8F" title="15 июня">15 июня</a> <a href="/wiki/1467_%D0%B3%D0%BE%D0%B4" title="1467 год">1467</a></span>, <a href="/wiki/%D0%91%D1%80%D1%8E%D0%B3%D0%B3%D0%B5" title="Брюгге">Брюгге</a>) — <a href="/wiki/%D0%93%D0%B5%D1%80%D1%86%D0%BE%D0%B3_%D0%91%D1%83%D1%80%D0%B3%D1%83%D0%BD%D0%B4%D0%B8%D0%B8" class="mw-redirect" title="Герцог Бургундии">герцог Бургундии</a> с <a href="/wiki/1419" class="mw-redirect" title="1419">1419</a>. Родившийся в <a href="/wiki/%D0%94%D0%B8%D0%B6%D0%BE%D0%BD" title="Дижон">Дижоне</a>, он был старшим сыном <a href="/wiki/%D0%98%D0%BE%D0%B0%D0%BD%D0%BD_%D0%91%D0%B5%D1%81%D1%81%D1%82%D1%80%D0%B0%D1%88%D0%BD%D1%8B%D0%B9" class="mw-redirect" title="Иоанн Бесстрашный">Иоанна (Жана) Бесстрашного</a> и Маргариты Баварской, он получил титул графа де Шароле и, вероятно, тогда же был обручен с Мишель Французской (<a href="/wiki/1395" class="mw-redirect" title="1395">1395</a>—<a href="/wiki/1422" class="mw-redirect" title="1422">1422</a>), дочерью <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BB_VI_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D1%8C_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8)" title="Карл VI (король Франции)">Карла VI Французского</a> и <a href="/wiki/%D0%98%D0%B7%D0%B0%D0%B1%D0%B5%D0%BB%D0%BB%D0%B0_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8)" class="mw-redirect" title="Изабелла (королева Франции)">Изабеллы Баварской</a>. Свадьба же состоялась в <a href="/wiki/1409" class="mw-redirect" title="1409">1409</a>. После смерти жены Филипп женился вторично на Бонне д’Артуа (<a href="/wiki/1393" class="mw-redirect" title="1393">1393</a>—<a href="/wiki/1425" class="mw-redirect" title="1425">1425</a>), дочери Филиппа Артуа, которая была к тому же вдовой дяди Филиппа Доброго, <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_II_(%D0%B3%D1%80%D0%B0%D1%84_%D0%9D%D0%B5%D0%B2%D0%B5%D1%80%D0%B0)" title="Филипп II (граф Невера)">Филиппа II</a>, <a href="/wiki/%D0%93%D1%80%D0%B0%D1%84_%D0%9D%D0%B5%D0%B2%D0%B5%D1%80%D0%B0" class="mw-redirect" title="Граф Невера">графа Неверского</a>, <a href="/wiki/30_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="30 ноября">30 ноября</a>, <a href="/wiki/1424" class="mw-redirect" title="1424">1424</a>.  <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_%D0%94%D0%BE%D0%B1%D1%80%D1%8B%D0%B9" class="mw-redirect" title="Филипп Добрый">Далее..</a>
</p>
