<div class="floatleft"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Coat_of_Arms_of_Balearic_Islands.svg" class="image" title="Герб Балеарских островов"><img alt="Герб Балеарских островов" src="//upload.wikimedia.org/wikipedia/commons/thumb/c/cf/Coat_of_Arms_of_Balearic_Islands.svg/130px-Coat_of_Arms_of_Balearic_Islands.svg.png" decoding="async" width="130" height="135" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/c/cf/Coat_of_Arms_of_Balearic_Islands.svg/195px-Coat_of_Arms_of_Balearic_Islands.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/c/cf/Coat_of_Arms_of_Balearic_Islands.svg/260px-Coat_of_Arms_of_Balearic_Islands.svg.png 2x" data-file-width="917" data-file-height="950" /></a></div>
<p><b>Балеа́рские острова́</b>&#160;— <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%BF%D0%B5%D0%BB%D0%B0%D0%B3" title="Архипелаг">архипелаг</a> в <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F" title="Испания">Испании</a> на западе <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%B8%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Средиземное море">Средиземного моря</a>, рядом с восточным берегом <a href="/wiki/%D0%9F%D0%B8%D1%80%D0%B5%D0%BD%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Пиренейский полуостров">Пиренейского полуострова</a>.
</p><p>Четыре самых крупных острова: <a href="/wiki/%D0%9C%D0%B0%D0%BB%D1%8C%D0%BE%D1%80%D0%BA%D0%B0" title="Мальорка">Мальорка</a>, <a href="/wiki/%D0%9C%D0%B5%D0%BD%D0%BE%D1%80%D0%BA%D0%B0" title="Менорка">Менорка</a>, <a href="/wiki/%D0%98%D0%B2%D0%B8%D1%81%D0%B0_(%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2)" class="mw-redirect" title="Ивиса (остров)">Ивиса</a> (больше известная как Ибица) и <a href="/wiki/%D0%A4%D0%BE%D1%80%D0%BC%D0%B5%D0%BD%D1%82%D0%B5%D1%80%D0%B0" title="Форментера">Форментера</a>. Архипелаг является <a href="/wiki/%D0%90%D0%B4%D0%BC%D0%B8%D0%BD%D0%B8%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE%D0%B5_%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5_%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D0%B8" title="Административное деление Испании">автономным сообществом</a> и <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B2%D0%B8%D0%BD%D1%86%D0%B8%D0%B8_%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D0%B8" title="Провинции Испании">провинцией</a> Испании со столицей <a href="/wiki/%D0%9F%D0%B0%D0%BB%D1%8C%D0%BC%D0%B0-%D0%B4%D0%B5-%D0%9C%D0%B0%D0%B9%D0%BE%D1%80%D0%BA%D0%B0" class="mw-redirect" title="Пальма-де-Майорка">Пальмой</a>. Официальные языки на Балеарских островах&#160;— <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BB%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Каталанский язык">каталанский</a> и <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Испанский язык">испанский</a>. На настоящий момент автономный устав гласит, что Балеарские острова являются одним из <i>народов</i> Испании.
</p><p>Официальное название Балеарских островов на каталанском <i>Illes Balears</i>, хотя на испанском <i>Islas Baleares</i>. Название «Балеарский» уходит корнями в <a href="/wiki/%D0%93%D1%80%D0%B5%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Греческий язык">греческий</a> (<span lang="grc">Γυμνησίαι</span>/<i> Гимнесия</i> и <span lang="grc">Βαλλιαρεῖς</span>/<i>Баллиарьес</i>). На <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">латинском языке</a> (<i>Балеарес</i>).
</p><p>Что касается различных теорий о происхождении двух древних греческих и латинских названиях островов&#160;— <i>Гимнесия</i> и <i>Балеарес</i>&#160;— классические источники подтверждают обе.
</p><p><a href="/wiki/%D0%91%D0%B0%D0%BB%D0%B5%D0%B0%D1%80%D1%81%D0%BA%D0%B8%D0%B5_%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2%D0%B0" title="Балеарские острова"><b>Далее …</b></a>

<li>2. Перевести <a class="external text" href="http://tools.wmflabs.org/joanjoc/sugart.php?l1=en&amp;pr=wiki&amp;l2=ru&amp;cat=Balearic+Islands&amp;dpt=3&amp;tpl=&amp;uselang=en&amp;go=Proceed">статьи из английской Википедии из категории «Balearic Islands» без русских интервик</a></li>

<li>4.
