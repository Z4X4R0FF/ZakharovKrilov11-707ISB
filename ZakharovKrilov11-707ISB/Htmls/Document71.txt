<div style="display:block; border:1px solid #C0C0C0; vertical-align:top; background:white; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<p><b><a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D0%B8%D1%8F" title="Болгария">Болгария</a></b>&#160;— <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a> в Юго-Восточной Европе, в восточной части <a href="/wiki/%D0%91%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Балканский полуостров">Балканского полуострова</a>. Занимает 22&#160;% его площади. Страна получила название по этнониму народа&#160;— <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D1%8B" title="Болгары">болгары</a>.
</p>
<div class="floatright"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Flag_map_of_Bulgaria.svg" class="image"><img alt="Flag map of Bulgaria.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/72/Flag_map_of_Bulgaria.svg/150px-Flag_map_of_Bulgaria.svg.png" decoding="async" width="150" height="99" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/72/Flag_map_of_Bulgaria.svg/225px-Flag_map_of_Bulgaria.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/72/Flag_map_of_Bulgaria.svg/300px-Flag_map_of_Bulgaria.svg.png 2x" data-file-width="797" data-file-height="524" /></a></div>
<p>С востока омывается <a href="/wiki/%D0%A7%D1%91%D1%80%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Чёрное море">Чёрным морем</a>. Граничит с <a href="/wiki/%D0%93%D1%80%D0%B5%D1%86%D0%B8%D1%8F" title="Греция">Грецией</a> и <a href="/wiki/%D0%A2%D1%83%D1%80%D1%86%D0%B8%D1%8F" title="Турция">Турцией</a> на юге, с <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B1%D0%B8%D1%8F" title="Сербия">Сербией</a> и <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%9C%D0%B0%D0%BA%D0%B5%D0%B4%D0%BE%D0%BD%D0%B8%D1%8F" title="Северная Македония">Северной Македонией</a>&#160;— на западе и с <a href="/wiki/%D0%A0%D1%83%D0%BC%D1%8B%D0%BD%D0%B8%D1%8F" title="Румыния">Румынией</a>&#160;— на севере.
</p>

<p><b><a href="/wiki/%D0%9C%D1%83%D1%81%D0%B0%D0%BA%D0%B0" title="Мусака">Мусака</a></b>&#160;— традиционное блюдо из <a href="/wiki/%D0%91%D0%B0%D0%BA%D0%BB%D0%B0%D0%B6%D0%B0%D0%BD" title="Баклажан">баклажанов</a> на <a href="/wiki/%D0%91%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D1%8B" class="mw-redirect" title="Балканы">Балканах</a> и <a href="/wiki/%D0%91%D0%BB%D0%B8%D0%B6%D0%BD%D0%B8%D0%B9_%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA" title="Ближний Восток">Ближнем Востоке</a>.
</p><p>В <a href="/wiki/%D0%91%D0%BE%D0%BB%D0%B3%D0%B0%D1%80%D0%B8%D1%8F" title="Болгария">Болгарии</a> мусака&#160;— это запечённая смесь фарша и картофеля с заливкой. А блюдо наподобие греческой мусаки, состоящее из различных овощей, картофеля и мяса, называют «<b>гювеч</b>». Слово «гювеч» имеет в болгарском языке и переносный смысл&#160;— <a href="/wiki/%D0%A5%D0%B0%D0%BB%D1%8F%D0%B2%D0%B0" class="mw-disambig" title="Халява">халява</a>.
</p>
