<p><b>Барсело́на</b> (<a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BB%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Каталанский язык">кат.</a> <span lang="ca" style="font-style:italic;">Barcelona</span> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[bəɾsəˈɫonə]</a></span>, <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Испанский язык">исп.</a>&#160;<span lang="es"><i>Barcelona</i></span> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[baɾθeˈlona]</a></span>)&#160;— город в <a href="/wiki/%D0%98%D1%81%D0%BF%D0%B0%D0%BD%D0%B8%D1%8F" title="Испания">Испании</a>, столица <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BB%D0%BE%D0%BD%D0%B8%D1%8F_(%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%BD%D0%BE%D0%B5_%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE)" class="mw-redirect" title="Каталония (автономное сообщество)">автономной области Каталония</a> и <a href="/wiki/%D0%91%D0%B0%D1%80%D1%81%D0%B5%D0%BB%D0%BE%D0%BD%D0%B0_(%D0%BF%D1%80%D0%BE%D0%B2%D0%B8%D0%BD%D1%86%D0%B8%D1%8F)" title="Барселона (провинция)">одноимённой провинции</a>. Порт на <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%B8%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Средиземное море">Средиземном море</a> в 120&#160;км от границы <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a>. Крупнейший промышленный и торговый центр Испании. Один из важнейших туристических пунктов в европейских маршрутах. 
</p><p>Население Барселоны&#160;— 1 615 908 человек (INE 2008). Это второй город в Испании по численности населения после <a href="/wiki/%D0%9C%D0%B0%D0%B4%D1%80%D0%B8%D0%B4" title="Мадрид">Мадрида</a> и десятый в <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7" title="Европейский союз">Европейском союзе</a>. В пригородах города проживает 3,2 млн. Барселона насчитывает почти 5 млн жителей.
</p><p>В <a href="/wiki/1992_%D0%B3%D0%BE%D0%B4" title="1992 год">1992 году</a> Барселона принимала <a href="/wiki/%D0%9B%D0%B5%D1%82%D0%BD%D0%B8%D0%B5_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B_1992" title="Летние Олимпийские игры 1992">летние Олимпийские игры</a>. В <a href="/wiki/2004_%D0%B3%D0%BE%D0%B4" title="2004 год">2004 году</a> в Барселоне проходил Всемирный культурный форум 2004. В марте <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 года</a> Барселона стала столицей <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%B8%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%BC%D0%BE%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7" class="mw-redirect" title="Средиземноморский союз">Средиземноморского союза</a>, в который входят 43 страны. В 2013 году в Барселоне проходил <a href="/wiki/%D0%A7%D0%B5%D0%BC%D0%BF%D0%B8%D0%BE%D0%BD%D0%B0%D1%82_%D0%BC%D0%B8%D1%80%D0%B0_%D0%BF%D0%BE_%D0%B2%D0%BE%D0%B4%D0%BD%D1%8B%D0%BC_%D0%B2%D0%B8%D0%B4%D0%B0%D0%BC_%D1%81%D0%BF%D0%BE%D1%80%D1%82%D0%B0_2013" title="Чемпионат мира по водным видам спорта 2013">чемпионат мира по водным видам спорта</a>.
</p><p>Барселона расположена на северо-востоке <a href="/wiki/%D0%98%D0%B1%D0%B5%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" class="mw-redirect" title="Иберийский полуостров">Иберийского полуострова</a> на побережье <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%B8%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Средиземное море">Средиземного моря</a> на плато шириной в 5&#160;км, границы которого с юга составляют горная гряда <a href="/w/index.php?title=%D0%9A%D0%BE%D0%BB%D1%8C%D1%81%D0%B5%D1%80%D0%BE%D0%BB%D0%B0&amp;action=edit&amp;redlink=1" class="new" title="Кольсерола (страница отсутствует)">Кольсерола</a> (<a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BB%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Каталанский язык">кат.</a> <span lang="ca" style="font-style:italic;">Collserola</span>) и река <a href="/wiki/%D0%9B%D1%8C%D0%BE%D0%B1%D1%80%D0%B5%D0%B3%D0%B0%D1%82_(%D1%80%D0%B5%D0%BA%D0%B0)" class="mw-redirect" title="Льобрегат (река)">Льобрегат</a>, а на севере&#160;— река <a href="/wiki/%D0%91%D0%B5%D1%81%D0%BE%D1%81" title="Бесос">Бесос</a>. <a href="/wiki/%D0%9F%D0%B8%D1%80%D0%B5%D0%BD%D0%B5%D0%B8" title="Пиренеи">Пиренеи</a> находятся приблизительно в 120&#160;км к северу от города.

<ul><li>1. Написать или дополнить какую-нибудь статью из списка <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B5%D0%BA%D1%82:%D0%A2%D0%B5%D0%BC%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BD%D0%B5%D0%B4%D0%B5%D0%BB%D1%8F_%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%BE%D0%B2/%D0%91%D0%B0%D1%80%D1%81%D0%B5%D0%BB%D0%BE%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BD%D0%B5%D0%B4%D0%B5%D0%BB%D1%8F" title="Проект:Тематическая неделя городов/Барселонская неделя">Барселонской недели</a>.</li>
<li>2. Перевести <a class="external text" href="http://tools.wmflabs.org/joanjoc/sugart.php?l1=en&amp;pr=wiki&amp;l2=ru&amp;cat=Barcelona&amp;dpt=3&amp;tpl=&amp;uselang=en&amp;go=Proceed">статьи из английской Википедии из категории «Barcelona» без русских интервик</a></li>
<li>4. Улучшить дерево категорий по теме Барселона.</li>
<li>5. Вручить <a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%9E%D1%80%D0%B4%D0%B5%D0%BD%D0%B0/%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B0%D1%81%D0%BB%D1%83%D0%B3%D0%B8" title="Википедия:Ордена/Национальные заслуги">вики-награду</a> «<a href="/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%9E%D1%80%D0%B4%D0%B5%D0%BD%D0%B0/%D0%9D%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B0%D1%81%D0%BB%D1%83%D0%B3%D0%B8#Испания" title="Википедия:Ордена/Национальные заслуги">Испанская звезда</a>» редактору, вклад которого в развитие барселонской тематики и (или) в целом испанской в русском разделе Википедии, вы считаете существенным.</li></ul>
<p>Пожалуйста пользуйтесь <a href="/wiki/%D0%9A%D0%B0%D1%82%D0%B0%D0%BB%D0%B0%D0%BD%D1%81%D0%BA%D0%BE-%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%B0%D0%BA%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%86%D0%B8%D1%8F" title="Каталанско-русская практическая транскрипция">правилами каталанско-русской транскрипции</a>.
