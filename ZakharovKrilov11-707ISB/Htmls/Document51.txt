<li>Едва отразив нападение <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA_%D0%90%D0%BD%D1%82%D0%BE%D0%BD%D0%B8%D0%B9" title="Марк Антоний">римского полководца</a>, <b><a href="/wiki/%D0%90%D1%80%D1%82%D0%B0%D0%B2%D0%B0%D0%B7%D0%B4_I_(%D0%BF%D1%80%D0%B0%D0%B2%D0%B8%D1%82%D0%B5%D0%BB%D1%8C_%D0%90%D1%82%D1%80%D0%BE%D0%BF%D0%B0%D1%82%D0%B5%D0%BD%D1%8B)" title="Артавазд I (правитель Атропатены)">царь</a></b> <a href="/wiki/%D0%90%D1%82%D1%80%D0%BE%D0%BF%D0%B0%D1%82%D0%B5%D0%BD%D0%B0" title="Атропатена">Атропатены</a> предложил ему союз против <a href="/wiki/%D0%90%D1%80%D1%82%D0%B0%D0%B2%D0%B0%D0%B7%D0%B4_II" title="Артавазд II">своего тёзки</a>.</li>
<li>Возможно, <a href="/wiki/%D0%92%D0%B0%D1%81%D0%BA%D0%BE_%D0%B4%D0%B0_%D0%93%D0%B0%D0%BC%D0%B0" title="Васко да Гама">Васко да Гаму</a> <b><a href="/wiki/%D0%9F%D0%BB%D0%B0%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5_%D1%84%D0%B8%D0%BD%D0%B8%D0%BA%D0%B8%D0%B9%D1%86%D0%B5%D0%B2_%D0%B2%D0%BE%D0%BA%D1%80%D1%83%D0%B3_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B8" title="Плавание финикийцев вокруг Африки">опередили на 2000 лет</a></b>.</li>
<li>На <b><a href="/wiki/%D0%93%D0%B5%D0%BA%D1%81%D0%B5%D1%80%D0%B0" title="Гексера">гексерах</a></b> никогда не было шести рядов вёсел, а вот <a href="/wiki/%D0%A1%D0%BA%D0%BE%D1%80%D0%BF%D0%B8%D0%BE%D0%BD_(%D0%BC%D0%B5%D1%82%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%BC%D0%B0%D1%88%D0%B8%D0%BD%D0%B0)" title="Скорпион (метательная машина)">скорпионов</a> действительно могло быть шесть.</li>
<li>Во время <b><a href="/wiki/%D0%92%D0%BE%D1%81%D1%81%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_%D0%BD%D0%B0%D1%91%D0%BC%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2" class="mw-redirect" title="Восстание наёмников">восстания наёмников</a></b> <a href="/wiki/%D0%9A%D0%B0%D1%80%D1%84%D0%B0%D0%B3%D0%B5%D0%BD_(%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE)" title="Карфаген (государство)">Карфагену</a> оказали поддержку <a href="/wiki/%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B8%D0%B9_%D0%A0%D0%B8%D0%BC" title="Древний Рим">Рим</a> и <a href="/wiki/%D0%A1%D0%B8%D1%80%D0%B0%D0%BA%D1%83%D0%B7%D1%8B" title="Сиракузы">Сиракузы</a>, только что бывшие его противниками в <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%9F%D1%83%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая Пуническая война">Первой Пунической войне</a>.</li>
<li>В древних <a href="/wiki/%D0%9F%D0%BE%D0%BC%D0%BF%D0%B5%D0%B8" title="Помпеи">Помпеях</a> жители и гости города легко могли найти городской <a href="/wiki/%D0%9B%D1%83%D0%BF%D0%B0%D0%BD%D0%B0%D1%80%D0%B8%D0%B9" title="Лупанарий">лупанарий</a>, ориентируясь по стрелкам в виде фаллического символа, вырубленным прямо на камнях мостовой.</li>
<li>Во время похода на <a href="/wiki/%D0%92%D0%B0%D0%B2%D0%B8%D0%BB%D0%BE%D0%BD" title="Вавилон">Вавилон</a> в 539 г. до н. э. персидский царь <a href="/wiki/%D0%9A%D0%B8%D1%80_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B8%D0%B9" class="mw-redirect" title="Кир Великий">Кир Великий</a> приказал на время остановить наступление и с помощью сооружения каналов «наказать», то есть стереть с лица земли, реку <a href="/wiki/%D0%94%D0%B8%D1%8F%D0%BB%D0%B0" class="mw-disambig" title="Дияла">Диялу</a>, в которой утонула священная белая лошадь…</li>
<li>Журнал <a href="/wiki/Forbes_(%D0%B6%D1%83%D1%80%D0%BD%D0%B0%D0%BB)" class="mw-redirect" title="Forbes (журнал)">Forbes</a> подсчитал, что состояние <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA_%D0%9B%D0%B8%D1%86%D0%B8%D0%BD%D0%B8%D0%B9_%D0%9A%D1%80%D0%B0%D1%81%D1%81" title="Марк Лициний Красс">Марка Лициния Красса</a> в современном эквиваленте составило бы 169,8 миллиардов <a href="/wiki/%D0%94%D0%BE%D0%BB%D0%BB%D0%B0%D1%80_%D0%A1%D0%A8%D0%90" title="Доллар США">долларов</a>.</li>
<li><a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D0%93%D1%80%D0%B5%D1%86%D0%B8%D1%8F" title="Великая Греция">Великая Греция</a> находилась в <a href="/wiki/%D0%90%D0%BF%D0%BF%D0%B5%D0%BD%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" class="mw-redirect" title="Аппенинский полуостров">Южной Италии</a>.</li>
<li><b><a href="/wiki/%D0%92%D0%B5%D0%B7%D1%83%D0%B2%D0%B8%D0%B9" title="Везувий">Вулкан</a></b> стал убежищем для <a href="/wiki/%D0%92%D0%BE%D1%81%D1%81%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_%D0%A1%D0%BF%D0%B0%D1%80%D1%82%D0%B0%D0%BA%D0%B0" title="Восстание Спартака">восставших рабов</a>.</li></ul>
