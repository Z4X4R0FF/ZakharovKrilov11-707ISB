<td width="65%" valign="top" style="background:&#123;&#123;&#123;bakgrund_färg&#125;&#125;&#125;; padding-left:10px; padding-bottom:2em; padding-right:20px; padding-top:1em;"><b><a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Англиканство">Англиканство</a></b> — одно из направлений <a href="/wiki/%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%B8%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Христианство">Христианства</a>. Англиканские <a href="/wiki/%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Христианская церковь">церкви</a> либо имеют особую историческую связь с <a href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B8" title="Церковь Англии">Церковью Англии</a>, либо объединены с ней общими <a href="/wiki/%D0%A2%D0%B5%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" class="mw-redirect" title="Теология">теологией</a>, <a href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D1%81%D0%BB%D1%83%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Богослужение">богослужением</a> и церковной структурой. Термин «Англиканство» восходит к латинской фразе «ecclesia anglicana», первое упоминание которой относится к 1246 году и означает в дословном переводе на русский язык «Английская Церковь». Последователи Англиканства именуются англиканами, а также епископалианами.
<p>Абсолютное большинство англикан принадлежат к церквям, входящим в <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%A1%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" class="mw-redirect" title="Англиканское Сообщество">Англиканское Сообщество</a>, которое носит международный характер, представляет собой ассоциацию этих церквей, состоящих в полном каноническом общении с <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%9A%D0%B5%D0%BD%D1%82%D0%B5%D1%80%D0%B1%D0%B5%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9" title="Архиепископ Кентерберийский">Архиепископом Кентерберийским</a> и насчитывает около 77 миллионов последователей. Англиканское Сообщество занимает третье место по численности среди направлений Христианства после Римско-католической церкви и Вселенского Православия. Англиканское вероучение основывается на <a href="/wiki/%D0%A1%D0%B2%D1%8F%D1%89%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BF%D0%B8%D1%81%D0%B0%D0%BD%D0%B8%D1%8F" class="mw-redirect" title="Священные писания">Писаниях</a>, традициях Апостольской церкви и учении ранних Отцов церкви.

<td style="padding-top:1em;">
<p><b><a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%A1%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE" class="mw-redirect" title="Англиканское Сообщество">Англиканское Сообщество</a></b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">The Anglican Communion</span>) — всемирное объединение поместных (национальных) <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D1%82%D0%B2%D0%BE" title="Англиканство">англиканских</a> <a href="/wiki/%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%B8%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Христианская церковь">церквей</a>. Не существует единой «Англиканской Церкви», обладающей общей юрисдикцией, так как каждая национальная англиканская Церковь — административно независима и находятся в полном каноническом единстве с <a href="/wiki/%D0%A6%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C_%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B8" title="Церковь Англии">Церковью Англии</a>, которая рассматривается как «Материнская Церковь» Сообщества по всему миру, с её главным <a href="/wiki/%D0%9F%D1%80%D0%B8%D0%BC%D0%B0%D1%81" title="Примас">примасом</a> <a href="/wiki/%D0%90%D1%80%D1%85%D0%B8%D0%B5%D0%BF%D0%B8%D1%81%D0%BA%D0%BE%D0%BF_%D0%9A%D0%B5%D0%BD%D1%82%D0%B5%D1%80%D0%B1%D0%B5%D1%80%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9" title="Архиепископ Кентерберийский">Архиепископом Кентерберийским</a>. Статус полного канонического единства означает, что между этими Церквями существует взаимное согласие по важнейшим доктринам и <a href="/wiki/%D0%95%D0%B2%D1%85%D0%B0%D1%80%D0%B8%D1%81%D1%82%D0%B8%D1%8F" title="Евхаристия">Причастие</a> в каждой поместной Церкви возможно для каждого англиканина.

<p>Упоминание о мученичестве Альбана является одним из первых исторических свидетельств о христианстве в Британии. Согласно сообщениям историков <a href="/wiki/%D0%93%D0%B8%D0%BB%D1%8C%D0%B4%D0%B0" class="mw-redirect" title="Гильда">Гильды</a> (<a href="/wiki/VI_%D0%B2%D0%B5%D0%BA" title="VI век">VI век</a>) и <a href="/wiki/%D0%91%D0%B5%D0%B4%D0%B0_%D0%94%D0%BE%D1%81%D1%82%D0%BE%D0%BF%D0%BE%D1%87%D1%82%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9" title="Беда Достопочтенный">Беды Достопочтенного</a>, Альбан погиб во время гонений на христиан <a href="/wiki/302" class="mw-redirect" title="302">302</a>—<a href="/wiki/305" class="mw-redirect" title="305">305</a> при императоре <a href="/wiki/%D0%94%D0%B8%D0%BE%D0%BA%D0%BB%D0%B5%D1%82%D0%B8%D0%B0%D0%BD" title="Диоклетиан">Диоклетиане</a>. По другим данным его кончина относится ко времени императора <a href="/wiki/%D0%A1%D0%B5%D0%BF%D1%82%D0%B8%D0%BC%D0%B8%D0%B9_%D0%A1%D0%B5%D0%B2%D0%B5%D1%80" title="Септимий Север">Септимия Севера</a> (около <a href="/wiki/200" class="mw-redirect" title="200">200</a>) или императора <a href="/wiki/%D0%94%D0%B5%D1%86%D0%B8%D0%B9" class="mw-redirect" title="Деций">Деция</a> (<a href="/wiki/249" class="mw-redirect" title="249">249</a>—<a href="/wiki/251" class="mw-redirect" title="251">251</a>).
</p><p>Альбан был римским воином. Обратившись в христианство под влиянием священника <a href="/wiki/%D0%90%D0%BC%D1%84%D0%B8%D0%B1%D0%B0%D0%BB" title="Амфибал">Амфибала</a>, которого он укрывал во время преследований, Альбан был осуждён на смерть и обезглавлен в городе <a href="/wiki/%D0%92%D0%B5%D1%80%D1%83%D0%BB%D0%B0%D0%BC%D0%B8%D1%83%D0%BC" class="mw-redirect" title="Веруламиум">Веруламиуме</a>, ныне&#160;— город <a href="/wiki/%D0%A1%D0%B5%D0%BD%D1%82-%D0%9E%D0%BB%D0%B1%D0%B0%D0%BD%D1%81_(%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D1%8F)" title="Сент-Олбанс (Англия)">Сент-Олбанс</a>, названный в честь святого.
</p>

Позднее по проекту архитектора Бориса Викторовича Фрейденберга был построен двухэтажный дом для священника.
