<p><b>Dan Farmer</b> (born April 5, 1962) is an American <a href="/wiki/Computer_security" title="Computer security">computer security</a> researcher and <a href="/wiki/Programmer" title="Programmer">programmer</a> who was a pioneer in the development of <a href="/wiki/Vulnerability_scanner" title="Vulnerability scanner">vulnerability scanners</a> for <a href="/wiki/Unix" title="Unix">Unix</a> <a href="/wiki/Operating_system" title="Operating system">operating systems</a> and <a href="/wiki/Computer_network" title="Computer network">computer networks</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>
</p>

<h2><span class="mw-headline" id="Life_and_career">Life and career</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Dan_Farmer&amp;action=edit&amp;section=1" title="Edit section: Life and career">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Farmer developed his first <a href="/wiki/Software_suite" title="Software suite">software suite</a> while he was a <a href="/wiki/Computer_science" title="Computer science">computer science</a> student at <a href="/wiki/Purdue_University" title="Purdue University">Purdue University</a> in 1989. <a href="/wiki/Gene_Spafford" title="Gene Spafford">Gene Spafford</a>, one of his professors, helped him to start the project. The software, called the <a href="/wiki/Computer_Oracle_and_Password_System" class="mw-redirect" title="Computer Oracle and Password System">Computer Oracle and Password System</a> (COPS), comprises several small, specialized <a href="/wiki/Vulnerability_scanner" title="Vulnerability scanner">vulnerability scanners</a> designed to identify security weaknesses in one part of a Unix operating system.
</p><p>In 1995, Farmer and <a href="/wiki/Wietse_Venema" title="Wietse Venema">Wietse Venema</a> (a Dutch programmer and physicist) developed a second vulnerability scanner called the <a href="/wiki/Security_Administrator_Tool_for_Analyzing_Networks" title="Security Administrator Tool for Analyzing Networks">Security Administrator Tool for Analyzing Networks</a> (SATAN). Due to a misunderstanding of SATAN's capabilities, when it was first published, some <a href="/wiki/Network_administrator" title="Network administrator">network administrators</a> and law enforcement personnel believed that <a href="/wiki/Hacker_(computer_security)" class="mw-redirect" title="Hacker (computer security)">hackers</a> would use it to identify and break into vulnerable computers. Consequently, <a href="/wiki/Silicon_Graphics" title="Silicon Graphics">SGI</a> terminated Farmer's employment.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>  However, contrary to popular opinion, SATAN did not function as an automatic hacking program that undermined network security. Rather, it operated as an audit on network security that identified vulnerabilities and made suggestions to help prevent them. No information about how security vulnerabilities could be exploited was provided by the tool.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup> Within a few years, the use of <a href="/wiki/Vulnerability_scanner" title="Vulnerability scanner">vulnerability scanners</a> such as SATAN became an accepted method for auditing computer and network security.
</p><p>He co-developed the Titan vulnerability scanner with Brad Powell and Matt Archibald, which they presented at the <a href="/wiki/Large_Installation_System_Administration_Conference" title="Large Installation System Administration Conference">Large Installation System Administration Conference</a> (LISA) in 1998.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup>
</p><p>Farmer and Venema collaborated again to develop a <a href="/wiki/Computer_forensics" title="Computer forensics">computer forensics</a> suite called <a href="/wiki/The_Coroner%27s_Toolkit" title="The Coroner&#39;s Toolkit">The Coroner's Toolkit</a>, and later coauthored <i>Forensic Discovery</i> (2005), a book about computer forensics.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p><p>Farmer co-founded Elemental Security with Dayne Myers, and served as the corporation's <a href="/wiki/Chief_technical_officer" class="mw-redirect" title="Chief technical officer">chief technical officer</a>.
</p>
