<div style="display:block; border:1px solid #1C39BB; vertical-align:top; background:#FFFACD; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<p><b><a href="/wiki/%D0%92%D0%B5%D0%BB%D0%BE%D1%81%D0%B8%D0%BF%D0%B5%D0%B4%D0%BD%D1%8B%D0%B9_%D1%81%D0%BF%D0%BE%D1%80%D1%82" title="Велосипедный спорт">Велоспорт</a></b> (<b>Велосипедный спорт</b>) — это перемещение по земле с использованием транспортных средств (<a href="/wiki/%D0%92%D0%B5%D0%BB%D0%BE%D1%81%D0%B8%D0%BF%D0%B5%D0%B4" title="Велосипед">велосипедов</a>), движимых мускульной силой человека.
</p><p>Велосипедный спорт включает в себя такие дисциплины, как <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%BE%D0%B3%D0%BE%D0%BD%D0%BA%D0%B0" class="mw-redirect" title="Велогонка">гонки</a> на <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%BE%D1%82%D1%80%D0%B5%D0%BA%D0%BE%D0%B2%D1%8B%D0%B5_%D0%B3%D0%BE%D0%BD%D0%BA%D0%B8" class="mw-redirect" title="Велотрековые гонки">треке</a>, <a href="/wiki/%D0%A8%D0%BE%D1%81%D1%81%D0%B5%D0%B9%D0%BD%D1%8B%D0%B9_%D0%B2%D0%B5%D0%BB%D0%BE%D1%81%D0%BF%D0%BE%D1%80%D1%82" title="Шоссейный велоспорт">шоссе</a> и <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%BE%D0%BA%D1%80%D0%BE%D1%81%D1%81" title="Велокросс">пересеченной местности</a>, <a href="/w/index.php?title=%D0%92%D0%B5%D0%BB%D0%BE%D1%81%D0%BF%D0%BE%D1%80%D1%82-%D0%BC%D0%B0%D1%83%D0%BD%D1%82%D0%B8%D0%BD%D0%B1%D0%B0%D0%B9%D0%BA&amp;action=edit&amp;redlink=1" class="new" title="Велоспорт-маунтинбайк (страница отсутствует)">маунтинбайк</a>, <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%BE%D1%82%D1%80%D0%B8%D0%B0%D0%BB" title="Велотриал">велотриал</a>, <a href="/wiki/BMX" class="mw-disambig" title="BMX">BMX</a>, соревнования в фигурной езде и игре в мяч на велосипедах — <a href="/w/index.php?title=%D0%92%D0%B5%D0%BB%D0%BE%D0%BF%D0%BE%D0%BB%D0%BE&amp;action=edit&amp;redlink=1" class="new" title="Велополо (страница отсутствует)">велополо</a> и <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%BE%D0%B1%D0%BE%D0%BB" title="Велобол">велобол</a> и др. Основная цель гоночных дисциплин — наиболее быстрое преодоление дистанции.
</p><p>Как вид спорта, велосипедный спорт управляется <a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%81%D0%BE%D1%8E%D0%B7_%D0%B2%D0%B5%D0%BB%D0%BE%D1%81%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%81%D1%82%D0%BE%D0%B2" title="Международный союз велосипедистов">Международным союзом велосипедистов</a>, находящимся в Швейцарии. В <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a> аналогичные функции выполняет <a href="/wiki/%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F_%D0%B2%D0%B5%D0%BB%D0%BE%D1%81%D0%B8%D0%BF%D0%B5%D0%B4%D0%BD%D0%BE%D0%B3%D0%BE_%D1%81%D0%BF%D0%BE%D1%80%D1%82%D0%B0_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Федерация велосипедного спорта России">Федерация велосипедного спорта России</a>.
</p>

<p><b><a href="/wiki/%D0%94%D0%B6%D0%B8%D1%80%D0%BE_%D0%B4%E2%80%99%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Джиро д’Италия">Джиро д’Италия</a> <a href="/wiki/2009" class="mw-redirect" title="2009">2009</a></b> (<a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D1%8C%D1%8F%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Итальянский язык">итал.</a>&#160;<span lang="it" style="font-style:italic;">Giro d'Italia 2009</span>) — 92-я по счёту <a href="/wiki/%D0%A1%D1%83%D0%BF%D0%B5%D1%80%D0%B2%D0%B5%D0%BB%D0%BE%D0%BC%D0%BD%D0%BE%D0%B3%D0%BE%D0%B4%D0%BD%D0%B5%D0%B2%D0%BA%D0%B0" class="mw-redirect" title="Супервеломногодневка">супервеломногодневка</a> по дорогам <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италии</a>, проходила с 9 по 31 мая 2009 года, в столетний юбилей гонки. Двадцать две команды, включавшие 198 гонщиков, соревновались на 21 этапе, стартовав в <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B5%D1%86%D0%B8%D1%8F" title="Венеция">Венеции</a> и финишировав в <a href="/wiki/%D0%A0%D0%B8%D0%BC" title="Рим">Риме</a>.
