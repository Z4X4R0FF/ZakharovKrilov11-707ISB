<p><b>Дворец Грешем</b> (также <b>Дворец Грешема</b>, <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B3%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Венгерский язык">венг.</a> <span lang="hu" style="font-style:italic;">Gresham-palota</span>)&#160;— здание в <a href="/wiki/%D0%91%D1%83%D0%B4%D0%B0%D0%BF%D0%B5%D1%88%D1%82" title="Будапешт">Будапеште</a>, яркий образец стиля <a href="/wiki/%D0%9C%D0%BE%D0%B4%D0%B5%D1%80%D0%BD" title="Модерн">модерн</a> в архитектуре. Построенное как офисно-жилой комплекс в начале <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a>, ныне используется как один из люксовых отелей сети <a href="/wiki/Four_Seasons_Hotels_and_Resorts" title="Four Seasons Hotels and Resorts">Four Seasons Hotels and Resorts</a>.
</p><p>Дворец был назван в честь <a href="/wiki/%D0%93%D1%80%D0%B5%D1%88%D0%B5%D0%BC,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81" title="Грешем, Томас">Томаса Грешема</a>, английского купца и финансиста <a href="/wiki/XVI_%D0%B2%D0%B5%D0%BA" title="XVI век">XVI века</a>, основателя <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%B1%D0%B8%D1%80%D0%B6%D0%B0_(%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD)" title="Королевская биржа (Лондон)">Королевской биржи</a> в <a href="/wiki/%D0%9B%D0%BE%D0%BD%D0%B4%D0%BE%D0%BD" title="Лондон">Лондоне</a>.
</p><p><i>(<a href="/wiki/%D0%94%D0%B2%D0%BE%D1%80%D0%B5%D1%86_%D0%93%D1%80%D0%B5%D1%88%D0%B5%D0%BC" title="Дворец Грешем">далее…</a>)</i>

<ul><li><b><a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B8%D0%B3,_%D0%A8%D0%B0%D1%80%D0%B0" title="Кариг, Шара">Венгерская социал-демократка</a></b> спасала <a href="/wiki/%D0%95%D0%B2%D1%80%D0%B5%D0%B8" title="Евреи">евреев</a>, разоблачала фальсификацию выборов, попала за это в <a href="/wiki/%D0%92%D0%BE%D1%80%D0%BA%D1%83%D1%82%D0%BB%D0%B0%D0%B3" title="Воркутлаг">Воркутлаг</a>, а после возвращения на родину переводила <a href="/wiki/%D0%91%D1%83%D0%BB%D0%B3%D0%B0%D0%BA%D0%BE%D0%B2,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%90%D1%84%D0%B0%D0%BD%D0%B0%D1%81%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Булгаков, Михаил Афанасьевич">Булгакова</a>.</li></ul>
<div class="floatright"><a href="/wiki/%D0%A3%D0%BD%D0%B8%D0%BA%D1%83%D0%BC_(%D0%B1%D0%B0%D0%BB%D1%8C%D0%B7%D0%B0%D0%BC)" title="Уникум (бальзам)"><img alt="Unicum bottle.JPG" src="//upload.wikimedia.org/wikipedia/commons/thumb/4/44/Unicum_bottle.JPG/95px-Unicum_bottle.JPG" decoding="async" width="95" height="127" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/44/Unicum_bottle.JPG/143px-Unicum_bottle.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/44/Unicum_bottle.JPG/190px-Unicum_bottle.JPG 2x" data-file-width="1536" data-file-height="2048" /></a></div>
<ul><li>Капиталист, покинувший социалистическую Венгрию, лишил её <b><a href="/wiki/%D0%A3%D0%BD%D0%B8%D0%BA%D1%83%D0%BC_(%D0%B1%D0%B0%D0%BB%D1%8C%D0%B7%D0%B0%D0%BC)" title="Уникум (бальзам)">любимого ликёра</a></b> (<i>на илл.</i>) на весь социалистический период.</li>
<li>Чтобы попасть на <a href="/wiki/%D0%9B%D0%B5%D1%82%D0%BD%D0%B8%D0%B5_%D0%9E%D0%BB%D0%B8%D0%BC%D0%BF%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D0%B3%D1%80%D1%8B_1912" title="Летние Олимпийские игры 1912">Олимпийские игры</a>, <b><a href="/wiki/%D0%9A%D0%BE%D1%86%D0%B0%D0%BD,_%D0%9C%D0%BE%D1%80" title="Коцан, Мор">венгерскому пастору</a></b> пришлось взять псевдоним.</li>
<li><a href="/wiki/%D0%93%D0%B5%D0%B7%D0%B0_II" title="Геза II">Все</a> <a href="/wiki/%D0%9B%D0%B0%D1%81%D0%BB%D0%BE_II" title="Ласло II">три</a> <a href="/wiki/%D0%98%D1%88%D1%82%D0%B2%D0%B0%D0%BD_IV" title="Иштван IV">сына</a> <b><a href="/wiki/%D0%98%D0%BB%D0%BE%D0%BD%D0%B0_%D0%A1%D0%B5%D1%80%D0%B1%D1%81%D0%BA%D0%B0%D1%8F" title="Илона Сербская">этой сербской княжны</a></b> пребывали на <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BB%D0%B8_%D0%92%D0%B5%D0%BD%D0%B3%D1%80%D0%B8%D0%B8" class="mw-redirect" title="Короли Венгрии">венгерском троне</a> в разные годы.</li>
<li><b><a href="/wiki/%D0%A1%D0%B5%D0%BF%D0%B5%D1%88,_%D0%91%D0%B5%D0%BB%D0%B0" title="Сепеш, Бела">Шестикратный чемпион Венгрии</a></b> по <a href="/wiki/%D0%9B%D1%8B%D0%B6%D0%BD%D1%8B%D0%B9_%D1%81%D0%BF%D0%BE%D1%80%D1%82" title="Лыжный спорт">лыжам</a> завоевал олимпийское «серебро» в <a href="/wiki/%D0%9C%D0%B5%D1%82%D0%B0%D0%BD%D0%B8%D0%B5_%D0%BA%D0%BE%D0%BF%D1%8C%D1%8F" title="Метание копья">метании копья</a>.</li>
<li>Памятники <b><a href="/wiki/%D0%A8%D1%82%D0%B0%D0%B9%D0%BD%D0%BC%D0%B5%D1%86,_%D0%9C%D0%B8%D0%BA%D0%BB%D0%BE%D1%88" title="Штайнмец, Миклош">двум</a></b> расстрелянным <b><a href="/wiki/%D0%9E%D1%81%D1%82%D0%B0%D0%BF%D0%B5%D0%BD%D0%BA%D0%BE,_%D0%98%D0%BB%D1%8C%D1%8F_%D0%90%D1%84%D0%B0%D0%BD%D0%B0%D1%81%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Остапенко, Илья Афанасьевич">парламентёрам</a></b> встретились в парке «<a href="/wiki/%D0%9F%D0%B0%D1%80%D0%BA_%D0%9C%D0%B5%D0%BC%D0%B5%D0%BD%D1%82%D0%BE" class="mw-redirect" title="Парк Мементо">Мементо</a>».</li>

<p><b><a href="/wiki/%D0%A1%D0%B5%D0%BD%D0%B5%D1%88,_%D0%A5%D0%B0%D0%BD%D0%B0" title="Сенеш, Хана">Хана Сенеш</a></b> (<a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B3%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Венгерский язык">венг.</a> <span lang="hu" style="font-style:italic;">Szenes Hanna</span>, <a href="/wiki/%D0%98%D0%B2%D1%80%D0%B8%D1%82" title="Иврит">ивр.</a> &#8207;<span lang="he" dir="rtl" style="font-size: 110%; font-style: normal; line-height: 1em;">חנה סנש</span>&#8207;&#8206;, <a href="/wiki/17_%D0%B8%D1%8E%D0%BB%D1%8F" title="17 июля">17 июля</a> <a href="/wiki/1921" class="mw-redirect" title="1921">1921</a>, <a href="/wiki/%D0%91%D1%83%D0%B4%D0%B0%D0%BF%D0%B5%D1%88%D1%82" title="Будапешт">Будапешт</a>&#160;— <a href="/wiki/7_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="7 ноября">7 ноября</a> <a href="/wiki/1944" class="mw-redirect" title="1944">1944</a>, <a href="/wiki/%D0%91%D1%83%D0%B4%D0%B0%D0%BF%D0%B5%D1%88%D1%82" title="Будапешт">Будапешт</a>)&#160;— <a href="/wiki/%D0%92%D0%B5%D0%BD%D0%B3%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Венгерский язык">венгерская</a> и <a href="/wiki/%D0%98%D0%B2%D1%80%D0%B8%D1%82" title="Иврит">еврейская</a> поэтесса, <a href="/wiki/%D0%A1%D0%B8%D0%BE%D0%BD%D0%B8%D0%B7%D0%BC" title="Сионизм">сионистка</a>, <a href="/wiki/%D0%94%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5_%D0%A1%D0%BE%D0%BF%D1%80%D0%BE%D1%82%D0%B8%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" class="mw-redirect" title="Движение Сопротивления">партизанка Второй мировой войны</a>, национальная героиня <a href="/wiki/%D0%98%D0%B7%D1%80%D0%B0%D0%B8%D0%BB%D1%8C" title="Израиль">Израиля</a>.
Родилась и выросла в Венгрии. После окончания гимназии в 18 лет эмигрировала в <a href="/wiki/%D0%91%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%B0%D0%BD%D0%B4%D0%B0%D1%82_%D0%B2_%D0%9F%D0%B0%D0%BB%D0%B5%D1%81%D1%82%D0%B8%D0%BD%D0%B5" class="mw-redirect" title="Британский мандат в Палестине">Палестину</a>. Во время войны в составе группы <a href="/wiki/%D0%95%D0%B2%D1%80%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B5_%D0%BF%D0%B0%D1%80%D0%B0%D1%88%D1%8E%D1%82%D0%B8%D1%81%D1%82%D1%8B_%D0%B2_%D0%BF%D0%BE%D0%B4%D0%BC%D0%B0%D0%BD%D0%B4%D0%B0%D1%82%D0%BD%D0%BE%D0%B9_%D0%9F%D0%B0%D0%BB%D0%B5%D1%81%D1%82%D0%B8%D0%BD%D0%B5" title="Еврейские парашютисты в подмандатной Палестине">еврейских парашютистов</a> заброшена в немецкий тыл по заданию британского командования. Была арестована и расстреляна нацистами в Будапеште. В 1950 году её останки были торжественно перезахоронены в Израиле. Стихи Ханы Сенеш стали частью израильского фольклора, а её образ канонизирован в национальном еврейском самосознании.
