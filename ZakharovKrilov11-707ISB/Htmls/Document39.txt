<p><b>Ара́бский халифа́т</b> (<a href="/wiki/%D0%90%D1%80%D0%B0%D0%B1%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Арабский язык">араб.</a> <span lang="ar" style="font-size: 110%; font-style: normal; line-height: 1em;">خلافة إسلامية</span>&#8206;&#160;—  Исламский халифат&#8206;)&#160;— <a href="/wiki/%D0%A2%D0%B5%D0%BE%D0%BA%D1%80%D0%B0%D1%82%D0%B8%D1%8F" title="Теократия">теократическое</a> <a href="/wiki/%D0%98%D1%81%D0%BB%D0%B0%D0%BC%D1%81%D0%BA%D0%BE%D0%B5_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE_(%D0%BA%D0%BE%D0%BD%D1%86%D0%B5%D0%BF%D1%86%D0%B8%D1%8F)" title="Исламское государство (концепция)">исламское государство</a>, возникшее в результате мусульманских завоеваний в VII—IX&#160;вв. и возглавлявшееся <a href="/wiki/%D0%A5%D0%B0%D0%BB%D0%B8%D1%84" title="Халиф">халифами</a>. В старорусских источниках известно также под именами <b>царство Агарянское</b> и <b>царство Измаильтянское</b>, что таким образом включало его в общий список царств (империй) мира, известных книжным людям на Руси того времени.
</p><p>Первоначальным ядром халифата стала созданная арабским проповедником&#160;единобожия&#160;и&#160;пророком ислама <a href="/wiki/%D0%9C%D1%83%D1%85%D0%B0%D0%BC%D0%BC%D0%B5%D0%B4" title="Мухаммед">Мухаммедом</a> в начале <a href="/wiki/VII_%D0%B2%D0%B5%D0%BA" title="VII век">VII века</a> в <a href="/wiki/%D0%A5%D0%B8%D0%B4%D0%B6%D0%B0%D0%B7" title="Хиджаз">Хиджазе</a> (Западная Аравия) мусульманская община&#160;— <a href="/wiki/%D0%A3%D0%BC%D0%BC%D0%B0_(%D0%B8%D1%81%D0%BB%D0%B0%D0%BC)" class="mw-redirect" title="Умма (ислам)">умма</a>. Первоначально эта община была небольшой и представляла собой протогосударственное образование суперрелигиозного характера по аналогии с Моисеевым государством или Первыми христовыми общинами. В результате мусульманских завоеваний было создано огромное государство, включавшее <a href="/wiki/%D0%90%D1%80%D0%B0%D0%B2%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Аравийский полуостров">Аравийский полуостров</a>, <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BA" title="Ирак">Ирак</a>, <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BD" title="Иран">Иран</a>, большую часть <a href="/wiki/%D0%97%D0%B0%D0%BA%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7%D1%8C%D0%B5" title="Закавказье">Закавказья</a> (в частности <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BD%D0%B0%D0%B3%D0%BE%D1%80%D1%8C%D0%B5" title="Армянское нагорье">Армянское нагорье</a>, Прикаспийские территории, <a href="/wiki/%D0%9A%D0%BE%D0%BB%D1%85%D0%B8%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BD%D0%B8%D0%B7%D0%BC%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D1%8C" title="Колхидская низменность">Колхидскую низменность</a>, а также районы <a href="/wiki/%D0%A2%D0%B1%D0%B8%D0%BB%D0%B8%D1%81%D0%B8" title="Тбилиси">Тбилиси</a>), <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%BD%D1%8F%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Средняя Азия">Среднюю Азию</a>, <a href="/wiki/%D0%A1%D0%B8%D1%80%D0%B8%D1%8F" title="Сирия">Сирию</a>, <a href="/wiki/%D0%9F%D0%B0%D0%BB%D0%B5%D1%81%D1%82%D0%B8%D0%BD%D0%B0_(%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Палестина (историческая область)">Палестину</a>, <a href="/wiki/%D0%95%D0%B3%D0%B8%D0%BF%D0%B5%D1%82" title="Египет">Египет</a>, <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Северная Африка">Северную Африку</a>, большую часть <a href="/wiki/%D0%9F%D0%B8%D1%80%D0%B5%D0%BD%D0%B5%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Пиренейский полуостров">Пиренейского полуострова</a>, <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B4" title="Синд">Синд</a>.
</p><p>Слово <i>халиф</i> (<a href="/wiki/%D0%90%D1%80%D0%B0%D0%B1%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Арабский язык">араб.</a> <span lang="ar" style="font-size: 110%; font-style: normal; line-height: 1em;">خليفة</span>&#8206;&#160;— <i>Khalīfah</i>&#160;— «наследник», «представитель»)&#160;означает титул <a href="/wiki/%D0%A5%D0%B0%D0%BB%D0%B8%D1%84" title="Халиф">халифов</a>, создавших после смерти пророка Мухаммеда исламское государство <i>халифат</i>.
</p><p>Эпоха существования Арабского халифата, вместе с несколькими последующими веками расцвета общеисламской науки и культуры, именуются в западной историографии <a href="/wiki/%D0%97%D0%BE%D0%BB%D0%BE%D1%82%D0%BE%D0%B9_%D0%B2%D0%B5%D0%BA_%D0%B8%D1%81%D0%BB%D0%B0%D0%BC%D0%B0" title="Золотой век ислама">Золотым веком ислама</a>.
</p><p><a href="/wiki/%D0%90%D1%80%D0%B0%D0%B1%D1%81%D0%BA%D0%B8%D0%B9_%D1%85%D0%B0%D0%BB%D0%B8%D1%84%D0%B0%D1%82" title="Арабский халифат"><b>Далее …</b></a>

<p><b>Астрономия исламского Средневековья</b>&#160;— астрономические познания и взгляды, распространённые в <a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%BD%D0%B8%D0%B5_%D0%B2%D0%B5%D0%BA%D0%B0" title="Средние века">Средние века</a> в <a href="/wiki/%D0%90%D1%80%D0%B0%D0%B1%D1%81%D0%BA%D0%B8%D0%B9_%D1%85%D0%B0%D0%BB%D0%B8%D1%84%D0%B0%D1%82" title="Арабский халифат">Арабском халифате</a> и впоследствии государствах, возникших после распада халифата: <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%B4%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%85%D0%B0%D0%BB%D0%B8%D1%84%D0%B0%D1%82" title="Кордовский халифат">Кордовском халифате</a>, империях <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%B0%D0%BD%D0%B8%D0%B4%D1%8B" title="Саманиды">Саманидов</a>, <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0%D1%85%D0%B0%D0%BD%D0%B8%D0%B4%D1%81%D0%BA%D0%BE%D0%B5_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Караханидское государство">Караханидов</a>, <a href="/wiki/%D0%93%D0%B0%D0%B7%D0%BD%D0%B5%D0%B2%D0%B8%D0%B4%D1%81%D0%BA%D0%BE%D0%B5_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Газневидское государство">Газневидов</a>, <a href="/wiki/%D0%A2%D0%B8%D0%BC%D1%83%D1%80%D0%B8%D0%B4%D1%8B" title="Тимуриды">Тимуридов</a>, <a href="/wiki/%D0%A5%D1%83%D0%BB%D0%B0%D0%B3%D1%83%D0%B8%D0%B4%D1%8B" title="Хулагуиды">Хулагуидов</a>. Сочинения исламских астрономов были, как правило, написаны на арабском языке, который может считаться международным языком средневековой науки; по этой причине астрономия исламского Средневековья называется ещё арабской астрономией, хотя в её развитие внесли вклад не только арабы, но представители практически всех народов, проживавших на этой территории. Главным источником арабской астрономии была <a href="/wiki/%D0%90%D1%81%D1%82%D1%80%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%8F_%D0%94%D1%80%D0%B5%D0%B2%D0%BD%D0%B5%D0%B9_%D0%93%D1%80%D0%B5%D1%86%D0%B8%D0%B8" title="Астрономия Древней Греции">астрономия Древней Греции</a>, а на ранних стадиях развития — также <a href="/wiki/%D0%98%D0%BD%D0%B4%D0%B8%D1%8F" title="Индия">Индии</a> и <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE_%D0%A1%D0%B0%D1%81%D0%B0%D0%BD%D0%B8%D0%B4%D0%BE%D0%B2" title="Государство Сасанидов">государства Сасанидов</a>, расположенного на территориях современных <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BA" title="Ирак">Ирака</a> и <a href="/wiki/%D0%98%D1%80%D0%B0%D0%BD" title="Иран">Ирана</a>. Период наивысшего развития приходится на VIII—XV века.
</p><p><b>VII век.</b> Начало знакомства с астрономическими достижениями греков и индийцев. По повелению <a href="/wiki/%D0%9E%D0%BC%D0%B0%D1%80_%D0%B8%D0%B1%D0%BD_%D0%A5%D0%B0%D1%82%D1%82%D0%B0%D0%B1" class="mw-redirect" title="Омар ибн Хаттаб">халифа Омара</a> в <a href="/wiki/%D0%90%D1%80%D0%B0%D0%B1%D1%81%D0%BA%D0%B8%D0%B9_%D1%85%D0%B0%D0%BB%D0%B8%D1%84%D0%B0%D1%82" title="Арабский халифат">Арабском халифате</a> разработан религиозно-мотивированный <a href="/wiki/%D0%98%D1%81%D0%BB%D0%B0%D0%BC%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4%D0%B0%D1%80%D1%8C" title="Исламский календарь">лунный календарь</a>.
</p><p><b>VIII&#160;— первая половина IX века.</b> Интенсивный перевод индийской и греческой научной литературы на арабский. Основным покровителем учёных становится <a href="/wiki/%D0%A5%D0%B0%D0%BB%D0%B8%D1%84" title="Халиф">халиф</a> <a href="/wiki/%D0%9C%D0%B0%D0%BC%D1%83%D0%BD" class="mw-redirect" title="Мамун">ал-Мамун</a>, основавший в 832 году <a href="/wiki/%D0%94%D0%BE%D0%BC_%D0%BC%D1%83%D0%B4%D1%80%D0%BE%D1%81%D1%82%D0%B8" title="Дом мудрости">Дом мудрости</a> в Багдаде и две астрономические обсерватории, в <a href="/wiki/%D0%94%D0%B0%D0%BC%D0%B0%D1%81%D0%BA" title="Дамаск">Дамаске</a> и <a href="/wiki/%D0%91%D0%B0%D0%B3%D0%B4%D0%B0%D0%B4" title="Багдад">Багдаде</a>. На этот период приходится деятельность таких выдающихся астрономов и математиков, как <a href="/wiki/%D0%90%D0%BB-%D0%A4%D0%B0%D0%B7%D0%B0%D1%80%D0%B8" class="mw-redirect" title="Ал-Фазари">Ибрахим ал-Фазари</a> (ум. ок. 777), <a href="/wiki/%D0%AF%D0%BA%D1%83%D0%B1_%D0%B8%D0%B1%D0%BD_%D0%A2%D0%B0%D1%80%D0%B8%D0%BA" title="Якуб ибн Тарик">Якуб ибн Тарик</a> (ум. ок. 796),  <a href="/wiki/%D0%A5%D0%B0%D0%B1%D0%B1%D0%B0%D1%88_%D0%B0%D0%BB%D1%8C-%D0%A5%D0%B0%D1%81%D0%B8%D0%B1" class="mw-redirect" title="Хаббаш аль-Хасиб">Хаббаш аль-Хасиб</a> (770-870), <a href="/wiki/%D0%90%D0%BB%D1%8C-%D0%A5%D0%BE%D1%80%D0%B5%D0%B7%D0%BC%D0%B8" title="Аль-Хорезми">Мухаммад аль-Хорезми</a> (783–850), <a href="/wiki/%D0%90%D0%BB%D1%8C-%D0%A4%D0%B5%D1%80%D0%B3%D0%B0%D0%BD%D0%B8" title="Аль-Фергани">Аль-Фергани</a> (790—860), братья <a href="/wiki/%D0%91%D0%B0%D0%BD%D1%83_%D0%9C%D1%83%D1%81%D0%B0" title="Бану Муса">Бану Муса</a> (первая половина IX в.) и их ученик <a href="/wiki/%D0%A1%D0%B0%D0%B1%D0%B8%D1%82_%D0%B8%D0%B1%D0%BD_%D0%9A%D0%BE%D1%80%D1%80%D0%B0" class="mw-redirect" title="Сабит ибн Корра">Сабит ибн Корра</a> (836–901). Открытие изменения наклона эклиптики к экватору и мнимое открытие трепидации. Полное овладение математическим аппаратом греческой астрономии, включая теорию Птолемея.
</p>

<li>Благодаря <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D0%B2%D0%BE%D0%B4%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%B4%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5" title="Переводческое движение">средневековым переводам</a> на <b>«<a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D1%80%D0%B0%D0%B1%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Классический арабский язык">латынь Востока</a>»</b> было сохранено <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D1%84%D0%B8%D0%BB%D0%BE%D1%81%D0%BE%D1%84%D0%B8%D1%8F" title="Античная философия">античное философское наследие</a>.</li>
<li><b><a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Медников, Николай Александрович">Российский востоковед</a></b> доказал подложность <a href="/wiki/%D0%94%D0%BE%D0%B3%D0%BE%D0%B2%D0%BE%D1%80_%D0%A3%D0%BC%D0%B0%D1%80%D0%B0" title="Договор Умара">Договора Умара</a>.</li>
<li><b><a href="/wiki/%D0%91%D1%80%D0%B0%D1%82%D1%8C%D1%8F_%D1%87%D0%B8%D1%81%D1%82%D0%BE%D1%82%D1%8B" title="Братья чистоты">Первая арабская энциклопедия</a></b> была составлена в <a href="/wiki/X_%D0%B2%D0%B5%D0%BA" title="X век">X веке</a>.</li>
