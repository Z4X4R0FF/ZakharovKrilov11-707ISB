<p><b><a href="/wiki/%D0%90%D0%BB%D0%B1%D0%B0%D0%BD%D0%B8%D1%8F" title="Албания">Алба́ния</a></b> (<a href="/wiki/%D0%90%D0%BB%D0%B1%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Албанский язык">алб.</a>&#160;<span lang="sq" style="font-style:italic;">Shqipëria</span>, официальное название&#160;— <b>Респу́блика Алба́ния</b>, <a href="/wiki/%D0%90%D0%BB%D0%B1%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Албанский язык">алб.</a>&#160;<span lang="sq" style="font-style:italic;">Republika e Shqipërisë</span> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[ɾɛpuˈblika ɛ ʃcipəˈɾiːs]</a></span>)&#160;— <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a> в западной части <a href="/wiki/%D0%91%D0%B0%D0%BB%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BF%D0%BE%D0%BB%D1%83%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2" title="Балканский полуостров">Балканского полуострова</a>, на побережье <a href="/wiki/%D0%90%D0%B4%D1%80%D0%B8%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Адриатическое море">Адриатического</a> и <a href="/wiki/%D0%98%D0%BE%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Ионическое море">Ионического морей</a>. Член <a href="/wiki/%D0%9D%D0%90%D0%A2%D0%9E" title="НАТО">НАТО</a> с <a href="/wiki/2009_%D0%B3%D0%BE%D0%B4" title="2009 год">2009 года</a>.
</p><p>Пролив <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%BB%D0%B8%D0%B2_%D0%9E%D1%82%D1%80%D0%B0%D0%BD%D1%82%D0%BE" class="mw-redirect" title="Пролив Отранто">Отранто</a> отделяет Албанию от <a href="/wiki/%D0%98%D1%82%D0%B0%D0%BB%D0%B8%D1%8F" title="Италия">Италии</a>. На северо-востоке граничит с <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B1%D0%B8%D1%8F" title="Сербия">Сербией</a>, автономным краем <a href="/wiki/%D0%9A%D0%BE%D1%81%D0%BE%D0%B2%D0%BE_%D0%B8_%D0%9C%D0%B5%D1%82%D0%BE%D1%85%D0%B8%D1%8F" title="Косово и Метохия">Косово и Метохия</a> (<a href="/wiki/%D0%A7%D0%B0%D1%81%D1%82%D0%B8%D1%87%D0%BD%D0%BE_%D0%BF%D1%80%D0%B8%D0%B7%D0%BD%D0%B0%D0%BD%D0%BD%D1%8B%D0%B5_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0" class="mw-redirect" title="Частично признанные государства">частично признанным</a> <a href="/wiki/%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%9A%D0%BE%D1%81%D0%BE%D0%B2%D0%BE" title="Республика Косово">Косовом</a>), на северо-западе&#160;— с <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%BD%D0%BE%D0%B3%D0%BE%D1%80%D0%B8%D1%8F" title="Черногория">Черногорией</a>, на востоке&#160;— с <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%9C%D0%B0%D0%BA%D0%B5%D0%B4%D0%BE%D0%BD%D0%B8%D1%8F" title="Северная Македония">Северной Македонией</a>, на юго-востоке&#160;— с <a href="/wiki/%D0%93%D1%80%D0%B5%D1%86%D0%B8%D1%8F" title="Греция">Грецией</a>.
