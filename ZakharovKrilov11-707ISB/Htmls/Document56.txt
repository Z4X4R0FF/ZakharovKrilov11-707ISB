<p><b>Арха́нгельская область</b>&#160;— область на севере европейской части <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a>. В её состав входит <a href="/wiki/%D0%9D%D0%B5%D0%BD%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%BD%D1%8B%D0%B9_%D0%BE%D0%BA%D1%80%D1%83%D0%B3" title="Ненецкий автономный округ">Ненецкий автономный округ</a>.
Граничит на западе с <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B5%D0%BB%D0%B8%D1%8F" title="Карелия">Карелией</a>, на юге с <a href="/wiki/%D0%92%D0%BE%D0%BB%D0%BE%D0%B3%D0%BE%D0%B4%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Вологодская область">Вологодской</a> и <a href="/wiki/%D0%9A%D0%B8%D1%80%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Кировская область">Кировской</a> областями, на востоке с <a href="/wiki/%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%9A%D0%BE%D0%BC%D0%B8" title="Республика Коми">Республикой Коми</a>. Ненецкий АО граничит на юге и юго-востоке с Республикой Коми, на востоке&#160;— с <a href="/wiki/%D0%AF%D0%BC%D0%B0%D0%BB%D0%BE-%D0%9D%D0%B5%D0%BD%D0%B5%D1%86%D0%BA%D0%B8%D0%B9_%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%BD%D1%8B%D0%B9_%D0%BE%D0%BA%D1%80%D1%83%D0%B3" title="Ямало-Ненецкий автономный округ">Ямало-Ненецким автономным округом</a>.<br />
<i>Площадь</i> — 589 900 км²,  <i>Население</i> — 1 254 808 человек,  <br />
<i>Административный центр</i> — город <a href="/wiki/%D0%90%D1%80%D1%85%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D1%8C%D1%81%D0%BA" title="Архангельск">Архангельск</a>, <i>Образована</i> — <a href="/wiki/23_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="23 сентября">23 сентября</a> <a href="/wiki/1937_%D0%B3%D0%BE%D0%B4" title="1937 год">1937 года</a>

<p><b>Солове́цкий монасты́рь</b>&#160;— <a href="/wiki/%D0%A1%D1%82%D0%B0%D0%B2%D1%80%D0%BE%D0%BF%D0%B8%D0%B3%D0%B8%D1%8F" title="Ставропигия">ставропигиальный</a> мужской <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C" title="Монастырь">монастырь</a> <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%B0%D0%B2%D0%BE%D1%81%D0%BB%D0%B0%D0%B2%D0%BD%D0%B0%D1%8F_%D1%86%D0%B5%D1%80%D0%BA%D0%BE%D0%B2%D1%8C" title="Русская православная церковь">Русской православной церкви</a>, расположенный на <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BE%D0%B2%D0%B5%D1%86%D0%BA%D0%B8%D0%B5_%D0%BE%D1%81%D1%82%D1%80%D0%BE%D0%B2%D0%B0" title="Соловецкие острова">Соловецких островах</a> в <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Белое море">Белом море</a>.
</p><p>Возник в <a href="/wiki/1420-%D0%B5" class="mw-redirect" title="1420-е">1420</a>—<a href="/wiki/1430-%D0%B5" class="mw-redirect" title="1430-е">1430-е</a> годы, отстроен в камне трудами св. <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%B8%D0%BF%D0%BF_%D0%9A%D0%BE%D0%BB%D1%8B%D1%87%D0%B5%D0%B2" class="mw-redirect" title="Филипп Колычев">Филиппа (Колычева)</a>, в допетровское время числился среди крупнейших землевладельцев государства, в 1669—1676 гг. осаждён царскими войсками как <a href="/wiki/%D0%A1%D0%BE%D0%BB%D0%BE%D0%B2%D0%B5%D1%86%D0%BA%D0%BE%D0%B5_%D0%B2%D0%BE%D1%81%D1%81%D1%82%D0%B0%D0%BD%D0%B8%D0%B5" title="Соловецкое восстание">один из очагов сопротивления никонианским преобразованиям</a>.
</p><p>В <a href="/wiki/1992_%D0%B3%D0%BE%D0%B4" title="1992 год">1992 году</a> комплекс памятников Соловецкого музея-заповедника был внесён в список <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D0%B8%D0%B5" title="Всемирное наследие">Всемирного наследия</a> <a href="/wiki/%D0%AE%D0%9D%D0%95%D0%A1%D0%9A%D0%9E" title="ЮНЕСКО">ЮНЕСКО</a>, в <a href="/wiki/1995" class="mw-redirect" title="1995">1995</a>&#160;— в <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%81%D0%B2%D0%BE%D0%B4_%D0%BE%D1%81%D0%BE%D0%B1%D0%BE_%D1%86%D0%B5%D0%BD%D0%BD%D1%8B%D1%85_%D0%BE%D0%B1%D1%8A%D0%B5%D0%BA%D1%82%D0%BE%D0%B2_%D0%BA%D1%83%D0%BB%D1%8C%D1%82%D1%83%D1%80%D0%BD%D0%BE%D0%B3%D0%BE_%D0%BD%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D0%B8%D1%8F_%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BE%D0%B2_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B9_%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D0%B8" title="Государственный свод особо ценных объектов культурного наследия народов Российской Федерации">Государственный свод особо ценных объектов культурного наследия народов Российской Федерации</a>.

<p><b>Михаи́л Васи́льевич Ломоно́сов</b> (<abbr title="по юлианскому календарю">8</abbr>&#160;<a href="/wiki/19_%D0%BD%D0%BE%D1%8F%D0%B1%D1%80%D1%8F" title="19 ноября">(19)&#160;ноября</a>&#160;<a href="/wiki/1711_%D0%B3%D0%BE%D0%B4" title="1711 год">1711</a>, деревня <a href="/wiki/%D0%9B%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D1%81%D0%BE%D0%B2%D0%BE_(%D0%90%D1%80%D1%85%D0%B0%D0%BD%D0%B3%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Ломоносово (Архангельская область)">Мишанинская</a>, <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Россия</a>&#160;— <abbr title="по юлианскому календарю">4</abbr>&#160;<a href="/wiki/15_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="15 апреля">(15)&#160;апреля</a>&#160;<a href="/wiki/1765_%D0%B3%D0%BE%D0%B4" title="1765 год">1765</a>, <a href="/wiki/%D0%A1%D0%B0%D0%BD%D0%BA%D1%82-%D0%9F%D0%B5%D1%82%D0%B5%D1%80%D0%B1%D1%83%D1%80%D0%B3" title="Санкт-Петербург">Санкт-Петербург</a>, <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Российская империя">Российская империя</a>)&#160;— первый <a href="/wiki/%D0%A0%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9" class="mw-disambig" title="Русский">русский</a> <a href="/wiki/%D0%A3%D1%87%D1%91%D0%BD%D1%8B%D0%B9" title="Учёный">учёный</a>-<a href="/wiki/%D0%95%D1%81%D1%82%D0%B5%D1%81%D1%82%D0%B2%D0%BE%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B5" title="Естествознание">естествоиспытатель</a> мирового значения, энциклопедист, <a href="/wiki/%D0%A5%D0%B8%D0%BC%D0%B8%D1%8F" title="Химия">химик</a> и <a href="/wiki/%D0%A4%D0%B8%D0%B7%D0%B8%D0%BA%D0%B0" title="Физика">физик</a>; он вошёл в науку как первый химик, который дал <a href="/wiki/%D0%A4%D0%B8%D0%B7%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%85%D0%B8%D0%BC%D0%B8%D1%8F" title="Физическая химия">физической химии</a> определение, весьма близкое к современному, и предначертал обширную программу физико-химических исследований; его <a href="/wiki/%D0%9C%D0%BE%D0%BB%D0%B5%D0%BA%D1%83%D0%BB%D1%8F%D1%80%D0%BD%D0%BE-%D0%BA%D0%B8%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F_%D1%82%D0%B5%D0%BE%D1%80%D0%B8%D1%8F" title="Молекулярно-кинетическая теория">молекулярно-кинетическая теория</a> тепла во многом предвосхитила современное представление о строении материи,&#160;— многие фундаментальные законы, в числе которых одно из начал <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%BE%D0%B5_%D0%BD%D0%B0%D1%87%D0%B0%D0%BB%D0%BE_%D1%82%D0%B5%D1%80%D0%BC%D0%BE%D0%B4%D0%B8%D0%BD%D0%B0%D0%BC%D0%B8%D0%BA%D0%B8" title="Второе начало термодинамики">термодинамики</a>; заложил основы науки о <a href="/wiki/%D0%A1%D1%82%D0%B5%D0%BA%D0%BB%D0%BE" title="Стекло">стекле</a>.

Разработал проект <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%83%D0%BD%D0%B8%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82_%D0%B8%D0%BC%D0%B5%D0%BD%D0%B8_%D0%9C._%D0%92._%D0%9B%D0%BE%D0%BC%D0%BE%D0%BD%D0%BE%D1%81%D0%BE%D0%B2%D0%B0" class="mw-redirect" title="Московский государственный университет имени М.
