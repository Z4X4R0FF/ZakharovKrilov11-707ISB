<p><b><a href="/wiki/Queen" title="Queen">Queen</a></b> («Куин») — <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">британская</a> <a href="/wiki/%D0%A0%D0%BE%D0%BA-%D0%B3%D1%80%D1%83%D0%BF%D0%BF%D0%B0" title="Рок-группа">рок-группа</a>, добившаяся широчайшей известности в 1970-х — 1980-х годах и насчитывающая сотни миллионов поклонников по сей день. К самым знаменитым песням Queen относятся такие классические рок-хиты как «<a href="/wiki/Bohemian_Rhapsody" title="Bohemian Rhapsody">Bohemian Rhapsody</a>», «<a href="/wiki/We_Will_Rock_You" title="We Will Rock You">We Will Rock You</a>», «<a href="/wiki/We_Are_the_Champions" title="We Are the Champions">We Are the Champions</a>», «<a href="/wiki/A_Kind_of_Magic" title="A Kind of Magic">A Kind of Magic</a>», «<a href="/wiki/The_Show_Must_Go_On" title="The Show Must Go On">The Show Must Go On</a>» и другие. Широкую известность получили и <a href="/wiki/%D0%92%D0%B8%D0%B4%D0%B5%D0%BE%D0%BA%D0%BB%D0%B8%D0%BF" title="Видеоклип">видеоклипы</a>, снятые музыкантами Queen, причём один из них (на песню «Bohemian Rhapsody») является первым клипом в истории музыки. Кроме того, Queen снискали славу одной из величайших концертных команд за всю историю рока.
</p>

<p><b>Брюс Дикинсон</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">Paul Bruce Dickinson</span>; род.&#160;<a href="/wiki/7_%D0%B0%D0%B2%D0%B3%D1%83%D1%81%D1%82%D0%B0" title="7 августа">7 августа</a> <a href="/wiki/1958_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B5" title="1958 год в музыке">1958 года</a>, <a href="/wiki/%D0%A3%D0%BE%D1%80%D0%BA%D1%81%D0%BE%D0%BF" class="mw-redirect" title="Уорксоп">Уорксоп</a>, <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">Великобритания</a>) — британский рок-музыкант, <a href="/wiki/%D0%9F%D0%B8%D1%81%D0%B0%D1%82%D0%B5%D0%BB%D1%8C" title="Писатель">писатель</a>, <a href="/wiki/%D0%A1%D0%BF%D0%BE%D1%80%D1%82%D1%81%D0%BC%D0%B5%D0%BD" class="mw-redirect" title="Спортсмен">спортсмен</a> (<a href="/wiki/%D0%A4%D0%B5%D1%85%D1%82%D0%BE%D0%B2%D0%B0%D0%BB%D1%8C%D1%89%D0%B8%D0%BA" class="mw-redirect" title="Фехтовальщик">фехтовальщик</a>), <a href="/wiki/%D0%9B%D1%91%D1%82%D1%87%D0%B8%D0%BA" title="Лётчик">пилот</a> <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B6%D0%B4%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D0%B2%D0%B8%D0%B0%D1%86%D0%B8%D1%8F" title="Гражданская авиация">гражданской авиации</a>, теле- и радиоведущий, автор книг и сценарист, <a href="/wiki/%D0%9F%D1%80%D0%BE%D0%B4%D1%8E%D1%81%D0%B5%D1%80" class="mw-redirect" title="Продюсер">продюсер</a>, прежде всего известный как <a href="/wiki/%D0%A4%D1%80%D0%BE%D0%BD%D1%82%D0%BC%D0%B5%D0%BD" class="mw-redirect" title="Фронтмен">фронтмен</a> <a href="/wiki/%D0%A5%D1%8D%D0%B2%D0%B8-%D0%BC%D0%B5%D1%82%D0%B0%D0%BB" class="mw-redirect" title="Хэви-метал">хэви-метал</a> группы <a href="/wiki/Iron_Maiden" title="Iron Maiden">Iron Maiden</a> (по версии журнала <a href="/wiki/Classic_Rock" title="Classic Rock">Classic Rock</a>, он занимает 8 место в рейтинге лучших фронтменов <a href="/wiki/%D0%A0%D0%BE%D0%BA-%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B0" title="Рок-музыка">рок-музыки</a>).
</p><p>Покинув Iron Maiden в <a href="/wiki/1993_%D0%B3%D0%BE%D0%B4" title="1993 год">1993 году</a> на волне спада популярности хеви-метала, Дикинсон прошёл долгий путь творческого поиска, в процессе которого безуспешно экспериментировал с новыми музыкальными стилями (в частности с <a href="/wiki/%D0%93%D1%80%D0%B0%D0%BD%D0%B6" title="Гранж">гранжем</a>). После создания группы Skunkworks и её коммерческого провала возобновил сотрудничество с гитаристом <a href="/wiki/Roy_Z" title="Roy Z">Roy Z</a>. Результатом их сотрудничества стали успешные хеви-метал альбомы <i><a href="/wiki/Accident_of_Birth" title="Accident of Birth">Accident of Birth</a></i> и <i><a href="/wiki/The_Chemical_Wedding" title="The Chemical Wedding">The Chemical Wedding</a></i>. В итоге успех Дикинсона как сольного вокалиста стал расти, тогда как покинутая им группа постепенно теряла популярность. Его возвращение в Iron Maiden в <a href="/wiki/1999_%D0%B3%D0%BE%D0%B4" title="1999 год">1999 году</a> вернуло коллективу былую известность, но в то же время не стало препятствием для его сольной карьеры и работы в качестве пилота гражданской авиации.
</p>

<p>В русском нет единой устоявшейся терминологии для документов, принимаемых Парламентом Великобритании (билль, акт, закон); для соответствия английскому варианту (облегчения поиска переводов) предлагается использовать слово «акт».

<ul><li>Обязанности <b><a href="/wiki/%D0%9B%D0%BE%D1%80%D0%B4-%D0%BF%D1%80%D0%B5%D0%B4%D1%81%D0%B5%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D0%B0" title="Лорд-председатель Совета">четвёртого по значимости высшего сановника Великобритании</a></b> дважды исполняли премьер-министры <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0%D1%8F_%D0%97%D0%B5%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Новая Зеландия">Новой Зеландии</a>.</li>
<li><b><a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%83%D1%8D%D0%B9,_%D0%A8%D0%BE%D0%BD" title="Конуэй, Шон">Шону Конуэю</a></b> потребовалось 135 дней, чтобы обогнуть вплавь <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">Великобританию</a>.</li>

<li>Дольше всех на троне Великобритании пробыла <b><a href="/wiki/%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80%D0%B8%D1%8F_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D0%B8)" title="Виктория (королева Великобритании)">королева Виктория</a></b> — 63 года, с <a href="/wiki/1837" class="mw-redirect" title="1837">1837</a> по <a href="/wiki/1901" class="mw-redirect" title="1901">1901</a> годы.</li>
<li><a href="/wiki/%D0%92%D0%B5%D1%81" title="Вес">Вес</a> британской <b><a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BD%D0%B0_%D1%81%D0%B2%D1%8F%D1%82%D0%BE%D0%B3%D0%BE_%D0%AD%D0%B4%D1%83%D0%B0%D1%80%D0%B4%D0%B0" title="Корона святого Эдуарда">короны св. Эдуарда</a></b> велик&#160;— 4 <a href="/wiki/%D0%A4%D1%83%D0%BD%D1%82_(%D0%B5%D0%B4%D0%B8%D0%BD%D0%B8%D1%86%D0%B0_%D0%B8%D0%B7%D0%BC%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D1%8F)" title="Фунт (единица измерения)">фунта</a> 12 <a href="/wiki/%D0%A3%D0%BD%D1%86%D0%B8%D1%8F" title="Унция">унций</a> (2,155 кг). Из-за этого королева <a href="/wiki/%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80%D0%B8%D1%8F_(%D0%BA%D0%BE%D1%80%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0_%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D0%B8)" title="Виктория (королева Великобритании)">Виктория</a> и <a href="/wiki/%D0%AD%D0%B4%D1%83%D0%B0%D1%80%D0%B4_VII" title="Эдуард VII">Эдуард VII</a> отказались быть коронованными короной св. Эдуарда и короновались более лёгкой <a href="/wiki/%D0%9A%D0%BE%D1%80%D0%BE%D0%BD%D0%B0_%D0%91%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D1%81%D0%BA%D0%BE%D0%B9_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D0%B8" title="Корона Британской империи">короной Британской империи</a>.</li></ul>
