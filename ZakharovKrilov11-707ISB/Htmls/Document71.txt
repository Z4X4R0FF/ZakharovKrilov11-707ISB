<p><b>Ботсва́на</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">Botswana</span>, <a href="/wiki/%D0%A2%D1%81%D0%B2%D0%B0%D0%BD%D0%B0_(%D1%8F%D0%B7%D1%8B%D0%BA)" title="Тсвана (язык)">тсвана</a> <i>Botswana</i>), <b>Респу́блика Ботсва́на</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">Republic of Botswana</span>, <a href="/wiki/%D0%A2%D1%81%D0%B2%D0%B0%D0%BD%D0%B0_(%D1%8F%D0%B7%D1%8B%D0%BA)" title="Тсвана (язык)">тсвана</a> <i>Lefatshe la Botswana</i>)&#160;— <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a> в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Африка">Южной Африке</a>. Граничит на юге с <a href="/wiki/%D0%AE%D0%90%D0%A0" class="mw-redirect" title="ЮАР">ЮАР</a>, на западе и севере с <a href="/wiki/%D0%9D%D0%B0%D0%BC%D0%B8%D0%B1%D0%B8%D1%8F" title="Намибия">Намибией</a>, на северо-востоке с <a href="/wiki/%D0%97%D0%B0%D0%BC%D0%B1%D0%B8%D1%8F" title="Замбия">Замбией</a>, на востоке с <a href="/wiki/%D0%97%D0%B8%D0%BC%D0%B1%D0%B0%D0%B1%D0%B2%D0%B5" title="Зимбабве">Зимбабве</a>, не имеет выхода к морю. Географически 70&#160;% территории страны занимает пустыня <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B0%D1%85%D0%B0%D1%80%D0%B8" title="Калахари">Калахари</a>.
</p><p><a href="/wiki/30_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="30 сентября">30 сентября</a> <a href="/wiki/1966_%D0%B3%D0%BE%D0%B4" title="1966 год">1966 года</a>&#160;британский <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B5%D0%BA%D1%82%D0%BE%D1%80%D0%B0%D1%82" title="Протекторат">протекторат</a> <a href="/wiki/%D0%91%D0%B5%D1%87%D1%83%D0%B0%D0%BD%D0%B0%D0%BB%D0%B5%D0%BD%D0%B4" title="Бечуаналенд">Бечуаналенд</a> провозгласил независимость в рамках <a href="/wiki/%D0%A1%D0%BE%D0%B4%D1%80%D1%83%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%BD%D0%B0%D1%86%D0%B8%D0%B9" title="Содружество наций">Содружества наций</a> и сменил название на современное. <p><b>Наскальная живопись в районе Цодило</b>, расположенная на северо-западе <a href="/wiki/%D0%91%D0%BE%D1%82%D1%81%D0%B2%D0%B0%D0%BD%D0%B0" title="Ботсвана">Ботсваны</a>, содержит одно из самых значительных в мире собраний доисторического наскального искусства. Является объектом <a href="/wiki/%D0%92%D1%81%D0%B5%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B5_%D0%BD%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D0%B8%D0%B5" title="Всемирное наследие">Всемирного наследия</a> <a href="/wiki/%D0%AE%D0%9D%D0%95%D0%A1%D0%9A%D0%9E" title="ЮНЕСКО">ЮНЕСКО</a>, была включена в список в <a href="/wiki/2001_%D0%B3%D0%BE%D0%B4" title="2001 год">2001 году</a> в связи с исключительной религиозной и духовной важности объекта для местного населения и как доказательство человеческого поселения давностью несколько тысячелетий. Содержит более 4500 наскальных рисунков на территории пустыни <a href="/wiki/%D0%9A%D0%B0%D0%BB%D0%B0%D1%85%D0%B0%D1%80%D0%B8" title="Калахари">Калахари</a> площадью примерно 10 км². Археологические находки района отражают хронологию жизнедеятельности человека на протяжении периода не менее 100 тыс. лет.
