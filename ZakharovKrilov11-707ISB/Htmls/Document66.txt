<p><b>Буркина́-Фасо́</b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">Burkina Faso</span> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[buʁkiˈna faˈso]</a></span>)  — <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a> в <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Западная Африка">Западной Африке</a>. До августа <a href="/wiki/1984_%D0%B3%D0%BE%D0%B4" title="1984 год">1984 года</a> — <b><a href="/wiki/%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%92%D0%B5%D1%80%D1%85%D0%BD%D1%8F%D1%8F_%D0%92%D0%BE%D0%BB%D1%8C%D1%82%D0%B0" title="Республика Верхняя Вольта">Ве́рхняя Во́льта</a></b>.
</p><p>Граничит с <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B8" title="Мали">Мали</a> на севере, с <a href="/wiki/%D0%9D%D0%B8%D0%B3%D0%B5%D1%80" title="Нигер">Нигером</a> — на востоке, с <a href="/wiki/%D0%91%D0%B5%D0%BD%D0%B8%D0%BD" title="Бенин">Бенином</a> — на юго-востоке, с <a href="/wiki/%D0%A2%D0%BE%D0%B3%D0%BE" title="Того">Того</a> и <a href="/wiki/%D0%93%D0%B0%D0%BD%D0%B0" title="Гана">Ганой</a> — на юге и с <a href="/wiki/%D0%9A%D0%BE%D1%82-%D0%B4%E2%80%99%D0%98%D0%B2%D1%83%D0%B0%D1%80" title="Кот-д’Ивуар">Кот-д’Ивуаром</a> — на юго-западе. <a href="/wiki/%D0%9D%D0%B5_%D0%B8%D0%BC%D0%B5%D1%8E%D1%89%D0%B8%D0%B5_%D0%B2%D1%8B%D1%85%D0%BE%D0%B4%D0%B0_%D0%BA_%D0%BC%D0%BE%D1%80%D1%8E_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B0" class="mw-redirect" title="Не имеющие выхода к морю государства">Выхода к морю не имеет</a>.
</p><p>Название означает «родина честных людей» (в переводе с языка <a href="/wiki/%D0%9C%D0%BE%D0%BE%D1%80%D0%B5" title="Мооре">мооре</a> «буркина́» — «честные люди», в переводе с языка <a href="/wiki/%D0%94%D1%8C%D1%8E%D0%BB%D0%B0_(%D1%8F%D0%B7%D1%8B%D0%BA)" title="Дьюла (язык)">дьюла</a> «фасо́» — «отечество»). Другая версия перевода — «страна достойных людей». Территория — <span class="nowrap">273 187&#160;<a href="/wiki/%D0%9A%D0%B2%D0%B0%D0%B4%D1%80%D0%B0%D1%82%D0%BD%D1%8B%D0%B9_%D0%BA%D0%B8%D0%BB%D0%BE%D0%BC%D0%B5%D1%82%D1%80" title="Квадратный километр">км²</a></span>, население — 15,7 млн человек. Столица — <a href="/wiki/%D0%A3%D0%B0%D0%B3%D0%B0%D0%B4%D1%83%D0%B3%D1%83" title="Уагадугу">Уагадугу</a>.

<p><b>Уагаду́гу</b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">Ouagadougou</span> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[wa.ga.du.gu]</a></span>, <a href="/wiki/%D0%9C%D0%BE%D0%BE%D1%80%D0%B5" title="Мооре">мооре</a> <span lang="mos" style="font-style:italic;">Ouagadougou</span> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[ˈwaɡᵊdᵊɡᵊ]</a></span>) — <a href="/wiki/%D0%A1%D1%82%D0%BE%D0%BB%D0%B8%D1%86%D0%B0" title="Столица">столица</a> <a href="/wiki/%D0%91%D1%83%D1%80%D0%BA%D0%B8%D0%BD%D0%B0-%D0%A4%D0%B0%D1%81%D0%BE" title="Буркина-Фасо">Буркина-Фасо</a>, в междуречье <a href="/wiki/%D0%9A%D1%80%D0%B0%D1%81%D0%BD%D0%B0%D1%8F_%D0%92%D0%BE%D0%BB%D1%8C%D1%82%D0%B0" title="Красная Вольта">Красной</a> и <a href="/wiki/%D0%91%D0%B5%D0%BB%D0%B0%D1%8F_%D0%92%D0%BE%D0%BB%D1%8C%D1%82%D0%B0" title="Белая Вольта">Белой</a> Вольты, является административным, экономическим, транспортным и культурным центром страны.
</p><p>Население — свыше 1 181 702 жителей (<a href="/wiki/2006" class="mw-redirect" title="2006">2006</a>), то есть <b>Уагадугу</b> — самый большой город страны. Часто используется сокращенное название города — <i>Уага</i>. <a href="/wiki/%D0%AD%D1%82%D0%BD%D0%BE%D1%85%D0%BE%D1%80%D0%BE%D0%BD%D0%B8%D0%BC" class="mw-redirect" title="Этнохороним">Этнохоронимы</a>: уагадугцы, уагадугец. Прилагательное — уагадугский.
</p><p>В городе расположены предприятия пищевой и текстильной промышленности. Имеется международный аэропорт, железнодорожный вокзал. Культурная жизнь представлена <a href="/wiki/%D0%9A%D0%B8%D0%BD%D0%BE%D1%82%D0%B5%D0%B0%D1%82%D1%80" title="Кинотеатр">кинотеатрами</a>, ночными клубами, французским и американским культурными центрами. Ранее в Уагадугу располагался один из крупнейших <a href="/wiki/%D0%91%D0%B0%D0%B7%D0%B0%D1%80" title="Базар">базаров</a> Западной Африки, закрытый после пожара, произошедшего в <a href="/wiki/2003_%D0%B3%D0%BE%D0%B4" title="2003 год">2003 году</a>. Достопримечательностями города являются: Национальный музей Буркина Фасо, дворец Моро-Наба, Национальный музей музыки и несколько заведений, торгующих произведениями традиционных промыслов.
</p>

<p><b>Фульбе</b>, или <b>фула</b>, <b>фулани</b> (<a href="/wiki/%D0%A4%D1%83%D0%BB%D0%B0_(%D1%8F%D0%B7%D1%8B%D0%BA)" title="Фула (язык)">фула</a>: <i>fulɓe</i>, ед. ч. <i>fullo</i>) — народ, проживающий на обширной территории в <a href="/wiki/%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Западная Африка">Западной Африке</a>: от <a href="/wiki/%D0%9C%D0%B0%D0%B2%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Мавритания">Мавритании</a>, <a href="/wiki/%D0%93%D0%B0%D0%BC%D0%B1%D0%B8%D1%8F" title="Гамбия">Гамбии</a>, <a href="/wiki/%D0%A1%D0%B5%D0%BD%D0%B5%D0%B3%D0%B0%D0%BB" title="Сенегал">Сенегала</a> и <a href="/wiki/%D0%93%D0%B2%D0%B8%D0%BD%D0%B5%D1%8F" title="Гвинея">Гвинеи</a> на западе до <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%B5%D1%80%D1%83%D0%BD" title="Камерун">Камеруна</a> и даже <a href="/wiki/%D0%A1%D1%83%D0%B4%D0%B0%D0%BD" title="Судан">Судана</a> на востоке. Говорят на языке <a href="/wiki/%D0%A4%D1%83%D0%BB%D0%B0_(%D1%8F%D0%B7%D1%8B%D0%BA)" title="Фула (язык)">фула</a> атлантической семьи нигеро-кордофанской макросемьи языков.
</p><p><i>Фульбе</i> — самоназвание этого народа. Иногда может употребляться название <i>фула</i> (заимствованное из <a href="/wiki/%D0%AF%D0%B7%D1%8B%D0%BA%D0%B8_%D0%BC%D0%B0%D0%BD%D0%B4%D0%B5%D0%BD" title="Языки манден">языков манден</a>) или <i>фулани</i> (из <a href="/wiki/%D0%A5%D0%B0%D1%83%D1%81%D0%B0_(%D1%8F%D0%B7%D1%8B%D0%BA)" title="Хауса (язык)">хауса</a>). В долине <a href="/wiki/%D0%A1%D0%B5%D0%BD%D0%B5%D0%B3%D0%B0%D0%BB_(%D1%80%D0%B5%D0%BA%D0%B0)" title="Сенегал (река)">реки Сенегал</a> проживает оседлая народность <a href="/wiki/%D0%A2%D1%83%D0%BA%D1%83%D0%BB%D1%91%D1%80" title="Тукулёр">тукулёр</a>, также говорящая на языке фула. Фульбе и тукулёр вместе могут называться <i>haalpulaar’en</i> («говорящие на фула»).

<ul><li>Большинство слонов Западной Африки обитает именно в <a href="/wiki/%D0%91%D1%83%D1%80%D0%BA%D0%B8%D0%BD%D0%B0-%D0%A4%D0%B0%D1%81%D0%BE" title="Буркина-Фасо">Буркина-Фасо</a>, в национальном парке <i>Deux Balés</i></li>

<li><a href="/wiki/%D0%91%D0%BE%D0%B1%D0%BE-%D0%94%D0%B8%D1%83%D0%BB%D0%B0%D1%81%D0%BE" title="Бобо-Диуласо">Бобо-Диуласо</a> является вторым по величине городом страны и известен своей Великой мечетью, построенной в <a href="/wiki/1880_%D0%B3%D0%BE%D0%B4" title="1880 год">1880 году</a> из грязи.</li></ul>

Придерживался <a href="/wiki/%D0%A0%D0%B5%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9_%D1%81%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Революционный социализм">революционных</a> <a href="/wiki/%D0%9C%D0%B0%D1%80%D0%BA%D1%81%D0%B8%D0%B7%D0%BC" title="Марксизм">марксистских</a> взглядов и был одним из теоретиков <a href="/wiki/%D0%9F%D0%B0%D0%BD%D0%B0%D1%84%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D0%B8%D0%B7%D0%BC" title="Панафриканизм">панафриканизма</a>. За свои <a href="/wiki/%D0%9B%D0%B5%D0%B2%D1%8B%D0%B5_(%D0%BF%D0%BE%D0%BB%D0%B8%D1%82%D0%B8%D0%BA%D0%B0)" class="mw-redirect" title="Левые (политика)">левые</a> убеждения, революционную деятельность и амбициозность прозван «<a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Африка">африканским</a> <a href="/wiki/%D0%A7%D0%B5_%D0%93%D0%B5%D0%B2%D0%B0%D1%80%D0%B0,_%D0%AD%D1%80%D0%BD%D0%B5%D1%81%D1%82%D0%BE" title="Че Гевара, Эрнесто">Че Геварой</a>».
<p>33-летний Санкара захватил власть в 1983 году в результате <a href="/wiki/%D0%92%D0%BE%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D0%B2%D0%BE%D1%80%D0%BE%D1%82" class="mw-redirect" title="Военный переворот">военного переворота</a>, получившего народную поддержку, и имел цель побороть широко распространенную <a href="/wiki/%D0%9A%D0%BE%D1%80%D1%80%D1%83%D0%BF%D1%86%D0%B8%D1%8F" title="Коррупция">коррупцию</a> и влияние бывшей колониальной державы — <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a>. После прихода к власти приступил к реализации широкомасштабной амбициозной программы социальных и экономических реформ, не имевшей аналогов на континенте. Чтобы подчеркнуть независимость и возрождение страны, изменил название страны с <a href="/wiki/%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%92%D0%B5%D1%80%D1%85%D0%BD%D1%8F%D1%8F_%D0%92%D0%BE%D0%BB%D1%8C%D1%82%D0%B0" title="Республика Верхняя Вольта">Верхней Вольты</a> (которое дали <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BE%D0%BD%D0%B8%D0%B8_%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D0%B8" title="Колонии Франции">французские колонизаторы</a>) на Буркина-Фасо («Страна достойных людей»).
