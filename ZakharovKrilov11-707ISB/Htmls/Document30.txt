<p><b>Габо́н</b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">Gabon</span> <span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">[ɡaˈbõ]</a></span>), официально — <b>Габо́нская Респу́блика</b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">République gabonaise</span>) — <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a> в <a href="/wiki/%D0%A6%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Центральная Африка">Центральной Африке</a>, бывшая колония <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a>. Считается одним из самых богатых и стабильных государств на <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Африка">Африканском</a> континенте. На северо-западе граничит с <a href="/wiki/%D0%AD%D0%BA%D0%B2%D0%B0%D1%82%D0%BE%D1%80%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%93%D0%B2%D0%B8%D0%BD%D0%B5%D1%8F" title="Экваториальная Гвинея">Экваториальной Гвинеей</a>, с <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%B5%D1%80%D1%83%D0%BD" title="Камерун">Камеруном</a> на севере, на востоке и юге — с <a href="/wiki/%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0_%D0%9A%D0%BE%D0%BD%D0%B3%D0%BE" title="Республика Конго">Республикой Конго</a>. На западе страну омывают воды <a href="/wiki/%D0%90%D1%82%D0%BB%D0%B0%D0%BD%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%BE%D0%BA%D0%B5%D0%B0%D0%BD" title="Атлантический океан">Атлантического океана</a>. Столица — город <a href="/wiki/%D0%9B%D0%B8%D0%B1%D1%80%D0%B5%D0%B2%D0%B8%D0%BB%D1%8C" title="Либревиль">Либревиль</a>.
</p><p>Площадь — 267 667 км². Население — 1 497 525 человек (на 2008 год). 
</p><p>Валюта — <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D0%BA_%D0%9A%D0%A4%D0%90" title="Франк КФА">Франк КФА</a>.

<p><b>Государственный переворот в Габоне</b> был совершен 17 - 18 февраля <a href="/wiki/1964_%D0%B3%D0%BE%D0%B4" title="1964 год">1964 года</a> офицерами вооруженных сил Габона и был направлен против президента <a href="/wiki/%D0%9B%D0%B5%D0%BE%D0%BD_%D0%9C%D0%B1%D0%B0" class="mw-redirect" title="Леон Мба">Леона Мба</a>. До переворота Габон считался одной из самых политически стабильных стран <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Африка">Африки</a>.
</p><p>Переворот произошел после того, как президент Мба распутил законодательный орган страны 21 января 1964 года. В ходе переворота, обошедшегося почти без жертв, группа из 150 заговорщиков арестовала Мба и нескольких чиновников. По радио <a href="/wiki/%D0%9B%D0%B8%D0%B1%D1%80%D0%B5%D0%B2%D0%B8%D0%BB%D1%8C" title="Либревиль">Либревиля</a> заговорщики призвали народ Габона сохранять спокойствие и заверили, что про-французская политика страны останется неизменной.

<p><b>Эль-Хадж Омар Бонго Ондимба</b> (<a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">El Hadj Omar Bongo Ondimba</span>; имя при рождении — <i>Альбер Бернар Бонго</i>) (<a href="/wiki/30_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8F" title="30 декабря">30 декабря</a> <a href="/wiki/1935" class="mw-redirect" title="1935">1935</a> — <a href="/wiki/8_%D0%B8%D1%8E%D0%BD%D1%8F" title="8 июня">8 июня</a> <a href="/wiki/2009" class="mw-redirect" title="2009">2009</a>) — <a href="/wiki/%D0%93%D0%B0%D0%B1%D0%BE%D0%BD" title="Габон">габонский</a> государственный деятель, <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B7%D0%B8%D0%B4%D0%B5%D0%BD%D1%82_%D0%93%D0%B0%D0%B1%D0%BE%D0%BD%D0%B0" title="Президент Габона">президент Республики Габон</a> в <a href="/wiki/1967" class="mw-redirect" title="1967">1967</a>—<a href="/wiki/2009" class="mw-redirect" title="2009">2009</a> годах, стал <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B7%D0%B8%D0%B4%D0%B5%D0%BD%D1%82" title="Президент">президентом</a> в возрасте 32 лет, являлся самым молодым президентом в мире на тот момент. После смерти <a href="/wiki/%D0%A2%D0%BE%D0%B3%D0%BE" title="Того">тоголезского</a> президента <a href="/wiki/%D0%93%D0%BD%D0%B0%D1%81%D1%81%D0%B8%D0%BD%D0%B3%D0%B1%D0%B5_%D0%AD%D0%B9%D0%B0%D0%B4%D0%B5%D0%BC%D0%B0" class="mw-redirect" title="Гнассингбе Эйадема">Гнассингбе Эйадема</a> в феврале <a href="/wiki/2005_%D0%B3%D0%BE%D0%B4" title="2005 год">2005 года</a> он являлся самым долговечным президентом в <a href="/wiki/%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Африка">Африке</a>.
