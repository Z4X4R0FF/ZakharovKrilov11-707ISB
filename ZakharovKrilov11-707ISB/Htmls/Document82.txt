<p><b>Бронетанковое вооружение и техника</b>, или <b>бронетехника</b> — боевые <a href="/wiki/%D0%91%D1%80%D0%BE%D0%BD%D1%8F" title="Броня">бронированные</a> машины на <a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%B5%D1%81%D0%BE" title="Колесо">колёсном</a> и <a href="/wiki/%D0%93%D1%83%D1%81%D0%B5%D0%BD%D0%B8%D1%87%D0%BD%D1%8B%D0%B9_%D0%B4%D0%B2%D0%B8%D0%B6%D0%B8%D1%82%D0%B5%D0%BB%D1%8C" title="Гусеничный движитель">гусеничном</a> <a href="/wiki/%D0%A8%D0%B0%D1%81%D1%81%D0%B8" title="Шасси">шасси</a>, а также военные машины созданные на их базе или предназначенные для их обслуживания и <a href="/wiki/%D0%9C%D0%BE%D1%82%D0%BE%D1%86%D0%B8%D0%BA%D0%BB" title="Мотоцикл">мотоциклы</a>.
</p>

<p><b>Пехотный танк</b>, <b>Модель 3</b>, <b>«Валентайн»</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">Tank, Infantry Mk III, Valentine</span>) — <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">британский</a> <a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F_%D1%82%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2#Пехотный_танк" title="Классификация танков">пехотный</a> <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA" title="Танк">танк</a> периода <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a>, <a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F_%D1%82%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2#Лёгкий_танк" title="Классификация танков">лёгкий</a> по массе. Спроектирован в <a href="/wiki/1938" class="mw-redirect" title="1938">1938</a> году фирмой <i><a href="/wiki/Vickers-Armstrong" class="mw-redirect" title="Vickers-Armstrong">Vickers-Armstrong</a></i>. За время серийного выпуска, с июня <a href="/wiki/1940" class="mw-redirect" title="1940">1940</a> по апрель <a href="/wiki/1944" class="mw-redirect" title="1944">1944</a> года (по некоторым данным — до начала 1945 года), <a href="/wiki/%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%BE%D0%B1%D1%80%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F" title="Великобритания">Великобританией</a> и <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%B0%D0%B4%D0%B0" title="Канада">Канадой</a> выпущено 8275 «Валентайнов», что сделало его самым многочисленным британским танком Второй мировой. Состоял на вооружении Великобритании и ряда стран <a href="/wiki/%D0%A1%D0%BE%D0%B4%D1%80%D1%83%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%BD%D0%B0%D1%86%D0%B8%D0%B9" title="Содружество наций">Британского Содружества</a>, также в значительных количествах поставлялся по программе <a href="/wiki/%D0%9B%D0%B5%D0%BD%D0%B4-%D0%BB%D0%B8%D0%B7" title="Ленд-лиз">ленд-лиза</a> в <a href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="СССР">СССР</a>. Использовался вплоть до <a href="/wiki/1945" class="mw-redirect" title="1945">1945</a> года и признан военными специалистами одним из наиболее удачных танков в своём классе. После войны оставался на вооружении <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%B0%D1%8F_%D0%97%D0%B5%D0%BB%D0%B0%D0%BD%D0%B4%D0%B8%D1%8F" title="Новая Зеландия">Новой Зеландии</a> до <a href="/wiki/1955" class="mw-redirect" title="1955">1955</a> года.
</p>

<p><b>Тип 95</b> (<a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Японский язык">яп.</a> <span lang="ja" style="font-size: 110%; font-style: normal;">九五式</span>)&#160;— <a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D0%B8%D1%8F" title="Япония">японский</a> <a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F_%D1%82%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2#Лёгкий_танк" title="Классификация танков">лёгкий</a> <a href="/wiki/%D0%A2%D0%B0%D0%BD%D0%BA" title="Танк">танк</a> <a href="/wiki/1930-%D0%B5_%D0%B3%D0%BE%D0%B4%D1%8B" title="1930-е годы">1930-х годов</a>. Также известен под названием <b>«Ха-Го»</b>. Разработан в <a href="/wiki/1933_%D0%B3%D0%BE%D0%B4" title="1933 год">1933</a>—<a href="/wiki/1934_%D0%B3%D0%BE%D0%B4" title="1934 год">1934&#160;годах</a> для сопровождения механизированных <a href="/wiki/%D0%9A%D0%B0%D0%B2%D0%B0%D0%BB%D0%B5%D1%80%D0%B8%D1%8F" title="Кавалерия">кавалерийских соединений</a>. Серийно производился с <a href="/wiki/1936_%D0%B3%D0%BE%D0%B4" title="1936 год">1936</a> по <a href="/wiki/1943_%D0%B3%D0%BE%D0%B4" title="1943 год">1943 годы</a>, использовался во <a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D0%BE-%D0%BA%D0%B8%D1%82%D0%B0%D0%B9%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0_(1937%E2%80%941945)" title="Японо-китайская война (1937—1945)">Второй японо-китайской войне</a> и был, наряду со <a href="/wiki/%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F_%D1%82%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2#Средний_танк" title="Классификация танков">средним</a> «<a href="/wiki/%D0%A7%D0%B8-%D0%A5%D0%B0" title="Чи-Ха">Чи-Ха</a>», основным японским танком <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой войны</a>. Являясь довольно удачной конструкцией для середины 1930-х годов, в ходе войны безнадёжно устарел, однако нехватка бронетехники вынудила японцев использовать его до самого конца <a href="/wiki/%D0%92%D1%82%D0%BE%D1%80%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Вторая мировая война">Второй мировой</a> на всех <a href="/wiki/%D0%A2%D0%B5%D0%B0%D1%82%D1%80_%D0%B2%D0%BE%D0%B5%D0%BD%D0%BD%D1%8B%D1%85_%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D0%B9" title="Театр военных действий">театрах военных действий</a>.
</p>

<div style="display:block; border:1px solid #535549; vertical-align:top; background:#FFFFF0; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<p><b><a href="/wiki/%D0%9C%D0%B3%D0%B5%D0%B1%D1%80%D0%BE%D0%B2,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%90%D0%B2%D0%B5%D0%BB%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Мгебров, Владимир Авельевич">Влади́мир Аве́льевич Мге́бров</a></b> (<a href="/wiki/1886" class="mw-redirect" title="1886">1886</a> — <span data-wikidata-property-id="P570" class="no-wikidata">21 августа (<a href="/wiki/4_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="4 сентября">4 сентября</a>) <a href="/wiki/1915_%D0%B3%D0%BE%D0%B4" title="1915 год">1915</a></span>) — российский офицер, изобретатель и конструктор оружия и бронетехники. <a href="/wiki/%D0%A8%D1%82%D0%B0%D0%B1%D1%81-%D0%BA%D0%B0%D0%BF%D0%B8%D1%82%D0%B0%D0%BD" title="Штабс-капитан">Штабс-капитан</a> (1915). Автор ряда весьма прогрессивных для своего времени проектов <a href="/wiki/%D0%91%D1%80%D0%BE%D0%BD%D0%B5%D0%B0%D0%B2%D1%82%D0%BE%D0%BC%D0%BE%D0%B1%D0%B8%D0%BB%D1%8C" title="Бронеавтомобиль">бронеавтомобилей</a>, получивших впоследствии его имя: <a href="/wiki/%D0%9C%D0%B3%D0%B5%D0%B1%D1%80%D0%BE%D0%B2-%D0%A0%D0%B5%D0%BD%D0%BE" title="Мгебров-Рено">Мгебров-Рено</a>, <a href="/wiki/%D0%9C%D0%B3%D0%B5%D0%B1%D1%80%D0%BE%D0%B2-%D0%A3%D0%B0%D0%B9%D1%82" title="Мгебров-Уайт">Мгебров-Уайт</a> и другие.
</p><p>Родился в 1886 году в <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москве</a>. К началу <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">Первой мировой войны</a> (июнь <a href="/wiki/1914_%D0%B3%D0%BE%D0%B4" title="1914 год">1914 года</a>) проходил службу в Учебной автомобильной роте, вскоре переформированной в Военную автошколу. Впоследствии переведён в 1-ю запасную автомобильную роту. В 1915 году Мгебров по собственной инициативе разработал ряд проектов <a href="/wiki/%D0%91%D1%80%D0%BE%D0%BD%D0%B5%D0%B0%D0%B2%D1%82%D0%BE%D0%BC%D0%BE%D0%B1%D0%B8%D0%BB%D1%8C" title="Бронеавтомобиль">бронеавтомобилей</a>, отличавшихся крайним новаторством. В частности, Мгебров был одним из первых конструкторов бронетехники в мире, предложивших использование рационального бронирования (установку бронелистов под большими углами наклона) для повышения пулестойкости корпуса. В общей сложности в 1915 — 1916 годах по проектам Мгеброва было забронировано около 20 машин, активно применявшихся в боях Первой мировой и <a href="/wiki/%D0%93%D1%80%D0%B0%D0%B6%D0%B4%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0_%D0%B2_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Гражданская война в России">Гражданской</a> войн.
