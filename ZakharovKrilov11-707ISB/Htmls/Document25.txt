<p><b>Респу́блика Буря́тия</b> — субъект <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Российской Федерации</a>, согласно собственной конституции — демократическое правовое государство, объединенное с Российской Федерацией, <a href="/wiki/%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D0%BE%D0%B5_%D1%83%D1%81%D1%82%D1%80%D0%BE%D0%B9%D1%81%D1%82%D0%B2%D0%BE_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8" title="Федеративное устройство России">субъект</a> <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Российской Федерации</a>. Входит в состав <a href="/wiki/%D0%A1%D0%B8%D0%B1%D0%B8%D1%80%D1%81%D0%BA%D0%B8%D0%B9_%D1%84%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D0%BE%D0%BA%D1%80%D1%83%D0%B3_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B9_%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D0%B8" class="mw-redirect" title="Сибирский федеральный округ Российской Федерации">Сибирского Федерального округа</a>. Республика Бурятия с севера и запада (по акватории оз. <a href="/wiki/%D0%91%D0%B0%D0%B9%D0%BA%D0%B0%D0%BB" title="Байкал">Байкал</a>) граничит с <a href="/wiki/%D0%98%D1%80%D0%BA%D1%83%D1%82%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Иркутская область">Иркутской областью</a>, на крайнем западе на сравнительно коротком расстоянии пролегает граница с <a href="/wiki/%D0%A2%D1%8B%D0%B2%D0%B0" title="Тыва">Республикой Тыва</a>, на юге проходит государственная граница с <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B3%D0%BE%D0%BB%D0%B8%D1%8F" title="Монголия">Монголией</a>, на востоке&#160;— граница с <a href="/wiki/%D0%97%D0%B0%D0%B1%D0%B0%D0%B9%D0%BA%D0%B0%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D1%80%D0%B0%D0%B9" title="Забайкальский край">Забайкальским краем</a>.<br />
<i>Площадь</i> — 351 334 км²,  <i>Население</i> — 963 492 человек,  <br />
<i>Столица</i> — город <a href="/wiki/%D0%A3%D0%BB%D0%B0%D0%BD-%D0%A3%D0%B4%D1%8D" title="Улан-Удэ">Улан-Удэ</a>, <i>Образована</i> — <a href="/wiki/30_%D0%BC%D0%B0%D1%8F" title="30 мая">30 мая</a> <a href="/wiki/1923_%D0%B3%D0%BE%D0%B4" title="1923 год">1923 года</a>
</p>

<li>В <b><a href="/wiki/%D0%90%D0%B3%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D1%82%D0%B5%D0%BF%D0%BD%D0%B0%D1%8F_%D0%B4%D1%83%D0%BC%D0%B0" title="Агинская степная дума">одном историческом административном органе</a></b> в&#160;<a href="/wiki/%D0%90%D0%B3%D0%B8%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D1%81%D1%82%D0%B5%D0%BF%D1%8C" title="Агинская степь">глубине Сибири</a> работали три историка.</li></ul>
