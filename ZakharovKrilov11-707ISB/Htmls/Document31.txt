<div style="display:block; border:1px solid #F8C100; vertical-align:top; background:white; color:black; margin-bottom:10px; padding:.3em 1em 0.5em; margin-top:0">
<p><b><a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D1%8F" title="Бразилия">Брази́лия</a></b>, официальное название <b>Федерати́вная Респу́блика Брази́лия</b> (<a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D1%83%D0%B3%D0%B0%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Португальский язык">порт.</a> <span lang="pt" style="font-style:italic;">República Federativa do Brasil</span>) — самое большое по площади и населению <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%BE" title="Государство">государство</a> в <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Америка">Южной Америке</a>. Граничит со всем странами <a href="/wiki/%D0%9A%D0%BE%D0%BD%D1%82%D0%B8%D0%BD%D0%B5%D0%BD%D1%82" title="Континент">континента</a>, кроме <a href="/wiki/%D0%A7%D0%B8%D0%BB%D0%B8" title="Чили">Чили</a> и <a href="/wiki/%D0%AD%D0%BA%D0%B2%D0%B0%D0%B4%D0%BE%D1%80" title="Эквадор">Эквадора</a>, имеет около 16 тыс. км границ и 7,4 тыс. км береговой линии.
</p>

<p><b><a href="/wiki/%D0%93%D0%B0%D1%80%D1%80%D0%B8%D0%BD%D1%87%D0%B0" title="Гарринча">Мануэл Франсиско дос Сантос</a></b> (<a href="/wiki/28_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="28 октября">28 октября</a> <a href="/wiki/1933" class="mw-redirect" title="1933">1933</a>, <a href="/wiki/%D0%9C%D0%B0%D0%B6%D0%B5" title="Маже">Пау-Гранди</a> — <a href="/wiki/20_%D1%8F%D0%BD%D0%B2%D0%B0%D1%80%D1%8F" title="20 января">20 января</a> <a href="/wiki/1983" class="mw-redirect" title="1983">1983</a>, <a href="/wiki/%D0%A0%D0%B8%D0%BE-%D0%B4%D0%B5-%D0%96%D0%B0%D0%BD%D0%B5%D0%B9%D1%80%D0%BE" title="Рио-де-Жанейро">Рио-де-Жанейро</a>), более известный как <b>Гарринча</b> (<a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%B0%D1%80%D0%B8%D0%B0%D0%BD%D1%82_%D0%BF%D0%BE%D1%80%D1%82%D1%83%D0%B3%D0%B0%D0%BB%D1%8C%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D1%8F%D0%B7%D1%8B%D0%BA%D0%B0" title="Бразильский вариант португальского языка">порт.-браз.</a> <span lang="pt" style="font-style:italic;">Garrincha</span>) — <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D1%8F" title="Бразилия">бразильский</a> <a href="/wiki/%D0%A4%D1%83%D1%82%D0%B1%D0%BE%D0%BB" title="Футбол">футболист</a>, правый <a href="/wiki/%D0%9D%D0%B0%D0%BF%D0%B0%D0%B4%D0%B0%D1%8E%D1%89%D0%B8%D0%B9_(%D1%84%D1%83%D1%82%D0%B1%D0%BE%D0%BB)" title="Нападающий (футбол)">нападающий</a>, считается лучшим правым крайним нападающим в истории футбола. По <a href="/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D0%BB%D1%83%D1%87%D1%88%D0%B8%D1%85_%D1%84%D1%83%D1%82%D0%B1%D0%BE%D0%BB%D0%B8%D1%81%D1%82%D0%BE%D0%B2_XX_%D0%B2%D0%B5%D0%BA%D0%B0_%D0%BF%D0%BE_%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D0%B8_%D0%9C%D0%A4%D0%A4%D0%98%D0%98%D0%A1" title="Список лучших футболистов XX века по версии МФФИИС">опросу МФФИИС</a> он занимает 8 место среди лучших футболистов мира и 4 среди лучших футболистов <a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%B0%D1%8F_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0" title="Южная Америка">Южной Америки</a> в <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX веке</a>. Занимает 6 место среди лучших игроков в истории <a href="/wiki/%D0%A7%D0%B5%D0%BC%D0%BF%D0%B8%D0%BE%D0%BD%D0%B0%D1%82_%D0%BC%D0%B8%D1%80%D0%B0_%D0%BF%D0%BE_%D1%84%D1%83%D1%82%D0%B1%D0%BE%D0%BB%D1%83" title="Чемпионат мира по футболу">чемпионатов мира</a> по версии газеты «<a href="/wiki/The_Times" title="The Times">Times</a>». Входит в символическую сборную лучших игроков в истории чемпионатов и <a href="/wiki/%D0%9A%D1%83%D0%B1%D0%BE%D0%BA_%D0%90%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B8_%D0%BF%D0%BE_%D1%84%D1%83%D1%82%D0%B1%D0%BE%D0%BB%D1%83" title="Кубок Америки по футболу">Кубков Америки</a>. В Бразилии Гарринча считается вторым игроком в истории <a href="/wiki/%D0%A1%D0%B1%D0%BE%D1%80%D0%BD%D0%B0%D1%8F_%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%B8_%D0%BF%D0%BE_%D1%84%D1%83%D1%82%D0%B1%D0%BE%D0%BB%D1%83" title="Сборная Бразилии по футболу">национального футбола</a> после <a href="/wiki/%D0%9F%D0%B5%D0%BB%D0%B5" title="Пеле">Пеле</a>.

<div class="floatright"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Metalmania_2007_-_Sepultura_01.jpg" class="image"><img alt="Metalmania 2007 - Sepultura 01.jpg" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/52/Metalmania_2007_-_Sepultura_01.jpg/300px-Metalmania_2007_-_Sepultura_01.jpg" decoding="async" width="300" height="199" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/52/Metalmania_2007_-_Sepultura_01.jpg/450px-Metalmania_2007_-_Sepultura_01.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/52/Metalmania_2007_-_Sepultura_01.jpg/600px-Metalmania_2007_-_Sepultura_01.jpg 2x" data-file-width="3008" data-file-height="2000" /></a></div>
<p><b>Sepultúra</b> (<span class="IPA"><a href="/wiki/%D0%9C%D0%B5%D0%B6%D0%B4%D1%83%D0%BD%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D0%BE%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82" title="Международный фонетический алфавит">se.puwˈtu.ra</a></span>; <a href="/wiki/%D0%9F%D0%BE%D1%80%D1%82%D1%83%D0%B3%D0%B0%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Португальский язык">порт.</a> <span lang="pt" style="font-style:italic;">могила</span>) — самая известная и успешная <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D1%8F" title="Бразилия">бразильская</a> <a href="/wiki/%D0%9C%D0%B5%D1%82%D0%B0%D0%BB" title="Метал">метал</a>-группа, образованная в <a href="/wiki/1984_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B5" title="1984 год в музыке">1984</a> году.
</p><p>За всю свою карьеру Sepultura выпустила десять полноформатных альбомов и сменила много музыкальных жанров, среди которых <a href="/wiki/%D0%94%D1%8D%D1%82-%D0%BC%D0%B5%D1%82%D0%B0%D0%BB" title="Дэт-метал">дэт-метал</a>, <a href="/wiki/%D0%A2%D1%80%D1%8D%D1%88-%D0%BC%D0%B5%D1%82%D0%B0%D0%BB" title="Трэш-метал">трэш-метал</a>, <a href="/wiki/%D0%93%D1%80%D1%83%D0%B2-%D0%BC%D0%B5%D1%82%D0%B0%D0%BB" title="Грув-метал">грув-метал</a>, <a href="/wiki/%D0%9D%D1%8C%D1%8E-%D0%BC%D0%B5%D1%82%D0%B0%D0%BB" class="mw-redirect" title="Нью-метал">нью-метал</a>, <a href="/wiki/%D0%90%D0%BB%D1%8C%D1%82%D0%B5%D1%80%D0%BD%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9_%D0%BC%D0%B5%D1%82%D0%B0%D0%BB" title="Альтернативный метал">альтернативный метал</a> и <a href="/wiki/%D0%A5%D0%B0%D1%80%D0%B4%D0%BA%D0%BE%D1%80_(%D1%80%D0%BE%D0%BA-%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B0)" class="mw-redirect" title="Хардкор (рок-музыка)">хардкор</a>, на котором группа пока что остановилась. Sepultura сыграла важную роль в становлении бразильского метала и дэт-метала вообще; она является важнейшим исполнителем трэш-метала. На сегодняшний день во всём мире продано более одного миллиона копий альбома <i><a href="/wiki/Arise" title="Arise">Arise</a></i> (он <a href="/wiki/%D0%A1%D0%B5%D1%80%D1%82%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F_RIAA" title="Сертификация RIAA">сертифицирован RIAA</a> как <a href="/wiki/%D0%9F%D0%BB%D0%B0%D1%82%D0%B8%D0%BD%D0%BE%D0%B2%D1%8B%D0%B9_%D0%B4%D0%B8%D1%81%D0%BA" class="mw-redirect" title="Платиновый диск">платиновый</a>); а два альбома группы — <i><a href="/wiki/Chaos_A.D." title="Chaos A.D.">Chaos A.D.</a></i> и <i><a href="/wiki/Roots_(%D0%B0%D0%BB%D1%8C%D0%B1%D0%BE%D0%BC_Sepultura)" title="Roots (альбом Sepultura)">Roots</a></i> — сертифицированы RIAA как «<a href="/wiki/%D0%97%D0%BE%D0%BB%D0%BE%D1%82%D0%BE%D0%B9_%D0%B4%D0%B8%D1%81%D0%BA" title="Золотой диск">золотые</a>»; это означает, что только в <a href="/wiki/%D0%A1%D0%A8%D0%90" class="mw-redirect" title="США">США</a> продано более полумиллиона копий каждого. Продажи альбома <i><a href="/wiki/Beneath_the_Remains" title="Beneath the Remains">Beneath the Remains</a></i> по всему миру составляют 600 000 копий.
Общие же продажи составляют более трёх с половиной миллионов копий альбомов, что достаточно редко для групп, играющих в экстремальных направлениях, и свидетельствует о коммерческом успехе Sepultura. (<a href="/wiki/Sepultura" title="Sepultura">далее…</a>)
</p>

<ul><li><b><a href="/wiki/%D0%A4%D1%83%D1%82%D0%B2%D0%BE%D0%BB%D0%B5%D0%B9" title="Футволей">Новый вид спорта</a></b> (<i>на илл.</i>) появился из-за запрета играть в футбол на пляже <a href="/wiki/%D0%9A%D0%BE%D0%BF%D0%B0%D0%BA%D0%B0%D0%B1%D0%B0%D0%BD%D0%B0" title="Копакабана">Копакабана</a>.</li>
<li>От решения о том, что у Бразилии будет <b><a href="/wiki/%D0%A0%D1%8B%D0%B6%D0%B5%D0%B1%D1%80%D1%8E%D1%85%D0%B8%D0%B9_%D0%B4%D1%80%D0%BE%D0%B7%D0%B4" title="Рыжебрюхий дрозд">птица-символ</a></b>, до её избрания прошло 34 года.</li>
<li><b><a href="/wiki/%D0%AE%D0%B6%D0%BD%D0%BE%D0%B0%D0%BC%D0%B5%D1%80%D0%B8%D0%BA%D0%B0%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%B4%D1%80%D0%B5%D0%B4%D0%BD%D0%BE%D1%83%D1%82%D0%BD%D0%B0%D1%8F_%D0%B3%D0%BE%D0%BD%D0%BA%D0%B0" title="Южноамериканская дредноутная гонка">Гонку вооружений</a></b> между Бразилией, <a href="/wiki/%D0%90%D1%80%D0%B3%D0%B5%D0%BD%D1%82%D0%B8%D0%BD%D0%B0" title="Аргентина">Аргентиной</a> и <a href="/wiki/%D0%A7%D0%B8%D0%BB%D0%B8" title="Чили">Чили</a> прервала <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B2%D0%B0%D1%8F_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0" title="Первая мировая война">война</a>, в которой участвовала <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D1%8F_%D0%B2_%D0%9F%D0%B5%D1%80%D0%B2%D0%BE%D0%B9_%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%BE%D0%B9_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B5" title="Бразилия в Первой мировой войне">только одна из стран</a>.</li>
<li>Ценой продолжения службы родине для <b><a href="/wiki/%D0%9A%D0%B8%D1%82%D0%B5%D1%80%D0%B8%D1%8F,_%D0%9C%D0%B0%D1%80%D0%B8%D1%8F" title="Китерия, Мария">отважной бразильянки</a></b> стало согласие надеть <a href="/wiki/%D0%9A%D0%B8%D0%BB%D1%82" title="Килт">килт</a> поверх штанов.</li>
<li><a href="/wiki/%D0%A1%D1%82%D0%B0%D1%82%D1%83%D1%8F_%D0%A5%D1%80%D0%B8%D1%81%D1%82%D0%B0-%D0%98%D1%81%D0%BA%D1%83%D0%BF%D0%B8%D1%82%D0%B5%D0%BB%D1%8F" title="Статуя Христа-Искупителя">Статуя Христа-Искупителя</a>, в <a href="/wiki/%D0%A0%D0%B8%D0%BE-%D0%B4%D0%B5-%D0%96%D0%B0%D0%BD%D0%B5%D0%B9%D1%80%D0%BE" title="Рио-де-Жанейро">Рио-де-Жанейро</a>, удостоена в 2007 году звания <b><a href="/wiki/%D0%9D%D0%BE%D0%B2%D1%8B%D0%B5_%D1%81%D0%B5%D0%BC%D1%8C_%D1%87%D1%83%D0%B4%D0%B5%D1%81_%D1%81%D0%B2%D0%B5%D1%82%D0%B0" title="Новые семь чудес света">Нового чуда света</a></b>.</li>
<li>Звёзды на <b><a href="/wiki/%D0%A4%D0%BB%D0%B0%D0%B3_%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%B8" title="Флаг Бразилии">флаге Бразилии</a></b> изображают небо над <a href="/wiki/%D0%A0%D0%B8%D0%BE-%D0%B4%D0%B5-%D0%96%D0%B0%D0%BD%D0%B5%D0%B9%D1%80%D0%BE_(%D0%B3%D0%BE%D1%80%D0%BE%D0%B4)" class="mw-redirect" title="Рио-де-Жанейро (город)">Рио-де-Жанейро</a>.</li>
<li>Очертание <b><a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%B0#Архитектурные_особенности" title="Бразилиа">столицы Бразилии</a></b> с высоты птичьего полета напоминает реактивный пассажирскый самолёт.</li>
<li>По инициативе самого популярного <a href="/wiki/Placar" title="Placar"><b>спортивного журнала</b></a> Бразилии был возрождён <a href="/wiki/%D0%A7%D0%B5%D0%BC%D0%BF%D0%B8%D0%BE%D0%BD%D0%B0%D1%82_%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%B8_%D0%BF%D0%BE_%D1%84%D1%83%D1%82%D0%B1%D0%BE%D0%BB%D1%83_(%D0%A1%D0%B5%D1%80%D0%B8%D1%8F_B)" title="Чемпионат Бразилии по футболу (Серия B)">второй дивизион</a> <a href="/wiki/%D0%A7%D0%B5%D0%BC%D0%BF%D0%B8%D0%BE%D0%BD%D0%B0%D1%82_%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D0%B8%D0%B8_%D0%BF%D0%BE_%D1%84%D1%83%D1%82%D0%B1%D0%BE%D0%BB%D1%83_(%D0%A1%D0%B5%D1%80%D0%B8%D1%8F_A)" title="Чемпионат Бразилии по футболу (Серия A)">чемпионата страны</a> по футболу.</li>
<li><b><a href="/wiki/%D0%A1%D0%BE%D0%B7%D0%B0,_%D0%98%D1%80%D0%B8%D0%BD%D0%B5%D1%83_%D0%AD%D0%B2%D0%B0%D0%BD%D0%B6%D0%B5%D0%BB%D0%B8%D1%81%D1%82%D0%B0_%D0%B4%D0%B8" title="Соза, Иринеу Эванжелиста ди">Строитель</a></b> первой железной дороги в <a href="/wiki/%D0%91%D1%80%D0%B0%D0%B7%D0%B8%D0%BB%D1%8C%D1%81%D0%BA%D0%B0%D1%8F_%D0%B8%D0%BC%D0%BF%D0%B5%D1%80%D0%B8%D1%8F" title="Бразильская империя">Бразилии</a> обладал состоянием, превышающим государственный бюджет.</li></ul>
